What is the Gene Expression Notebook?

The Gene Expression Notebook (GEN) is a modified Microsoft ® Excel ™ workbook for organizing and storing experimental data. Using GEN, you can:

  • store the details, results, and images of data from experiments
  • reuse previously entered reagents, specimens, and methods to define new experiments
  • copy reagents, specimens, and methods into new notebooks so you don't have to reenter them
  • create a central record-keeping resource for your laboratory reagents
  • submit your mouse data for inclusion in the Gene Expression Database.

Because GEN is a species-neutral tool, you can use it to manage expression data from any organism. You can also customize it to include fields for details you consider important such as dates, freezer locations of probes or specimens, or any other record details.

GEN system requirements

GEN runs in Microsoft Excel. The following are the version and memory requirements for optimum performance.
Macintosh Requirements
Version Excel 98 (GEN will not run in Excel 2000).
Operating systemMacOS 9.0 or later.
MemorySet the Excel memory to at least 256 megabytes
PC Requirements
VersionExcel 97 (GEN will not run in Excel 2000).
Operating systemWindows 95 or later.
MemoryA Windows-based system should adjust the memory automatically.

Note:

  • GEN currently does not run reliably in Excel 2000; future releases will address this issue. It can run in Windows 2000 if you use Excel 97 or 98.
  • The memory level recommended for a Macintosh is necessary so that you can insert a large number of images and experimental data into a workbook. The number of images the notebook can hold depends on the size and type of the images inserted. The number of images and amount of experimental data GEN can hold is only limited by the amount of memory assigned to GEN.

GEN terminology

assay sheetA modified Microsoft Excel spreadsheet that stores experimental results and images. The parameters of an experiment are defined on the assay sheet using worksheet data. The GEN has assay sheets for RNA In-Situ, Immunohistochemistry, Northern, Western, and RT-PCR experiments.
collapseAn operation done on assay sheets to hide rows containing experimental details in order to save screen space.
columnThe vertical arrangement of cells on a GEN worksheet or assay sheet. Columns are identified by alphabetical headings: A, B, C, D, E, and so on. The horizontal arrangement is called a row; rows are identified by numerical headings: 1, 2, 3, 4, 5, and so on.
command buttonRectangular button that carries out a command. GEN has four on the Main menu: Create New Assay Sheet, Create New Submission, Validate this Workbook, and Copy info to new Workbook.
Copy info to new Workbook Main menu command. When you click this button, GEN copies all worksheet data from the current notebook into a new notebook. It does not copy experimental data stored in any assay sheets.
Create New Assay Sheet Main menu command. Used in conjunction with the Assay Type drop-down menu to create an assay sheet for storing experimental results.
Create New Submission Main menu command. When you click this button, GEN allows to select the assay sheets of experiments you want to submit to GXD and prepares a file suitable for submission.
defaultA setting or action assumed by a computer program when none is specified by the user. For example, on GEN's Immunohistochemistry assay sheet, the default view shows the antibody, but not antigen, details.
drop-down list arrow Indicates a drop-down menu. You can select a desired value by scrolling the menu.
drop-down menu A drop-down list arrow to the right of a field indicates a drop-down menu. You can select a desired value by scrolling the menu. If there is no drop-down menu associated with a field, you must type the desired value into it.
Error ReportA worksheet in the GEN used in the Validation process. The validation function in the GEN checks for omissions in worksheet fields needed for submission of data to GXD. These omissions are then listed on the Error Report, along with the worksheet or assay sheet on which they occur.
expandAn operation done on assay sheets to display rows containing experimental details.
fieldAn item of information such as a strain name, the Theiler stage, or the notes for an experiment. A required field is one in which you must enter data, while an optional field is one you may leave blank. A collection of fields is called a record.
GXDGene Expression Database. A community database that integrates different types of gene expression information from the mouse and provides a searchable index of published experiments on gene expression during embryonic development. GXD is part of the Mouse Genome Informatics (MGI) resources.
IDThe text in the first column of each worksheet is the identifier for that row. Each data record in the GEN must have an ID that is a unique name or number.
Main menu Appears when you open GEN. Contains basic information and four command buttons for performing notebook functions: Create New Assay Sheet, Create New Submission, Validate this Workbook, and Copy info to new Workbook.
Manage window A pop-up window that allows you to create or modify worksheet entries from an assay sheet without leaving that sheet. Accessed by clicking on a Manage button.
navigation tabsTabs appear along the bottom of notebook pages. Clicking a tab opens that notebook page. GEN adds a tab every time you create a new assay sheet. Not all of the tabs are visible at once. Either click the |<, <, >, >| arrows to scroll through the tabs or right-click (control-click for a single button mouse) on the arrows to open a window listing all the pages.
notebook/workbookThese terms are used interchangeably to refer to the Gene Expression Notebook.
notesThere are three types of Notes fields in the GEN.
  • Notes on worksheets
  • Assay Notes (on all assay sheets)
  • Results Notes (on all blot assay sheets).
Use the Notes field to record information that is important to know but that does not fit into any of the other assay or worksheet fields.
recordA collection of fields; for example, the contents of a GEN worksheet.
Results menuUsed on assay sheets. Allows you to add fields to the results section to help you organize the recording of your expression results.
rowThe horizontal arrangement of cells on a GEN worksheet or assay sheet. Rows are identified by numerical headings: 1, 2, 3, 4, 5, and so on. The vertical arrangement is called a column; columns are identified by alphabetical headings: A, B, C, D, E, and so on.
specimens/samplesAssay sheet results are directly associated with specific specimens or samples. RNA in situ and immunohistochemistry assays use specimens. Blot assays use samples. You can include multiple specimens or samples in an assay sheet.
submissionThis function allows you to select the assay sheets you want to submit to GXD. GEN takes this set of assays and makes a notebook containing the designated experimental data and any worksheet parameters used in these assays. You can then send this file to GXD for inclusion in the database.
validation GEN checks to see if an entry contains the field information required for submission to GXD. You can validate a worksheet, an assay sheet, or the entire notebook. See Validation in the GEN.
Validate this Workbook Main menu command button. Click to validate the entire workbook.
viewThere are several ways sections of an assay sheet can be displayed: expanded, intermediate, and collapsed view. The expanded view displays all the data from the worksheet used for that section. The collapsed view displays only the ID. You toggle between them by clicking the >  button. The <  button is present only for Antibodies.
worksheetA Microsoft Excel spreadsheet designed to hold specific types of data used to define an experiment. The following is a list of the GEN worksheets. The GEN tab name for the worksheet, if different, appears in parentheses.
  • Genes

Downloading and installing GEN

You can access GEN from the Gene Expression Notebook Resource Page. You can also access The Mouse Anatomical Dictionary here as well as access this Help document. When you download GEN, you'll have to provide your name, e-mail address, and computer system type. We use this information to keep track of users so we can let you know about new releases and features.

To download GEN:

  1. Click on the phrase Gene Expression Notebook, in the line reading Registration and Download of the Gene Expression Notebook.
  2. Fill out the registration form and click Submit.
  3. Click Mac version or PC version, depending on which system you are installing GEN on.
  4. At the prompt, choose Save to Disk and click OK.
  5. Type in a name to save the file as and a location to save it to.
  6. Click Save.

You can now open the notebook.

Opening GEN

Once you download GEN, there are two ways to open the application:

  1. Double-click the GEN icon on your desktop.
  2. Open Excel and click File—>Open and browse to locate the GEN.xls file.
  3. Click Enable Macros when the prompt asks if you want to disable macros or enable macros.

    The GEN Main menu appears.

Note:   If you open GEN by double-clicking the GEN icon, be sure that Excel 97 or 98 is your default program. If Excel 2000 is the default, GEN will open in Excel 2000 and not function properly. If Excel 2000 is your default program, open Excel 97 or 98 first, and then open GEN.

Using worksheets

Worksheets store information on your experimental reagents and protocols. Assay sheets use worksheet entries to define the parameters of an experiment.

To open a worksheet, click the appropriate tab on the bottom of the worksheet.

  • Worksheets contain fields. (You might be more familiar with the Excel term, cell). Fields hold data. The type of data to enter is identified at the top of each column. (The Probes worksheet, for example, contains the following fields: Probe Name, Gene, Sequence Type, Species, Sequence Acc #, Subsequence, Region Covered and Notes).
  • Some fields list possible selections on a drop-down menu. To see if a drop-down menu is available on a given field, click in the field. If a drop-down list arrow appears, click it to activate the list. Scroll the list, and click your selection. In most cases, if the value you need is not listed, you can type it directly into the field.
  • The first field in a worksheet is always the ID, the unique name or identifier for the reagent or method. A drop-down menu is generated from the first field of each worksheet. Therefore, this field cannot be left blank, and each entry within it must be unique. If you enter a previously used ID, a warning appears when you leave the worksheet.
  • We hope we have designed fields that are of use for you, but use the fields in the notebook as you see fit. If there are fields you regularly use, feel free to add them as columns to the appropriate worksheets. (See Creating new worksheet fields.) Although these fields will not be displayed on the assay sheet, they are copied when you create new notebooks.
  • You can edit a worksheet by changing the data in any field and saving the notebook. Note, however, that changes are propagated to all assays using that record. A warning appears when you use a Manage window to change data in a worksheet.
  • You must make your entries on consecutive rows. Skipping rows results in an error message when you leave that worksheet. A worksheet entry separated by blank rows will not appear in any drop-down menus.
  • The blue headings of each worksheet link to the Help Sheet.

Entering worksheet data

  1. There are two ways to enter data into a worksheet:
    • Type text directly into the fields on the worksheet.
    • Use the Manage window on an assay sheet.
  2. Type the identifier for this information in the first field. Remember that you must make this ID unique.
  3. Continue filling out the worksheet by entering the appropriate data in each cell of a row.

Saving worksheet data

There are several ways to save your data from the worksheet:

  • Click the Save icon on the toolbar.
  • Click the Excel File drop-down menu and select Save (File—>Save).
  • Click File—>Exit, and when the prompt asks if you want to save your changes, click Yes. (If you use this method, you will also exit the program).

Note: We recommend that you save frequently when using the Gene Expression Notebook. If Microsoft Excel (or any other application) locks up, you'll lose any changes you have made to GEN.

Adding experimental details to assay sheets

  • All fields above Assay Notes contain information on experimental reagents and conditions. These details are stored on worksheets. Each assay sheet uses different worksheets, depending on the data to be collected. See GEN assay sheets and worksheets  for a complete list.

    Determining standard nomenclature for mouse genes and alleles

    We encourage you to use the official gene, allele, and mutation symbols and names as approved by the International Committee on Standardized Genetic Nomenclature for Mice.

    A simple method to find the approved gene symbol and name is to use the quick gene search at Mouse Genome Informatics. You can also search for a gene symbol with a sequence accession number using the Accession IDs query form. If you cannot find the gene, include in the Gene worksheet Notes field any information that will be useful to help us identify the gene and assign a new symbol and name if necessary. Types of useful information include: references, gene structure or function, gene family information, homologies to other species, and so on.

    To find the approved allele symbol and name, you can use the Alleles and Phenotypes Query Form at Mouse Genome Informatics. (For a quick guide to the nomenclature, see the Allele and Mutant Quick Nomenclature Guide. If you cannot find the allele, include in the Specimen or Sample worksheet notes field any information that will be useful to help us identify the allele and assign a new name if necessary. Types of useful information include: references, type of mutation, lab generated in, and so on. Note: At this time GXD does include information from knock-ins and knock-outs but not from other kinds of transgenic animals.

    Submitting data to GXD

    The GEN includes a mechanism for voluntary submission of data to the Gene Expression Database (GXD). Electronic submission both increases the exposure of your work and the amount of data available to the scientific community. In addition, it strengthens GXD: the value of GXD is directly proportional to the amount of data it contains. Submitted data receives accession numbers that you can cite in publications and grant applications. In this manner, data that is not complete enough to warrant a standalone publication or that cannot be included in a publication due to space limitations can be submitted to GXD, accessed and analyzed by other researchers, and referenced.

    Note:  Submission is entirely voluntary; the data and time of submission are at your discretion. You can choose to submit an individual assay or group of assays while retaining others to submit later (or not at all).

    To create a submission notebook:

    1. For ease, we recommend that you validate the assays you propose to submit before creating a submission file (although you can validate the submission notebook).
    2. Click Create New Submission on the Main menu. A list of every unsubmitted assay in the workbook appears.
    3. Click the check box beside each assay you want to submit at this time.
    4. Click Create Submission. A dialog box appears.
    5. Enter a name for the submission file and the location to store it in.
    6. Click Save to create a submission notebook. The submission notebook closes and the notebook you created the submission from remains open.

    There are two ways to open the submission notebook:

    1. Double-click the submission notebook icon.
    2. From the Excel File menu, select Open (click File—>Open), then select the submission file.

    The submission notebook is exactly like a normal workbook except:

    • There is no Main menu. You can add additional probes, specimens, results, and so on, to this submission and you can validate individual assay sheets, but you cannot create new assay sheets or perform any other functions of the Main menu.
    • The only data it contains are from assays selected for submission. This means, for example, that if a probe is not used in the selected assays, GEN removes it from the Probes worksheet.

    Be sure to delete any data from Notes fields that you do not want made public, since any notes you've entered in assay sheets or worksheets will appear in the created submission.

    To submit your data:

    • Please send your validated submission notebook to GEN@informatics.jax.org as an e-mail attachment.
    • After we receive it, we will confirm receipt of your submission.
    • Once we review your data and enter it into GXD, we will e-mail you the accession number of your submission and detail where you can view your data.

    You can also send any questions regarding submissions to the same address: GEN@informatics.jax.org.

    Customizing GEN to your needs

    You can customize the Gene Expression Notebook to meet your specific needs. Instructions for various modifications appear below.

    Note:  If you use Excel functionality to modify the Notebook in ways other than those listed below, you may encounter unexpected results.
    Can I adjust column widths? Some worksheets have more columns than you can view on a single "page." You can modify the column width so you can see more columns or make some not visible at all. There are two ways to do this.

    To change a column width by dragging:

    1. Click the column heading to select the entire column.
    2. Move the mouse to the right edge of the box until you see the cross-hairs (arrows pointing left and right).
    3. Hold the mouse button down and drag the column to widen or narrow the view.

    To change a column width using Excel menus:

    1. Select a field in the column to be adjusted.
    2. Click Format—>Column—>Width.
    3. Type the desired width in the dialog box and click OK.
    How do I add additional fields to worksheets?To do this, type a new field heading in the row after the last field heading. The heading is saved and copied to new Notebooks or Submissions. Avoid inserting new columns within the provided columns since this causes the GEN to malfunction. The new field headings do not appear on the assay sheet or the Manage window.
    How to I add additional headings to the results section?To do this, type a new heading in the shaded row after the last heading. Avoid inserting new columns within the provided columns since this causes GEN to malfunction. The new results headings appear only on the modified assay sheet. Any new assay sheets do not contain the additional header. Only add additional results headings for In Situ and Immunohistochemistry assays. Additional headings added to blot assays are overwritten if you add more samples to the assay.
    How do I modify the names of the navigational tabs?You can change the name of the assay sheet navigational tabs only partially. You should not change the name of the worksheet navigational tabs. The assay sheet tabs list the type of assay followed by a number in parentheses. You can delete the number designation and/or add additional text, but if you alter the assay type designation, the notebook will malfunction. To rename:
    1. Double-click on the tab name to highlight it.
    2. Press the right mouse button (control-click for a single button mouse) to deselect and place the cursor after the current name.
    3. Delete the number or add additional text. [For example, you could change RNA In Situ (2) to RNA In Situ E13.5.]
    How do I delete worksheet or results entries? You can delete entries in an assay results section or a worksheet row. Note:   Never delete columns in the Gene Expression Notebook.

    To delete a worksheet row or a portion of a results table:

    1. Highlight the entire active portion of the row (any column with a worksheet header or any white field in a results table). If you do not highlight the entire active row, heading, label, or legend, you cannot delete it.
    2. Click Edit—>Delete.
    3. Select Shift rows up when the dialog asks how to deal with the deleted cell.

    Note:  You cannot delete using the Delete key on your keyboard. You must use the Excel menu.

    How do I delete assay sheets?
    1. Go to the assay sheet you wish to delete.
    2. Click Edit—>Delete Sheet.
    Note:   You cannot undo this function once performed.

    Getting started using the GEN (tutorial)

    To help you get started using the GEN, the following example shows how to enter data from an RNA in situ assay from start to finish. For this exercise, let's assume that you have performed nonradioactive RNA in situs on E10.5 tissue from wild-type and Shh mutant mice using a probe for the Bmp4 gene. In the wild-type embryo, you have staining in the limb—in the margins of the limb and in the apical ectodermal ridge—and at high levels in the somites. In the mutant embryo, the staining is still present in the apical ectodermal ridge but not in the margins of the limb or in the somites.

    1. Create an assay sheet for your experiment.
      2.
      • Go to the Main menu. If you just opened the GEN, the Main menu appears. Otherwise, to go to the Main menu, click the Menu tab at the bottom of the page.
      • Click in the white box to the left of the Create New Assay Sheet button. A drop-down list arrow appears next to the white box.
      • Click the drop-down list arrow. A menu of the assay types available in GEN appears.
        • For a PC, click RNA In Situ.
        • For a Mac, click the drop-down list arrow, hold down the (left) mouse button, and drag to RNA In Situ.
      • Click Create New Assay Sheet to create and open an assay sheet to store your experiment in. The assay sheet is entitled RNA In Situ (#). If this is the first in situ assay stored in the GEN, it will be RNA In Situ (2).

      Experimental details are used in assays and stored in worksheets. There are two methods for entering worksheet data: using the Manage windows or directly into a worksheet. Steps 2-4 are examples of data entry using Manage windows; steps 5 and 7 demonstrate entering data directly into a worksheet.

    2. Type the name of the gene whose expression you analyzed in your experiment into the assay sheet.
      3.
      • Often the assay sheets open at the bottom of the page. If this happens, scroll to the top of the page to start your entry.
      • Click the Manage Genes button in the upper-left corner. The Editing: Genes dialog box appears.
      • Type Bone morphogenetic protein 4 in the box labeled Name. Type Bmp4 in the box labeled Symbol. If there is any other information you want to add regarding the gene analyzed, put it in the Notes box.
      • Click Save Changes (otherwise the entry will be lost). The data is entered into the Genes worksheet. (You can view this directly by clicking on the Genes tab at the bottom of the assay sheet page.)
      • Close the window or click the Cancel button.
      • As you did when creating your assay sheet, click the white box to the right of Gene Name. A drop-down list arrow appears.
      • Click the drop-down list arrow and select Bone morphogenetic protein 4 from the menu of gene names. Tab out of the box. The remaining fields in the Gene section fill in automatically.

    3. Type the name of the probe you used in your experiment.
      4.
      • Click Manage Probes. The Editing: Probes dialog box appears.
      • In the first box, type something similar to what you'd use in referring to your own probes, such as Bmp4 Probe 1. Since the entries in this box will appear in the drop-down menus, each probe name must be unique for the GEN to function properly.
      • In the Gene box, type Bmp4. This should be the name of the gene the probe was derived from.
      • Select cDNA from the Sequence Type drop-down menu. If the option you want is not on the list, type it into the box.
      • Type Mouse into the Species box.
      • The following boxes further define the probe specificity. Type D14814 into the Sequence Acc # box, NA 5600-5724 into Subsequence, and Third Exon into Region Covered. Put any other information you want to add regarding the probe in Notes; for example, synthesized 7/11/02; not hydrolyzed.
      • Click Save Changes, close the dialog box, and select Bmp4 Probe1 from the drop-down menu of probe names on the assay sheet. Tab out of the box. The remaining fields in the Probe section fill in automatically.

    4. Next, type the method you used to prepare your probe into the assay sheet.
      5.
      • Click Manage Probe Preps. The Editing: Probe Prep dialog box appears.
      • In the first box, type the name of the method you used to make your probe, such as Nonradioactive. Each probe prep method name in the GEN must be unique.
      • Use the drop-down menus to fill in the other boxes. For this exercise, for Type, choose RNA; for Hybridization, choose Antisense; for Labeling, choose Digoxigenin; and for Visualized with, choose Horseradish peroxidase. For any of these values, if the option you want is not listed, you can type it into the box.
      • As above, click Save Changes, close the dialog box, and select Nonradioactive from the drop-down menu of Probe Prep Methods on the assay sheet. Tab out of the box for the remaining fields in the Prep Method section to be filled in automatically.

    5. Add your specimens to the GEN using the Specimens worksheet.
      • As you become an experienced GEN user, you'll probably start adding your experimental details directly to the various worksheets (rather than using the Manage window) since entry is usually quicker that way and you can review what you already have easily. The following instructions, therefore, illustrate how to enter data using a worksheet.
      • Click the Specimens tab at the bottom of the assay sheet to go to the Immunohostochemistry and In-Situ Specimens worksheet.
      • To enter your wild-type specimen, type a name in the first field under Specimen ID similar to what you'd use for your tissue samples, such as 6/25/02 E10.5 WT #1. Each in situ/immunohistochemistry specimen name in the GEN must be unique.
      • Type C57BL/6J into the Strain field.
      • Type WT into the Mutations field (or leave it blank).
      • Type 17 into the Theiler Stage field. (Alternatively, if you fill in the age unit and value boxes, feel free to leave this blank.)
      • Choose embryonic day from the Age Unit drop-down menu and type 10.5 into the Age Value field.
      • Choose 0.5 from the Morning of Plug = drop-down menu.
      • Choose Unknown from the Sex drop-down menu.
      • If there is anything else you want to add regarding the specimen, put it in the Notes box.
      • Click Save Changes.
      • Follow essentially the same procedure to add the mutant specimen in the second row fields. For the Specimen ID, use 6/25/02 E10.5 mut #1. For the Strain, use 129S1/Sv and 129X1/SvJ. For Mutations, use Shhtm1Chg/Shhtm1Chg. In the Notes box, write targeted mutation generated by Chin Chiang. For all other boxes, use the values for the wild-type specimen.

    6. Add the specimens to your experimental record.
      • Click the RNA In Situ(2) tab to go to the assay sheet.
      • You need as many white boxes on the Specimen ID line as you have samples in your assay. For this exercise you need two, so click Add Specimen once.
      • Click in the first box to make the drop-down list arrow to appear next to the box. Select the WT specimen from the drop-down menu and tab out of the box. Repeat this in the next box to add the mutant specimen to the assay sheet.

    7. Add the details of how you prepared the specimens to the experimental record, using the worksheet.
      • To open the Immunohistochemistry and In-Situ Specimens worksheet, click the Specimen Preps tab.
      • In the first field, type the name of the method you used to prepare your probe, such as Para fixed WM. Each specimen prep method name in the GEN must be unique.
      • Use the drop-down menus to fill in the other fields. For this exercise, for Type, choose Whole mount; for Fixation, choose 4% Paraformaldehyde; and for Embedding, choose Not applicable. In the Notes field, you can type para o/n , then 20% sucrose in PBS until sank. For any of the values with a drop-down menu, if the option you want is not listed, type it into the field.
      • Click the RNA In Situ tab. Click in the first box following the Spec Prep Method, choose Para fixed WM from the drop-down menu that appears, and tab out of the box. Repeat for the mutant specimen.

    8. Add any other details of the experimental procedure that you want recorded in the Assay Notes box.


    9. Add your results to the experimental record.
      • For optimal results, add the results fields in this order: Image; Figure legend and label; then Results.
      • The GEN does not contain an image for this exercise, so if you do not have an image on your computer to use, skip straight to Adding your results.
      • In all cases, the first step is to click Hide/Show Results Menu.

      Adding an image of your results:
      • Click in the results field to select the area where you want the upper left corner of the image.
      • Choose <Insert Picture> from the RNA In Situ: Results list box.
      • Choose the image file from the selection window. Click Open to select it for insertion. The image is inserted into the results area.
      • You can move this image by (left) clicking on it and dragging to a new position.

      Adding a label and legend to your image:
      • If you wish to add a legend for your image, click in the desired row in the results field below the image.
      • Choose <Insert Legend and Label> from the RNA In Situ: Results list box.
      • Add a label to the first line and add the legend in the box below that.

      Adding your results:
      • Click in the results section to select the row where you want to add the headings.
      • Choose <Insert Headings> from the RNA In Situ: Results list box. This adds column headers to help you organize your results.
      • When you add the headings, one blank row is also added for recording results.
      • You can add more rows by choosing <Insert Rows> from the RNA In Situ: Results list box. Generally, you should add one row for each structure analyzed in each specimen. For this exercise, you will need four lines.
      • At this point, you should probably close the RNA In Situ: Results list box. You can reopen it at any time by clicking Hide/Show Results Menu.
      • Use the drop-down menu to choose the first specimen whose staining pattern you want to describe. For this example, choose 6/25/02 E10.5 WT #1.
      • In the structure box, type the name of a structure you analyzed. For this example, use limb. You can use The Mouse Anatomical Dictionary as a reference for appropriate structure names. You can cut and paste the complete name and ID for the anatomical structure, e.g., TS17;limb[2095] or TS17;forelimb bud;ectoderm;apical ectodermal ridge [2098](AER) or TS17;hindlimb bud;ectoderm;apical ectodermal ridge [2102](AER).
      • To describe the intensity of the staining, use an adjective from the Expression Level drop-down menu. For this specimen, use strong.
      • To describe the Pattern of the staining, use a descriptor from the Pattern drop-down menu. Since in this specimen, only part of the limb was stained, choose regionally restricted.
      • If there is an image of this embryo on the assay sheet, type the label of that image in the next box.
      • In the Note box describe how the staining was regionally restricted, such as strong expression in the posterior and anterior margins of the limb, including in both the ectoderm and the underlying mesenchyme. Also present in the apical ectodermal ridge.
      • On the next line of the results field, describe the staining in the somites. For specimen, choose 6/25/02 E10.5 WT #1. For structure, type somites. For Expression Level, choose strong. For Pattern, choose homogeneous.
      • On the next line of the results field, describe the staining pattern in the mutant embryo. For specimen, choose 6/25/02 E10.5 mut #1. For structure, type limb. For Expression Level, choose present. For Pattern, choose regionally restricted. In the Note box, type staining present in the apical ectodermal ridge but absent from the posterior and anterior margins of the limb.
      • On the next line of the results field, describe the staining in the mutant somites. For specimen, choose 6/25/02 E10.5 mut #1. For structure, type somites. For Expression Level, choose absent. For Pattern, choose not applicable.

    Congratulations! You have now successfully entered an experiment into the GEN.
