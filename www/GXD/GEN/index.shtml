Gene Expression Notebook Resource Page

The Gene Expression Notebook (GEN) is an electronic tool to store and manage expression data in the laboratory. The GEN, a modified Microsoft Excel workbook, functions as a lab notebook to store and organize expression data and images. It can be customized by the user and is suitable for expression data from any organism. In addition, it provides a mechanism to allow for the easy submission of mouse data, selected by the researcher, to the Gene Expression Database. General guidelines for submitting data are available here.
