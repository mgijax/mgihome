Encyclopedia 1.0a17 for Macintosh

Version 1.0a17 of the Encyclopedia of the Mouse Genome incorporates a number of enhancements. If you have any questions or problems regarding this Macintosh version of the Encyclopedia, please contact Mouse Genome Informatics User Support

New Options for Interaction with WWW Browsers

From the Preferences dialog box in the Encyclopedia, you can select either NCSA Mosaic or Netscape for interaction with Encyclopedia.

You can also choose a web site other than www.informatics.jax.org. This is intended for those users who wish to specify an MGD mirror site for MGD/Encyclopedia interaction. Currently, there are mirror sites in Australia, France, Japan and the United Kingdom.

Interaction with Netscape is fully supported. When you activate the option to have the Encyclopedia interact with MGD, then when you click on a marker displayed on a map, MGD will automatically retrieve detailed information on the selected marker.

Enhancements

Zoom box
A scrollable Zoom box appears with each map displayed. Drag Zoom bar up or down to zoom in on, or out of, a displayed map.

Picture Clipping
You can select a region of a displayed map and "drag and drop" it to the desktop of to a folder. This region is copied to a "picture clipping." This is a file that can be saved on your Mac.
Here's how...

New Preferences dialog box
The Preferences dialog box provides options to enter a Host name or select a WWW browser for interaction with MGD.

Click ONCE on a symbol to retrieve from MGD
Point the cursor at a marker symbol on the map and click once to retrieve the merker details from MGD.

Modifying a marker display
Point to a marker on the list of markers and select info.. under the Locus... menu to open the dialog box that lets you modify the way a marker is displayed.

Problems fixed in 1.0a17

  • The option to save a copy of a map fails
    You can save a copy of a map as an Encyclopedia file. The option to save a map as a PICT file is not available.

  • Page breaks do not display properly
    Page breaks now display properly. The option to show page breaks is available from the Map menu.

  • Cannot see homologies displayed on a chromosome map
    When you first open a map, the map window is horizontally scrollable and homologies are visible.

  • The chromosome line does not appear in a print-out of a map
    When you print an Encyclopedia map, the chromosome line appears in the printout. You do not need to alter print options from default values.

  • Gray patches appear on the map pane
    The map window refreshes properly as you scroll through a map so that the background does not break-up into patches.
