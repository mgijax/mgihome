The Encyclopedia of the Mouse Genome

The Encylopedia of the Mouse Genome (Encyclopedia) is a genetic map-viewing application for the Macintosh. The Encyclopedia runs on your Macintosh and works in conjunction with a Web browser to retrieve marker information from MGD to build a graphical representation of a genetic map in Encyclopedia format.

Instructions for configuring some WWW browsers with the Encyclopedia

Currently, you can only configure Netscape to recognize the Encyclopedia as a "helper" application. However, you can use any Web browser to retrieve mapping information from MGD and save it in an Encyclopedia format file.
