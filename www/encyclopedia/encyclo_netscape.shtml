Configuring Netscape 3.x for interaction with the Encyclopedia of the Mouse Genome

The following instructions explain how you can configure Netscape for Macintosh to launch the Encyclopedia with a linkage map retrieved from MGD via the WWW interface.
  1. Start Netscape
  2. Select "General Preferences..." from the "Options" menu
  3. Select the "Helpers" folder from General Preferences.
  4. Click "New..."
  5. Enter "application/x-encyclopedia-linkmap-1" in the "Mime type" field.
  6. Leave the "Suffixes" field empty.
  7. In the "Handled By" box, select "Application"
  8. Click "Browse..."
  9. In the file dialog box, locate and select the executable file "Encyclopedia ...1.0a17" and click "Open."
  10. In the "File type" window, select "TEXT."
Netscape is now configured to retrieve maps from MGD.
