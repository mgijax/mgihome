How to... Encyclopedia

The Preferences dialog provides options to specify an MGD host system for Encyclopedia/MGD interaction and to select a WWW browser to support that interaction. The changes made to Preferences affect only the current Encyclopedia session and the Preferences will revert to default values when you end the session. Note: Be sure to configure the browsers for Encyclopedia/MGD interaction.

Using the new Preferences dialog box

Open the File menu and select Preferences...

To enter a new Host name to specify the home ot the MGD you wish to have the Encyclopedia interact with, highlight the field and type the new host address. Currently, there are mirror sites in Japan and the United Kingdom.

To select or deselect a browser, click on the appropriate button.

Retrieving detailed marker information from MGD

For MGD and Encyclopedia interaction, the link to MGD via a WWW browser must be activated. Then you can select a marker on the map and retrieve the detailed record from MGD.
  1. Open an Encyclopedia map.
  2. Either a) select a WWW browser from the Preferences dialog box or b) open the Map menu and click on link to browser.
  3. Click on a marker on the map.

If you use the (b) option, the Encyclopedia will start up Netscape if it's configured for the Encyclopedia, else, it will start up Mosaic.

Creating a Picture Clipping

You can create a picture clipping of part of a map or of some text. The graphic is dragged to the desktop or folder and can be opened for display later on. Text can be pasted into a text file. A picture clipping can only be displayed. You do not need to run Encyclopedia to display a picture clipping.

  1. Point at one corner of the area you want to clip; press and hold the mouse button
  2. Drag the cursor to create a box around the area you wish to copy.
  3. Click on the inside of the bordered area and drag it out of the Encyclopedia pane. You will see an icon. Point and click on the icon to view the clipping.
