Encyclopedia 1.0a17 for the Macintosh

User Guide

About This User Guide

Purpose

This user guide provides information on the features and operations of the Encyclopedia mouse genetic map viewing application, version 1.0a17 for the Macintosh.

Scope

This guide provides information only on version 1.0a17 of the Encyclopedia for the Macintosh.

Audience

The intended audience is anyone interested in using the Encyclopedia to graphically display mouse genetic marker information retrieved from the Mouse Genome Database (MGI).

Encyclopedia 1.0a17 for the Macintosh: Overview

 The Encyclopedia of the Mouse Genome ( Encyclopedia) was first released in 1990 for the UNIX operating system only. This original version provided options to access mouse genetic linkage maps, "comparative" maps of mouse and human (or other mammalian species) homologies, cytogenetic maps, phenotype records stored in the Mouse Locus Catalog (MLC), and marker records stored in "LocusBase." Thus, the early application was, indeed, "encyclopedic" in scope, providing unified access to several separate data stores.

In 1994, the Mouse Genome Database (MGI) integrated all these data sources into one comprehensive information resource available on the WWW, thus taking over the broad functionality of the Encyclopedia so that, now, the primary function of the Encyclopedia is to provide a graphic display of genetic linkage information for the mouse with the option to include homologous markers from one of over 60 mammalian species.

The first Macintosh version of the Encyclopedia also became available in 1994. Several updated versions followed and the current release, version 1.0a17, became available in April 1996.

Data Sources for the Encyclopedia

MGI provides data that can be retrieved and displayed in Encyclopedia map format. Refer to the MGI user documentation describing maps and mapping data for details on available data sources summarized below:

  • Chromosome Committee Reports
  • MIT Whitehead Institute Genome Reports
  • MGI Marker Records
  • Mouse DNA Mapping Panels

Aside from mouse markers, an Encyclopedia map can include homologous markers from one of over 60 mammalian species.

Installing the Encyclopedia on Your Macintosh

Version 1.0a17 for the Macintosh can be downloaded from the Mouse Genome Informatics (MGI) web site or ftp server to your Macintosh computer. There are two versions for the Macintosh:

  • Version 1.0a17 for Macintosh System 7 680x0
  • Version 1.0a17 for Macintosh System 7 PowerPC

Requirements for successful installation on your system are

  • Memory: Minimum 16 megabytes
  • Disk space: Minimum 6 megabytes
  • Mac OS level: System 7.0 or above

Downloading the Encyclopedia... from the WWW

The simplest way to install the Encyclopedia is to download it from the MGI web site.

  1. From the MGI Home Page (http://www.informatics.jax.org), select the link to the Encyclopedia of the Mouse Genome page.

  2. Select the version of the Encyclopedia appropriate for your computer.

    When you click on the version you want, the system will download the application file to the default download folder as specified in your browser preferences. For example, in Netscape, see the "Downloads directory" found under "General Preferences" options for this folder specification.

  3. In this folder, double-click on the file encycloPPC 1.0a17.sea (or encyclo-1.0a17-68k.sea) to expand the file.

Downloading the Encyclopedia from the MGI FTP Server

You can use any one of a variety of file transfer protocol (ftp) applications, such as Fetch or Anarchie, to download the Encyclopedia application from the MGI ftp server.

  1. Open an ftp connection to ftp.informatics.jax.org.
  2. Log in as user "anonymous" with your E-mail address as the password.
  3. "Get" the Encyclopediaapplication file appropriate for your machine.
  4. Expand the file on your Macintosh as in the above procedure.

Configuring Netscape for MGI/ Encyclopedia Interaction

There are several ways in which MGI and the Encyclopedia can interact:

  • Using the Linkage Maps query form in MGI, you can search for markers on a chromosome and retrieve the information in Encyclopedia format for display as a linkage map.

  • MGI can automatically start the Encyclopedia application when you retrieve mapping information in Encyclopedia format.

  • The Encyclopedia application can activate a search for a marker record in MGI when you click on a marker symbol displayed on an Encyclopedia map.

Currently, you can take advantage of all these features only with the Netscape browser (version 2.0 or above). Netscape must be configured to see the Encyclopedia as a helper application if you want to execute a linkage map query in MGI and have the Encyclopedia application automatically start-up. The procedure, given below, is also available from the Encyclopedia Web page.

It is possible to configure NCSA Mosaic for this interaction but since this browser application has ceased development, instructions for configuring Mosaic are not given here.

  1. Start Netscape
  2. Select General Preferences... from the "Options" menu.
  3. Select the Helpers folder from General Preferences.
  4. Click on New... .
  5. Type application/x-encyclopedia-linkmap-1 in the "Mime type" field.
  6. Leave the "Suffixes" field empty.
  7. In the "Handled By" box, select Application .
  8. Click Browse.. .
  9. In the file dialog box, locate and select the Encyclopedia application file, Encyclopedia...1.0a17 and click on Open... .
  10. In the "File type" window, select TEXT .
  11. Click on OK to make the changes and close the Preferences dialog.

This completes the installation and browser configuration for the Macintosh version of the Encyclopedia.

Note: If you are using another browser, such as Internet Explorer, you can still use the MGI Linkage Map query form to retrieve a map in Encyclopedia format. However, you will need to save the resulting "get_map" file as prompted and then start the Encyclopedia application to view the file.

Working with the Encyclopedia Map Display

Starting Up the Encyclopedia

If you have the Encyclopedia application configured as a Netscape helper, the Encyclopedia will start-up when you use the Linkage Maps query form in MGI to build a linkage map in Encyclopedia manually by double-clicking on the Encyclopedia icon, or, by double-clicking on the icon for an Encyclopedia format file saved on your machine.

Features of the Encyclopedia Map Display Window

The Encyclopedia map display window has four panes. In the example shown in Figure 1, the chromosome map is displayed on the left side of the display. The pane on the lower right side of the window shows the alphabetical list of markers included in this map display. (The other panes on the right side of the display are not used, currently.) If you click on a marker in this list, the map display will be adjusted so that the region of the chromosome where this marker is located will be displayed.

Figure 1 shows a region of Chromosome 1 with human homologies. Note that the mouse markers are to the right of the chromosome line; homologies appear on the left. For each human homolog, the Chromosome and cytogenetic location are given.

Figure 1. Encyclopedia Map Display of a Region of Chromosome 1

Scrolling Through the Map Display

The chromosome pane has vertical and horizontal scroll bars so that you can easily scan the chromosome map. The marker list provides a vertical scroll bar.

Re-sizing Panes in the Map Display

You can point the cursor at a corner of the map display window, press the mouse button and drag the corner to enlarge or decrease the size of the window. You can point the cursor on the vertical line on the right side of the chromosome pane, press and hold the mouse button while you drag the line to the right or left, to change the width of the chromosome pane.

Zooming-In on a Chromosome

When an Encyclopedia map display opens, you see a small "zoom box" with a movable bar. If you point at the bar inside the box, press the mouse button and "drag" the bar from "0" to a higher number, you see the chromosome elongate so that markers are not so densely packed together. When you use the zoom box, you need to use the scroll bars to center the image within the window. You can drag the bar down to "zoom out" and bring the map display back to normal resolution. You can also use the Zoom In and Zoom Out options under the Map menu to accomplish the same thing.

Modifying the Display of a Marker

You can modify several features of the display of a marker:

  • color
  • font
  • size
  • marker name
  • marker location

Under the Locus menu on the menu bar, the first option is Info.... When you select a marker on the map, and then select the Locus/Info... menu option, a dialog box is displayed as shown in Figure 2.

 

 

Figure 2. Locus Information Dialog Box

You can select font style or size; edit the marker symbol or name; modify the cM position; choose whether to display or hide the marker; or, select a new color for the symbol displayed on the map. These changes only apply to the image displayed on your screen and do not affect anything in the underlying database.

Saving a Map File

Under the File menu on the menu bar, you see an option to Save as.... This provides the option to save an Encyclopedia map as a PICT file. This feature enables you to incorporate a map into a document or display it using another graphics application.

Linking to Netscape

At the present time, the Encyclopedia provides options for MGI/ Encyclopedia interaction via Netscape 2.0 or above and NCSA Mosaic. Since Mosaic has fallen into disuse, the discussion that follows applies primarily the use of Netscape options. At present, the features described below are not available for use with Internet Explorer or other browsers.

Using the Preferences Dialog Box

Under the File menu, the last option is Preferences.... When you select this option, the Preferences dialog box is displayed as shown in Figure 3. At the top of this screen is shown a text field with a host name. By default, the field contains the address for the MGI Home Site in the US. This address determines which MGI site will be accessed for marker detail queries initiated by clicking a marker symbol displayed on an Encyclopedia map. So, for example, if you are a MGI user in Japan, and want to run marker queries from an Encyclopedia map, you would want to change the host name in this box to the Japan mirror site, namely, www.niai.affrc.go.jp.

 

Figure 3. Preferences Dialog Box

The Preferences dialog box also provides the option to select either Netscape or NCSA Mosaic as the browser for the Encyclopedia to access for MGI/Encyclopedia interaction. If you choose None, then the Encyclopedia will not work with any browser until you specify, during an Encyclopedia session, that you want to "link to browser," at which point the Encyclopedia will link to Netscape, by default.

If you select one of the browsers, then, when you click on a marker displayed on an Encyclopedia map, the Encyclopedia will run a query for the marker in MGI using the browser specified in the Preferences dialog and will start up the browser application, if it is not already running. The Host Name and browser selection buttons are the only accessible options in the Preferences dialog box.

Using the Map Menu "Link to Browser" Option

Under the Map menu on the menu bar, the last option is Link to Browser. This is an on/off switch that you can use anytime during an Encyclopedia session to activate or de-activate MGI/ Encyclopedia interaction. If you choose to link to a browser and no browser is currently running, then, the next time you select a marker symbol on an Encyclopedia map display, the browser specified in your Encyclopedia Preferences will start-up and MGI will display the detailed marker record. If no browser is specified in the Encyclopedia preferences, the Encyclopedia will search for and start-up Netscape. Once you have linked to a browser and the browser is running, then whenever you click on a marker on an Encyclopedia map, MGI will display the detail record until you de-select the option to Link to Browser, or, you end the Encyclopedia session.

Retrieving a Marker Record

Once you have activated the link between the Encyclopedia and a web browser, you can point the cursor at a mouse marker displayed on an Encyclopedia map and click once to execute an MGI search for the marker record.

Picture "Clipping"

You can create a picture clipping of part of a map. You "drag" the clipping to the desktop or folder. A picture clipping can only be displayed. You do not need to run the Encyclopedia to display a picture clipping. To create a picture clipping:
  1. Point at one corner of the area you want to clip; press and hold the mouse button
  2. Drag the cursor to create a box around the area you wish to copy.
  3. Click on the inside of the bordered area and drag it out of the Encyclopedia pane. You will see an icon. Point and click on the icon to view the clipping.
