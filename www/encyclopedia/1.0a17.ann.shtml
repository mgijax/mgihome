Encyclopedia of the Mouse Genome 3.0
Version 1.0a17 for the Macintosh

April 16, 1996

Release Announcement

Version 1.0a17 of the Encyclopedia of the Mouse Genome (Encyclopedia) for Macintosh is now available. You can download the application from the Encyclopedia page on the WWW accessed from the Mouse Genome Informatics Home Page.

New Features

  • Integration with MGD via Netscape is supported
    Now you can configure Netscape to recognize the Encyclopedia so that you can retrieve marker details from MGD by selecting a symbol displayed on an Encyclopedia map.

  • Encyclopedia Prefenences box has been modified
    You can change the host name in the Preferences to specify an MGD mirror site for MGD/Encyclopedia interaction. Currently, there are mirror sites in Australia, France, Israel, Japan and the United Kingdom.

    You can select either Netscape or Mosaic for MGD/Encyclopedia interaction. You can also disable this interaction.

  • Option to create text and picture clippings
    You can point and drag the cursor to define an area of an Encyclopedia map or text pane, then drag it to the desktop as a "clipping."

Problems Fixed in 1.0a17

  • When you first open a linkage map, homologies are immediately visible.
  • The map window refreshes properly so that patches of background color do not appear.
  • The page breaks are now displayed properly when you select that option.
  • You can save a map as an Encyclopedia file. (The option to save as a PICT files has been disabled.)
  • When you print an Encyclopedia map, all vertical lines print properly.
