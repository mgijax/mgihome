Deltagen Knockout Mice
Phenotypic Data Download Package Summary

Each zipped package contains data and image files for one line of knock-out mice.  The data and format of most of the files were provided by Deltagen, Inc. MGI has not verified the content or altered the format of the material except in specific cases described below.

The name of the zip file is the MGI Accession ID of a gene whose data are within. After you open the zip file, the name of the main folder is the MGI Accession ID number of the gene and the gene symbol current at the time we created the package.  For example, MGI1891209.zip opens a main folder named MGI1891209_Efemp2.
Main folder
When opened, the main folder contains the following files and folders:

  • N/A - Not applicable
  • NOS - Not otherwise specified
  • NSA - Negative symptom assessment
