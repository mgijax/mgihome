Citing These Resources

Please use the following citation formats when referring to resources available from the Mouse Genome Informatics Web Site in your publications.

Publications to cite in your manuscripts

The following are the most recently published references for MGD, GXD, and MTB. Please cite these as appropriate.

MGD: Blake JA, Eppig JT, Richardson JE, Davisson MT, the Mouse Genome Database Group. 2000. The Mouse Genome Database (MGD): expanding genetic and genomic resources for the laboratory mouse. Nucleic Acids Res 28: 108-111.

GXD: Ringwald M, Eppig JT, Kadin JA, Richardson JE, the Gene Expression Database Group. 2000. GXD: a Gene Expression Database for the laboratory mouse: current status and recent enhancements. Nucleic Acids Res 28: 115-119.

MTB: Bult CJ, Krupke DM, Sundberg JP, Eppig JT. 2000. Mouse Tumor Biology Database (MTB): enhancements and current status. Nucleic Acids Res 28: 112-114.

An expanded list of publications about resources available at the Mouse Genome Informatics Web Site is available.

To cite specific database projects, use a format similar to that shown in following examples:

Mouse Genome Database (MGD), Mouse Genome Informatics Web Site, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/). [Type in date (month, yr) when you retrieved data cited].

Gene Expression Database (GXD), Mouse Genome Informatics Web Site, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/). [Type in date (month, yr) when you retrieved data cited].

Mouse Genome Sequencing (MGS) Project, Mouse Genome Informatics Web Site, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/). [Type in date (month, yr) when you retrieved data cited].

Mouse Tumor Biology Database (MTB), Mouse Genome Informatics Web Site, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/tumor/). [Type in date (month, yr) when you retrieved data cited].

Rat Data (RGD), Mouse Genome Informatics Web Site, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/rat/). [Type in date (month, yr) when you retrieved data cited].


To cite a specific data area or display, use a format similar to that shown in following example.

Some homology data for this paper were retrieved from the Mouse Genome Database (MGD), Mouse Genome Informatics, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/). (June, 1999 [i.e., the date you retrieved the data cited]).
