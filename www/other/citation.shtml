Citing These Resources  

Please use the following citation formats when referring to resources available from the Mouse Genome Informatics Web Site in your publications.

Publications to cite in your manuscripts

The following are recently published references for MGD, GXD, and MTB. Please cite these as appropriate.

MGD:

Blake JA, Richardson JE, Bult CJ, Kadin JA, Eppig JT and the Mouse Genome Database Group. 2002. The Mouse Genome Database (MGD): The Model Organism Database for the Laboratory Mouse. Nucl Acids Res 30: 113-115.

Eppig JT,. Blake, JA, Burkhart DL, Goldsmith CW, Lutz CM, Smith CL. 2002. Corralling conditional mutations: a unified resource for mouse phenotypes. Genesis 32:63-65.

GXD:
Begley DA, Ringwald M. 2002. Electronic tools to manage gene expression data. Trends Genet 18:108-110.

GXD: Ringwald M, Eppig JT, Begley DA, Corradi JP, McCright IJ, Hayamizu TF, Hill DP, Kadin JA, Richardson JE. 2001. The mouse gene expression database. Nucleic Acids Res 29:98-101.

MTB:

Bult CJ, Krupke DM, Naf D, Sundberg JP, Eppig JT. 2001. Web-based access to mouse models of human cancers: the mouse tumor biology (MTB) database. Nucleic Acids Res 29:95-97.

An expanded list of publications about resources available at the Mouse Genome Informatics Web Site is available.

To cite specific database projects, use a format similar to that shown in following examples:

Mouse Genome Database (MGD), Mouse Genome Informatics Web Site, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/). [Type in date (month, yr) when you retrieved data cited].

Gene Expression Database (GXD), Mouse Genome Informatics Web Site, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/). [Type in date (month, yr) when you retrieved data cited].

Mouse Genome Sequencing (MGS) Project, Mouse Genome Informatics Web Site, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/). [Type in date (month, yr) when you retrieved data cited].

Mouse Tumor Biology Database (MTB), Mouse Genome Informatics Web Site, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://tumor.informatics.jax.org. [Type in date (month, yr) when you retrieved data cited].


To cite a specific data area or display, use a format similar to that shown in following examples.

Some homology data for this paper were retrieved from the Mouse Genome Database (MGD), Mouse Genome Informatics, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/). (June, 1999 [i.e., the date you retrieved the data cited]).

Chromosome Committee Reports, Mouse Genome Database (MGD), Mouse Genome Informatics, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: http://www.informatics.jax.org/ccr/).(1999 [i.e., the year of the annual committee report cited]).

Davisson MT, Cook SA, Eicher EM. The first spontaneous mutation in the mouse Herc2 gene, MGI Direct Data Submission to Mouse Genome Database (MGD), MGI:1349786,  (URL:http://www.informatics.jax.org). (1999).
