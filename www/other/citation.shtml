Citing Resources Available Through the Mouse Genome Informatics Web Site

Please use the following citation formats when referring to MGD, GXD, or the Encyclopedia of the Mouse Genome in your publication.

Mouse Genome Database (MGD), Mouse Genome Informatics, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: "> ). (Type in date (month, yr) when you retrieved data cited).

Gene Expression Database (GXD), Mouse Genome Informatics, The Jackson Laboratory, Bar Harbor, Maine. (Type in date (month, yr) when you retrieved data cited).

Encyclopedia Of The Mouse Genome, Release 3.0. Mouse Genome Informatics, The Jackson Laboratory, Bar Harbor, Maine (July, 1994).

In addition, to cite a specific area of MGD, GXD, or the Encyclopedia, you may use a format similar to that shown in following example:

Some homology data for this paper were retrieved from the Mouse Genome Database (MGD), Mouse Genome Informatics, The Jackson Laboratory, Bar Harbor, Maine. World Wide Web (URL: ">). (June, 1996 i.e., the date you retrieved the data cited).
