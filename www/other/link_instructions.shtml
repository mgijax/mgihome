Create Web Links to MGI

This page provides instructions for creating URL links to MGI. We recommend strongly that you use these URLs for linking to MGI since alternative URLs may not be supported in the future. The topics covered are:

  • URL and syntax
  • Objects supported by the URL
  • Examples
  • What the CGI will return
  • Data dumps for obtaining accession IDs

URL and Syntax

A CGI program links to MGI via accession ID. Use the following syntax to access it:

    http://www.informatics.jax.org/searches/accession_report.cgi?id=xxxx
where xxxx is a comma-separated list of Accession IDs.

The result is a display of or more MGI objects associated with the given IDs.

MGI stores objects with accession IDs from many sources. Some examples of valid Accession IDs are:

  • MGI:12345 (MGI Accession ID)
  • J:12345 (J Number)
  • K02800 (GenBank Sequence ID)
  • GO:0030264 (GO Accession ID)
  • NM_021099, AF193796, AB005457, U20370, M22115 (GenBank accession IDS)
  • TC1572427 (DFCI Mouse Gene Index accession ID)
  • B830009P17 (Riken clone ID)

MGI uses the sequence provider's accession ID for sequences.
For more details about sequences and MGI accession IDs, see Using the Accession IDs Query Form.

    http://www.informatics.jax.org/searches/accession_report.cgi?id=MGI:57297

    http://www.informatics.jax.org/searches/accession_report.cgi?id=J:9230
  • Reference (PubMed Accession ID)
    http://www.informatics.jax.org/searches/accession_report.cgi?id=90215200
  • Mouse Marker (MGI Accession ID)
    http://www.informatics.jax.org/searches/accession_report.cgi?id=MGI:97898
  • Allele (MGI Accession ID)
    http://www.informatics.jax.org/searches/accession_report.cgi?id=MGI:1860962

    http://www.informatics.jax.org/searches/allele_report.cgi?markerID=696677
  • Orthology (MGI Accession ID)
    http://www.informatics.jax.org/searches/homology_report.cgi?id=MGI:108413
  • Probe (MGI Accession ID)
    http://www.informatics.jax.org/searches/accession_report.cgi?id=MGI:32903
  • Probe (GenBank Accession ID)
    http://www.informatics.jax.org/searches/accession_report.cgi?id=K02800

    http://www.informatics.jax.org/searches/accession_report.cgi?id=MGI:36495,MGI:36496,MGI:36501
  • Expression Assay (MGI Accession ID)
    http://www.informatics.jax.org/searches/accession_report.cgi?id=MGI:1205940

    http://www.informatics.jax.org/searches/accession_report.cgi?id=GO:0030264

Additional linking options are available upon request.

What the CGI Will Return

If the search expression matches a single reference, segment, marker, sequence, or experiment, a detailed description appears. If the search expression matches a non-mouse marker (such as a human gene), a list of homologies appears for that marker. If the search expression matches more than one object, a summary report lists the MGI accession ID, external database links, MGI links, record types in MGI associated with the accession ID(s), and a short description.

Data Dumps for Obtaining Accession IDs

All data dumps are tab-delimited and generated nightly. In most of the reports, the MGI Accession ID appears in the first column. See MGI Data and Statistical Reports for an index with links to the report categories.
