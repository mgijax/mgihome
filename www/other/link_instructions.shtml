This page provides instructions for creating URL links to MGI. We strongly recommend that you use these URLs for linking to MGI since alternative URLs may not be supported in the future.
URL and Syntax
A CGI program links to MGI via accession ID. Use the following syntax to access it:
    http://www.informatics.jax.org/searches/accession_report.cgi?id=xxxx
where xxxx is a list of Accession IDs separated by a comma and a space.

The result is a display of or more MGI objects associated with the given IDs.

MGI stores objects with accession IDs from many sources. Some examples of valid Accession IDs are:
  • MGI:12345 (MGI Accession ID)
  • J:12345 (J Number)
  • K02800 (GenBank Sequence ID)
  • GO:0030264 (GO Accession ID)
  • NM_021099, AF193796, AB005457, U20370, M22115 (GenBank accession IDS)
  • TC1572427 (DFCI Mouse Gene Index accession ID)
  • B830009P17 (Riken clone ID)
Objects Supported by the URL
There are many types of objects supported by this URL. Examples include references, segments, mouse markers, sequences, alleles, GO terms, and so on. See Querying by Accession ID - Examples for accession IDs types to use.

What the CGI Will Return
  • If the search expression matches a single reference, segment, marker, sequence, or experiment, a detailed description appears.
  • If the search expression matches a non-mouse marker (such as a human gene), a list of homologies appears for that marker.
  • If the search expression matches more than one object, a summary report lists the MGI accession ID, external database links, MGI links, record types in MGI associated with the accession ID(s), and a short description.
Data Dumps for Obtaining Accession IDs
All data dumps are tab-delimited and generated nightly. In most of the reports, the MGI Accession ID appears in the first column. See MGI Data and Statistical Reports for an index with links to the report categories.
