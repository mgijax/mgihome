This page provides instructions for creating URL links to MGI. We strongly recommend that you use these URLs for linking to MGI since alternative URLs may not be supported in the future.
URL and Syntax
For most MGI objects, use the following syntax to access them:
    http://www.informatics.jax.org/accession/xxxx
where xxxx is a single Accession ID.

The result is a display of one or more MGI objects associated with the given ID. Examples and some exceptions are shown below.

MGI stores objects with accession IDs from many sources. Some examples of valid Accession IDs are:
  • MGI:12345 (MGI Accession ID)
  • J:12345 (J Number)
  • K02800 (GenBank Sequence ID)
  • GO:0030264 (GO Accession ID)
  • NM_021099, AF193796, AB005457, U20370, M22115 (GenBank accession IDs)
  • TC1572427 (DFCI Mouse Gene Index accession ID)
  • B830009P17 (Riken clone ID)
  • rs28257452 (RefSNP ID)
Mapping experiment IDs are NOT supported.
Objects Supported by the URL
There are many types of objects supported by this URL. Examples include references, segments, mouse markers, sequences, alleles, GO terms, and so on. See Querying by Accession ID - Examples for accession IDs types to use.
Examples

  • Reference (MGI J Number)
    accession/J:9230
  • Reference (PubMed Accession ID)
    accession/21764761
  • Mouse Marker (MGI Accession ID)
    accession/MGI:97898
  • Allele (MGI Accession ID)
    accession/MGI:1860962
  • Phenotypic Alleles Summary page (MGI Accession ID)
    allele/summary?markerId=MGI:96677
  • Homology Class (HomoloGene ID)
    homology/36030
  • Probe (MGI Accession ID)
    accession/MGI:32903
  • Probe (GenBank Accession ID)
    accession/K02800
    • accession/MGI:36495,MGI:36496,MGI:36501"> accession/MGI:36495,MGI:36496,MGI:36501 -->
  • Expression Assay (MGI Accession ID)
    accession/MGI:1205940
  • Image (MGI Accession ID)
    accession/MGI:1205939
  • GO Term (GO ID)
    accession/GO:0030264
  • SNP Summary (MGI Marker Accession ID)
    snp/marker/MGI:96677
  • SNP Detail (RefSNP ID)
    snp/rs28257452
  • Human and Mouse genes associated with a human disease (DO ID)
    disease/DOID:14330?openTab=genes
  • Mouse Models of a human disease (DO ID)
    disease/DOID:14330?openTab=models

Additional linking options are available upon request.
What the URL Will Return
  • If the search expression matches a single reference, segment, marker, sequence, or experiment, a detailed description appears.
  • If the search expression matches a non-mouse marker (such as a human gene), the Vertebrate Homology Class details are displayed.
  • If the search expression matches more than one object, a summary report lists the MGI accession ID, external database links, MGI links, record types in MGI associated with the accession ID, and a short description.
Data Dumps for Obtaining Accession IDs
All data dumps are tab-delimited and generated weekly. In most of the reports, the MGI Accession ID appears in the first column. See MGI Data and Statistical Reports for an index with links to the report categories.
