MGI Professional Opportunities

The Jackson Laboratory is considering applications for the following positions in the Mouse Genome Informatics group:
  • Sybase System Administrator
  • Scientific Software Engineers
  • Scientific Curators

Sybase System Administrator

The Mouse Genome Informatics Group at The Jackson Laboratory is seeking a Sybase system administrator. This position will have primary responsibility for the administration of Sybase database servers (ASE 11.5) and web servers (Apache 1.3.4) running on Solaris 2.6. Our hardware environment includes 26 Unix machines (Solaris 2.6), ~30 Macs (Mac OS 9.0), and ~20 PCs (NT 4.0 Workstation & Server).

The successful applicant will work in a team of Mac, PC, and Unix system administrators and assume a wide variety of responsibilities including Sybase installation/setup, physical layout of databases, Sybase account and user permission administration, database schema maintenance and migration, database replication and backup, performance tuning, and web server administration. Additional duties include some Unix and Windows NT system administration. Advanced database server knowledge and two or more years of database administration experience of Sybase (preferred), Oracle, or Informix on Unix platforms is required. Working knowledge of web server administration (Apache preferred) is required. Unix and shell programming experience required.

Contact Human Resources to apply.

Scientific Software Engineers

The Mouse Genome Informatics Group at The Jackson Laboratory has openings for scientific software engineers. Our group develops biological databases and software tools used worldwide by scientists and medical researchers who use the laboratory mouse to study genetics and human diseases.

Applicants must be energetic, imaginative, and ready to participate in a highly interdisciplinary environment combining biology and computer science. Desired skills include: Web site design and implementation (HTML, CGI, Java Servlet API, Java Applets), relational database programming (SQL, JDBC), GUI design and implementation, object-oriented design and programming, Perl, Python, Java, Sybase. Excellent communication skills required. Unix experience required, Windows and Mac experience helpful. B.S. in Computer Science required. Strong interest in Biology preferred. Please refer to job posting #050/0T.

Contact Human Resources to apply.

Scientific Curators

Professional positions are available in the expanding Bioinformatics programs at The Jackson Laboratory working with the Mouse Genome Database and a new genomic sequence oriented database for mouse. Successful candidates will be primarily responsible for data acquisition and analysis, evaluating and annotating data to be incorporated into the databases, integrating information from disparate sources, and interacting with research laboratories to facilitate electronic data contributions. In addition, they will participate in database and interface design work through input into database content and display concepts. Applicants should have experience with molecular biology or mammalian systems and be versed in genetics, genomics, cellular or developmental biology. Other desirable attributes include excellent writing/communication skills and user-level experience with UNIX/Windows/MAC-OS. Ph.D.in Life Sciences or M.S. with appropriate experience required.

Contact Human Resources to apply.
