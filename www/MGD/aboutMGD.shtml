What is the Mouse Genome Database (MGD)?

The Mouse Genome Database (MGD) includes information on mouse genetic markers, molecular clones (probes, primers and YACs), phenotypes, comparative mapping data, graphical displays of linkage, cytogenetic and physical maps and experimental mapping data, as well as strain distribution patterns for recombinant inbred strains (RIs) and cross haplotypes. Sources for information in the database include the scientific literature, DNA mapping panel data produced by various researchers, the MIT Whitehead Institute (MIT genetic and physical mapping data), WashU, and the Integrated Molecular Analysis of Genomes and their Expression (I.M.A.G.E.) consortium. MGD is updated daily. Since it first became available on the Web, MGD has continued to evolve, expanding its data coverage, improving data handling, and providing new data manipulation and display tools.

Phenotype records in MGD have their origin in the Mouse Locus Catalog (MLC), formerly an independent database. You can search for phenotype records using either the Genes and Markers query form or the Alleles and Phenotypes query form. You can search the MLC using the full-text searching capability of the Search tool, which provides very reliable query results. An integrated OMIM/MLC query form lets you enter one search string to execute a query in both MLC and On-line Mendelian Inheritance in Man (OMIM), a database maintained by the National Center for Biotechnology Information (NCBI).

MGD is closely linked with the Gene Expression Literature Index. The Gene Expression Literature Index is a collection of references to the scientific literature reporting data on endogenous gene expression during mouse embryonic development. MGD provides links to Gene Expression Literature Index records through genetic marker detail records. The Gene Expression Literature Index provides links to MGD references and genetic marker records.

Where does MGD data come from?

MGD is updated on a daily basis by biologists on our curatorial staff who scan the current scientific literature, extract relevant data, and enter them in MGD. Increasingly, MGD acquires data through large scale electronic transfer. Such data include mouse physical mapping data downloaded from MIT, data generated by the collaborative efforts of WashU, the I.M.A.G.E. consortium and the American Type Culture Collection (ATCC), and by Mouse Genome Informatics. The data interface is intended to be flexible and comprehensive so that each view of particular records in MGD provides links to any related data throughout MGD, and, where possible, to other databases on the Internet.

What data does MGD contain?

MGD contains the following kinds of information:

    Gene, DNA marker, QTL and cytogenetic marker descriptions
    Mouse genetic phenotypes, genetic interrelationships, and polymorphic loci
    Mammalian orthology data
    Molecular probes and clones(probes, clones, primers, and YACs)
    Genetic and physical mapping data
    Polymorphic loci related to specified strains
    Information on inbred strains (M. Festing's listing)
    References supporting all data in MGD

Does MGD link to any external databases?

MGD provides links to relevant information in external databases wherever possible. Through markers, MGD links to GenBank, SWISS-PROT, EnsEMBL, InterPro, and EC. Through orthologies, MGD links to GDB, PigBase, SheepBase, BovMap and RATMAP. Through molecular probes and clones, MGD links to GenBank, EMBL, and DDBJ. Through references, MGD links to PubMed.

How is MGD organized?

Genes and Markers

MGD contains information on mouse genes, DNA segments, cytogenetic markers and QTLs. Each record may include the marker symbol, name, other names or symbols and synonyms, nomenclature history, alleles, STSs, chromosomal assignment (number), centimorgan location relative to the centromere, cytogenetic band, EC number (for enzymes), phenotypic classifications, Gene Ontology (GO) terms, MGI accession IDs and supporting references. (See How do I interpret the summary results of my query? and How do I interpret the detail results of my query? for more about marker records.)

Alleles and Phenotypes

Information on alleles, formerly imbedded in phenotype descriptions, is stored as a separate data set. More information on alleles will be available and accessible in MGD as time goes on. Links to alleles are provided in gene detail records. In addition, there is an Alleles and Phenotypes Query Form for direct queries against the allele data set. (See How do I interpret the detail results of my query? for more about allele records).

Molecular Probes and Clones

MGD contains information on probes, clones, primers, YACs, and STSs. Information may include the origin of the probe/clone, sequence, vector type, hybridizing loci, RFLPs, and references. (See the Probes Query Form, Using the Molecular Probes and Clones Query Form, Using the Molecular Probes and Clones Summary Report and Using the Molecular Probes and Clones Detail Report for more about probe/clone records.)

In addition, information on genetic polymorphisms is extracted from probe/clone records in MGD. The Polymorphisms Query Form enables you to search directly for such information and display it. (See How do I interpret the summary results of my query? and How do I interpret the detail results of my query? for more about polymorphism records.)

Mammalian Orthology

MGD contains orthology information for mouse, human and 15 other mammalian species. Most of the information for orthologies is extracted from the published scientific literature. In general, these papers explicitly report orthology relationships and MGD reflects that published information. (See MGD Mammalian Orthology Criteria for the information used to establish orthology.)

An orthology record shows the symbol, species, chromosome, cM position, cytogenetic band, and accession ID for each gene in the set of orthologies. In addition, links to other mammalian databases are provided where possible. MGD orthology records may link to GDB, BovMap, PiGBASE, RATMAP, SheepBase, or ArkDB.

Every orthology assertion is supported by one or more references. The display of an orthology reference includes the J number of the reference, a short format citation, gene symbol, and species. (See How do I interpret the detail results of my query? for more about orthology record content in query results.)

Orthology information is presented in three other formats: Oxford Grid, comparative map, and Orthology Map View. An Oxford Grid display presents an overview of orthology between two species. Currently, this kind of comparison is supported for the following species: mouse, human, cat, cattle, rat, pig, and sheep.

The comparative map display relies on the Linkage Map application to build a graphic display of a map of a mouse chromosome (or chromosomal region) which includes orthologies from the comparison species. See, for example, the comparative map for a 2 cM region around the mouse gene Chrng showing human orthology.

The Orthology Map View displays the mouse genome as the consensus genetic map, with either human or rat orthologs labeled by chromosome of origin and displayed next to the corresponding segment of the mouse genome. The Orthology Map View is accessible from the Orthology Menu page.

Mapping Data

MGD contains genetic mapping and linkage data, including haplotype data for linkage crosses, recombinant inbred (RI)and recombinant congenic (RD) strain distribution patterns, in-situ hybridization data, deletion mapping information, translocation breakpoint mapping, somatic cell hybrid concordance tables, congenic strain information, and physical mapping information.

Recombinant Inbred Strain Distribution Patterns Composite Data Set

The RI Data portion of MGD mapping data records has been collected and organized into composite reports accessible using the Composite Recombinant Inbred Data Sets from MGD query form. The information is displayed in tabular format with each COLUMN representing a strain and each ROW representing the distribution of alleles of a locus across these strains.

Recombinant Congenic Strain Distribution Patterns Composite Data Set

Composite Recombinant Congenic Strain Distribution Patterns data can be retrieved using a new query form accessed in the "Maps and Mapping Data" section of MGD. Information on four recombinant congenic strain sets is currently available. Data include the progenitor strains for each RCS set and tables showing the allelic distribution for each genetic marker that has been typed in each of the recombinant congenic strains. Each row represents the typing for a genetic marker and each column represents alleles carried by a specific recombinant congenic strain of the RCS set. Format of the report is very similar to that of the Composite Recombinant Inbred Strains distribution patterns.

For more information regarding the use of congenic strains to analyze traits controlled by multiple genes, see Peter Demant's paper,

"Recombinant Congenic Strains - A New Tool for Analyzing Genetic Traits Determined by More Than One Gene, " Immunogenetics 24: 416-422, 1986.

    JAX (BSB):(C57BL/6J x M. spretus)F1 x C57BL/6J

    JAX Mouse Mutant Resource BCB: (C57BL/6J x CAST/Ei)F1 x C57BL/6J
    JAX Mouse Mutant Resource BSS: (C57BL/6J x SPRET/Ei)F1 x SPRET/Ei
    Kozak FvC58: (NFS/N x M. spretus)F1 x C58/J Kozak FvSpr: (NFS/N x M. spretus)F1 x M. spretus

DNA Mapping Panel data may appear in tabular format, where each column represents a single offspring of the cross, and each row indicates, for each locus, which allele is present in each of the offspring. The order of rows is determined by linkage on the chromosome, and the locus nearest the centromere appears at the top of the display. Centimorgan locations for loci in the cross are determined by the provider of the cross.

Graphical Map Displays

Linkage Maps

Marker records in MGD include available information on genetic mapping locations from several sources: MGD, Chromosome Committees (up to the year 2000), and MIT. MGD provides mapping tools to generate graphic displays of genetic maps. Using the Linkage Maps form, you can generate a graphical map display in Web Map format to display linkage information for a chromosome from a particular cross. Centimorgan locations are specific to the cross and may differ from estimates for the same loci found in MGD marker records. You can print the display using the PostScript format option.

Cytogenetic Maps

MGD stores cytogenetic band information for mouse markers. The Cytogenetic Maps query form provides access to map images. Relatively few markers have been assigned cytogenetic locations.

Physical Maps

As of June 1997, MGD stores physical mapping data from MIT's mouse physical mapping project in which MIT primers and STSs are mapped against a mouse YAC library to build a contig map of the mouse. MGD provides query access to the MIT primers, STSs, and YACs. In addition, MGD provides integrated genetic/physical displays in Web Map format. These map files are updated when the MIT physical mapping data set is updated.

References

MGD contains references for all data in the database. As of November 2002, there were nearly 75,000 references. MGD reference format is similar to that used by PubMed. Data for the full MGD reference are extracted from PubMed when possible. References are first entered into MGD in an abbreviated format; these records are updated when the PubMed records become available. If the journal is not indexed by PubMed, our curators enter the full reference into MGD manually. (See the References Query Form How do I interpret the detail results of my query? for more information about reference record content in the query results.)

Chromosome Committee Reports

The Chromosome Committee Reports were published annually (up to the year 2000) on the MGI web site. The reports provide a consensus view on linkage information for each chromosome. These reports were also the source for cM values for markers stored in MGD.
