Mouse Genome Informatics Overview

This overview presents a brief description of the features and data content of the Mouse Genome Informatics (MGI) resource available on the World Wide Web.

The MGI web site provides integrated access to various sources for information on the genetics and biology of the laboratory mouse. MGI projects comprise the Mouse Genome Database (MGD), the Gene Expression Database (GXD), Mouse Genome Sequence (MGS) project, and related resources including the Mouse Tumor Biology database, Michael Festing's Listing of Inbred Strains of Mice and Rats, and MouseBLAST.

The goals of the MGI projects are to capture, store and manage high quality data suitable for analysis and integration with other biological data; provide efficient access to these data through an easy-to-use, flexible user interface; develop software tools that facilitate analysis of data and enable researchers to display data in a variety of graphical and textual formats; promote integration of MGI information with other scientific information resources by creating links between MGI and other scientific web sites. We aim to continually modify and develop our information resources in response to the changing information needs of the scientific community. (See the MGI Chronology for highlights of MGI project history.)

MGD Project

MGD includes information on mouse genetic markers, molecular segments (probes, primers and YACs), phenotypes, comparative mapping data, graphical displays of linkage, cytogenetic, and physical maps, experimental mapping data, as well as strain distribution patterns for recombinant inbred strains(RIs) and cross haplotypes. Sources for information in the database include the scientific literature, DNA mapping panel data produced by various researchers, the MIT Whitehead Institute (MIT genetic and physical mapping data), WashU and the I.M.A.G.E. consortium. MGD is updated daily. Since it first became available on the WWW, MGD has continued to evolve, expanding its data coverage, improving data handling, and providing several new data manipulation and display tools.

MGD is integrated with the Encyclopedia of the Mouse Genome, a genetic map viewing application for the Macintosh platform. You can configure Netscape to recognize the Encyclopedia as a "helper" application. You can then use the Linkage Maps query form to generate a linkage map in "Encyclopedia" format. The Encyclopedia will automatically start up when you execute the search. The resulting map displays marker symbols arranged along the chromosome. You can click on a marker and Netscape will search MGD and retrieve the detail record for the selected marker for display in a browser window. You can build an Encyclopedia linkage map using information from MGD marker records, Chromosome Committee Reports, MIT mouse DNA marker maps, and DNA mapping panels.

Another format for map display is the "Web Map," a type of HTML file that provides a graphical display of cytogenetic and linkage maps on a WWW browser page. The appearance is similar to that of a "PostScript" map. The marker symbols on the map are linked to detail marker records in MGD.

Phenotype records in MGD have their origin in the Mouse Locus Catalog (MLC), formerly an independent database. You can search for phenotype records using the Genes and Markers query form. MLC full-text searching capability relies on the Glimpse search engine. Glimpse is easy-to-use and provides very reliable query results. In addition, an integrated OMIM/MLC query form is available which lets you enter one search string to execute a query in both MLC and On-line Mendelian Inheritance in Man (OMIM), a database maintained by the National Center for Biotechnology Information (NCBI).

MGD is closely linked with the Gene Expression Literature Index. The Gene Expression Literature Index is a collection of references to the scientific literature reporting data on endogenous gene expression during mouse embryonic development. MGD provides links to Gene Expression Literature Index records through genetic marker detail records. The Gene Expression Literature Index provides links to MGD references and genetic marker records.

MGD Data

MGD is updated on a daily basis by biologists on the editorial staff who scan the current scientific literature, extract relevant data and enter it in MGD. Increasingly, MGD acquires data through large scale electronic transfer. Such data include mouse physical mapping data downloaded from MIT, data generated by the collaborative efforts of WashU, the I.M.A.G.E. consortium, ATCC, and Mouse Genome Informatics. The data interface is intended to be flexible and comprehensive so that each view of particular records in MGD provides links to any related data throughout MGD and, where possible, to other databases on the Internet.

MGD contains the following kinds of information:

    Gene, DNA marker, QTL and Cytogenetic marker descriptions
    Mouse genetic phenotypes, genetic interrelationships, and polymorphic loci
    Mammalian homology data
    Molecular probes and segments(probes, clones, primers and YACs)
    Genetic and physical mapping data
    Polymorphic loci related to specified strains
    Information on inbred strains (M. Festing's listing)
    References supporting all data in MGD

Data Links to External Databases

MGD provides links to relevant information in external databases wherever possible. Through homologies, MGD links to GDB, PigBase, SheepBase, BovMap and RATMAP. Through molecular probes and segments, MGD links to Genbank, Entrez, and DDBJ. Through references, MGD links to Medline. Through markers, MGD links to EC and SWISS-PROT.

Genes and Markers
MGD contains information on mouse genes, DNA segments, cytogenetic markers and QTLs. Each record may include marker symbol, name,other withdrawn names or symbols and synonyms, nomenclaturehistory, alleles, STSs, chromosomal assignment, centimorgan location, cytogenetic band, EC number (for enzymes), phenotypic classifications, Gene Ontology (GO) terms, MGI accession IDs and supporting references. (See details on the content of the display of a marker record as it appears in the query results.)

Information on alleles, formerly imbedded in phenotype descriptions, is stored as a separate data set. More information on alleles will be available and accessible in MGD as time goes on. Links to alleles are provided in gene detail records. In addition, there is an Alleles and Phenotypes Query Form for direct queries against the allele data set. (See Phenotypic Alleles -- Query Results--Details for the content of an allele record as displayed in query results.

Molecular Probes and Segments

MGD contains information on probes, clones, primers, YACs and STSs. Information may include the origin of the probe/segment, sequence, vector type, hybridizing loci, RFLPs, and references. (See details on the content of a probe/segment record as it appears in the query results.)

In addition, information on genetic polymorphisms is extracted from probe/segment records in MGD. The Polymorphisms query form enables you to search directly for such information and display it. (See details on the content of the display in polymorphism query results.)

Mammalian Homology
MGD contains homology information for mouse, human and 15 other mammalian species. Most of the information for homologies is extracted from the published scientific literature. In general, these papers explicitly report homology relationships and MGD reflects that published information. (See information on criteria used for establishing homology.)

An homology record shows the symbol, species, chromosome, cM position, cytogenetic band and accession ID for each gene in the set of homologies. In addition, links to other mammalian databases are provided where possible. MGD homology records may link to GDB, BovMap, PiGBASE, RATMAP, SheepBase, or ArkDB.

Every homology assertion is supported by one or more references. The display of an homology reference includes the J number of the reference, a short format citation, gene symbol and species. (See details on the content of a homology record as it appears in the query results.)

Homology information is presented in three other formats: Oxford Grid, comparative map and Homology Map View. An Oxford Grid display presents an overview of homology between two species. Currently, this kind of comparison is supported for the following species: Mouse, Human, Cat, Cattle, Rat, Pig, Sheep.

The comparative map display relies on the Linkage Map application to build a graphic display of a map of a mouse chromosome (or chromosomal region) which includes homologies from the comparison species. See, for example, the comparative map for a 2 cM region around the mouse gene Acrg showing human homology.

The Homology Map View displays the mouse genome as the consensus genetic map, with either human or rat orthologs labeled by chromosome of origin and displayed next to the corresponding segment of the mouse genome. The Homology Map View is accessible from the Homology Menu page.

Mapping Data
MGD contains genetic mapping and linkage data, including haplotype data for linkage crosses, recombinant inbred (RI) strain distribution patterns, in-situ hybridization data, deletion mapping information, translocation breakpoint mapping, somatic cell hybrids, concordance tables, congenic strains information, and physical mapping information.

Recombinant Inbred Strain Distribution Patterns Composite Data Set
The RI Data portion of MGD mapping data records has been collected and organized into composite reports accessible using the Composite Recombinant Inbred Data Sets from MGD query form. The information is displayed in tabular format with each COLUMN representing a strain and each ROW representing the distribution of alleles of a locus across these strains.
Recombinant Congenic Strain Distribution Patterns Composite Data Set
Composite Recombinant Congenic Strain Distribution Patterns data can be retrieved using a new query form accessed in the "Maps and Mapping Data" section of MGD. Information on four recombinant congenic strain sets is currently available. Data include the progenitor strains for each RCS set and tables showing the allelic distribution for each genetic marker that has been typed in each of the recombinant congenic strains. Each row represents the typing for a genetic marker and each column represents alleles carried by a specific recombinant congenic strain of the RCS set. Format of the report is very similar to that of the Composite Recombinant Inbred Strains distribution patterns.

For more information regarding the use of congenic strains to analyze traits controlled by multiple genes, see Peter Demant's paper,

"Recombinant Congenic Strains - A New Tool for Analyzing Genetic Traits Determined by More Than One Gene, " Immunogenetics 24: 416-422, 1986.

DNA Mapping Panel data may be displayed in tabular format, where each column represents a single offspring of the cross, and each row indicates for each locus which allele is present in each of the offspring. The order of rows is determined by linkage on the chromosome and the locus nearest the centromer is at the top of the display. Centimorgan locations for loci in the cross are determined by the provider of the cross.

Graphical Map Displays

Linkage Maps
Marker records in MGD include available information on genetic mapping locations from several sources: MGD, Chromosome Committees (up to the year 2000), MIT. MGD provides mapping tools to generate graphic displays of genetic maps. Using the Linkage Maps form, you can generate a graphical map display in either Encyclopedia or Web Map format to display linkage information for a chromosome from a particular cross. Centimorgan locations are specific to the cross and may differ from estimates for the same loci found in MGD marker records. You can use the PostScript format option to generate a file for printing.
Cytogenetic Maps
MGD stores cytogenetic band information for mouse markers. The Cytogenetic Maps query form provides access to map images. Relatively few markers have been assigned cytogenetic locations.
Physical Maps
As of June 1997, MGD stores physical mapping data from MIT's mouse physical mapping project in which MIT primers and STSs are mapped against a mouse YAC library to build a contig map of the mouse. MGD provides query access to the MIT primers, STSs and YACs. In addition, MGD provides integrated genetic/physical displays in Web Map format. These map files are updated when the MIT mapping physical mapping data set is updated.

References
MGD contains references for all data in the database. As of February 1998, MGD contains over 44,000 references. MGD reference format is similar to that used by MEDLINE. Data for the full MGD reference are extracted from MEDLINE when possible. References are first entered into MGD in an abbreviated format and these records are updated when the MEDLINE records become available. If the journal is not indexed by MEDLINE, the full reference is manually entered into MGD. (See details on the content of reference records as displayed in query results.)

Chromosome Committee Reports
The Chromosome Committee Reports were published annually (up to the year 2000) on the MGI web site. The reports provide a consensus view on linkage information for each chromosome. These reports are also the source for cM values for markers stored in MGD.

Mouse Genome Sequence (MGS) Project

The overall goal of the Mouse Genome Sequence (MGS) project is to integrate emerging mouse genomic sequence data with the genetic and biological data available in MGD and GXD. MGS is part of the informatics infrastructure needed to support mouse-human comparative genomics. The specific aims of the project include:
  • Working cooperatively with sequencing centers and genome annotation groups to apply baseline sequence annotation to mouse genomic sequence.

  • Implementing a data integration process for connecting mouse genome sequence with existing biological knowledge.

  • Developing a web-based interface for querying and visualizing the results of the mouse genome sequence annotation-integration process, including sequence alignments of mouse-human orthologs.

The first tool released by MGS is a prototypic MouseBLAST server which enables you to compare a sequence of interest to a database of either mouse or rodent DNA or protein sequences.

Encyclopedia of the Mouse Genome (Encyclopedia)

The Encyclopedia is a genetic map-viewing application that runs on a Macintosh computer. The Encyclopedia uses information retrieved from MGD to generate a graphic display of a linkage map. (See details on the Encyclopedia application.)
