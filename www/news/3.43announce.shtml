MGI Release 3.43 Announcement - February 16, 2006  

MGI is pleased to announce its 3.43 release featuring an updated Mouse SNP Query Form. In this version, the genome coordinate discrepancy is resolved. This discrepancy occurred because SNP data for Build 34 of the mouse genome were not available from dbSNP at the time MGI moved to Build 34 gene coordinates. See Understanding Mouse SNP Coordinate Discrepancies for further details.

This "in sync" release includes over 1,757,816 SNPs for 86 mouse strains from dbSNP build 125, mapped to mouse genome sequence Build 34. You can:

  • use the Mouse SNP Query Form to search dbSNP for SNPs by strains, strain comparisons, SNP attributes (variation type and functional class), map position, marker range, or by association to MGI genes and genetic markers.
  • view comprehensive data on MGI SNP summary and detail pages.

Query results include a summary of each returned SNP, including consensus strain allele calls derived from SNP assays. You can:

  • choose an output strain sort order.
  • elect to see strain allele calls marked as Same or Different relative to a reference strain.
  • use the spreadsheet-compatible tab-delimited option for displaying SNP query results and visualizing presumptive haplotype blocks.

You can perform queries such as:

  • Show me all SNPs on Chromosome 1 with genome coordinates between 161079124 and 161984763 that are different between C57BL/6J and DBA/2 mice.
  • Return all SNPs, searching A/J, C57BL/6J, DBA/2J, C3H/HeJ, and CZECHII/EiJ, for any marker symbol/name beginning with pax.
  • Compare A/HeJ, A/J, BALB/cJ, C3H/HeJ, SM/J, 129S1/SvImJ, AKR/J, DBA/2J, LP/J, NZB/BlNJ and NZW/LacJ strains with C57BL/6J on Chromosome 1, and display all variation alleles with nonsynonymous substitutions.

MGI is the community model organism database for the laboratory mouse.
