February, 2000: NIH Sponsors Targeted Sequencing of Mouse BAC clones

In undertaking the sequencing of the C57BL/6J mouse genome, sequencing centers are using a combined strategy of whole genome shotgun sequencing and targeted BAC clone sequencing. Applications are being accepted by NHGRI from reseachers with interests in particular regions of biological significance to have BACs from those regions targeted for priority sequencing!

February, 2000: MGD and NCBI Join Forces to Develop LocusLink for Mouse

MGD provides NCBI with daily updates of gene nomenclature (symbols and names), map positions, and the curated data that identifies GenBank accession ids with particular genes. NCBI uses these data to construct a new view of this information in the LocusLink interface and determines, for all sequences available for a particular gene, which they are designating as a reference sequence. (Note these NCBI reference sequences may be composite sequences and are not necessarily from C57BL/6J, the strain being used for the mouse genome sequencing project). LocusLink is available for human, mouse, and rat, and provides links back to the primary databases from which data were derived, so users can find additional information.
