Mouse Genome Database (MGD) 3.1 Release Announcement

February 13, 1996

MGD 3.1 is now available from the World Wide Web (http://www.informatics.jax.org).

See MGD 3.1 Release Notes for details.

New Features

  • Access to the Gene Expression Database (GXD) Index

    Use the GXD icon on the MGI or MGD Home Page. GXD documentation and a GXD Index query form are now available.

  • WWW access to 1995 Chromosome Committee Reports

    The 1995 Chromosome Committee Reports are now available on our WWW site. Select "Chromosome Committee Reports" from the MGD Home Page. Note: Some reports are not yet complete. As updates are received, they will be made available.

  • Retrieve homology information in "Oxford grid" format

    An Oxford Grid provides an overview of homologies between the genomes of two selected species. Each colored box in the grid indicates known homologies and you can click on the box to retrieve the records. This option is available on the Homology query form.

  • New MGD Data and Statistical Reports available

    MGD Data and Statistical Reports now include a listing of homologies between mouse and rat and a listing of mutant markers that have been withdrawn and identified as alleles of other markers. These lists may be viewed sorted in various ways.

    See MGI Data and Statistical Reports on the MGD Home Page.

  • MGD Schema Query Form

    MGD is built using SYBASE relational database management system. Now, you can get information about the MGD database schema. This is especially useful to those interested in the public SQL interface to MGD. See MGD Schema Query Form on the MGD Home Page.

Enhancements

  • Modified MGI banner has new icons and other links.

  • On MGD query forms, the J number field can now accept multiple values separated by a comma.

  • Linkage Map tool changes

    The Linkage Map form provides an option to build a map using the markers entered in the Additional Markers field exclusively. To activate this option, select Use ONLY markers entered on this form from the Map Source list.

    In addition, a revised map generation algorithm has improved performance of the Linkage Map tool.

  • On the Genetic Markers... query form, you can now search for mutant alleles by using the phrase allele of in the Name field.
    Note: When searching for mutant alleles, you must use the Name alone.

  • The detailed query results for genetic marker queries now contain a link to Expression Data retrieved from the GXD Index. This link appears under Additional Information if GXD information is available for a marker.

  • On the References query form, fields for Chromosome, Volume and Pages have been added. Now, you can search for all the references related to a chromosome.

  • On the Homology form, the species lists now start with the two most frequently used species: mouse and human. Also, you now have the option view query results from two selected species. Previously, the homology search always retrieved all the homologies for a given gene including the selected species and any other species sharing the homology. Now, seeing the homologies for all species is optional.

NOTE:

Before using MGD 3.1, be sure to clear the Web browser cache on your computer to get rid of old MGD forms.
