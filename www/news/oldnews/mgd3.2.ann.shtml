Mouse Genome Database (MGD) 3.2 Release Announcement
February 10, 1997

1997 Chromosome Committee Reports

The 1997 Chromosome Committee Reports are available in hypertext format from MGD. The 1997 reports include options to download sections of the reports. A Master Locus list and Master References list provide summary listings generated from the reports.

Web Maps

A new "Web Map" option on the Linkage Map query form enables you to retrieve a linkage map from MGD and display it on a Web page. Marker symbols on the map are linked to MGD marker records.

Cytogenetic Maps Query Form

You can now retrieve cytogenetic maps from MGD in Web Map format providing a display of a chromosome divided into cytogenetic bands with markers positioned accordingly. Marker symbols on the map are linked to MGD marker records.

Reeves (Chr16) Sex-averaged intersubspecific mapping panel

RFLP/PCR Polymorphisms Query Form

You can now search for molecular polymorphisms related to a strain, gene, or chromosomal location, or, you can retrieve the allelic typings for a single mouse strain.

Inbred Strains of Mice and Rats (M. Festing)

The list of mouse strains has been updated and information on over 200 rat strains has been added.

New Link to ENZYME database

When you display the detailed Marker record for an enzyme in MGD, the record will include the EC (Enzyme Commission) number, if available. This EC number is now linked to the record in ENZYME.

Oxford Grid Display

The Oxford Grid, which provides a graphic display of homology between two species, has been modified to show the number of homologies between two chromosomes being compared. The option to retrieve an Oxford Grid is found on the Mammalian Homology query form.

For details on the new features and enhancements in MGD 3.2, refer to the MGD 3.2 Release Notes
