MGD 3.1 Release Update Announcement

October 1, 1996

Enhancements to MGD 3.1 are now on-line.

See the MGD 3.1 Update Release Notes for details.

NOTE: If you have used MGD recently, be sure to clear your browser cache before attempting to use the updated MGD.

Mouse Nomenclature Guidelines & Locus Symbol Registry

  • The Mouse Nomenclature Guidelines and Locus Symbol Registry page provides contact information for the MGD Mouse Nomenclature Committee and the International Committee on Standardized Genetic Nomenclature for Mice.

  • A form is now available which enables you to submit a proposed locus symbol via E-mail.

  • The Mouse Nomenclature Rules and Guidelines document has been edited and reformatted.

New full-text search engine: Glimpse

Glimpse replaces WAIS as the full-text searching engine in MGD and the GXD Index. Glimpse produces more accurate and reliable query results. This change affects full-text searching options on the following query forms:

  • Reference and GXD Index query forms - Abstract field
  • Genes, Markers, and Phenotypes query form - Phenotype (MLC) search term field
  • M. Festing's Inbred Strains query form
  • MLC/OMIM query form

NOTE: Be sure to read updated help text on full-text searching in MGD and GXD.

Chromosome Committee Reports

Some of the 1996 Chromosome Committee reports are now available on-line. In addition, a new editing interface will enable the Chromosome Committees to submit and update their files electronically.

Changes to Query Forms

There are a number of changes to query forms and query result formats. The more important changes are noted here. Refer to the release notes for details.

  • Several query forms have been renamed
  • Formatting and sort options for Mammalian Homology queries have been modified
  • The Probes and Clones query form now enables you to search for EST probes
MGD Data Updates

The following DNA Mapping Panel data sets have been updated:

  • EUCIB (BSB) (C57BL/6J x SPR or SEG/Pas)F1 x C57BL/6J
  • EUCIB (BSS) (C57BL/6J x SPR or SEG/Pas)F1 x SPR or SEG/Pas
  • Copeland-Jenkins (C57BL/6J x M. spretus)F1 x C57BL/6J
New data set:
  • MIT (C57BL/6J-ob/ob x CAST)F1 x (C57BL/6J-ob/ob x CAST)F1
