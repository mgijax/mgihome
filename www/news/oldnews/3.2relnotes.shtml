Mouse Genome Database (MGD) 3.2 Release Notes
February 10, 1997

MGD Release 3.2 New features and enhancements include

  • 1997 Chromosome Committee Reports on-line
  • Two new MGD query forms
  • Various cosmetic changes to MGD query forms
  • Updates and new additions to MGD data sets
  • Updates and reformatting of user documentation

1997 Chromosome Committee Reports
In MGD 3.2, the Chromosome Committee reports are presented in a standardized, hypertext format. For most chromosomes, the report includes:

  • Author list
  • Narrative
  • Table 1. Locus list (alphabetical by symbol)
  • Table 2. Locus list (ordered by cM position)
  • Acknowledgments
  • PostScript map

The components vary somewhat depending on the information received from the committees. In some cases, a committee report may have a single table, while other reports may include multiple tables and images. The PostScript maps are downloaded to your machine. You can either send the file to a printer or display the file using a PostScript viewer.

Chromosome Committees now submit their reports via a Web submission interface implemented in the latter half of 1996. The new interface greatly facilitates report submission, provides options for editing already submitted files, and ensures a standardized format for the reports.

Web Maps
Prior to MGD 3.2, the Encyclopedia was the only map viewing tool available from MGI. Because the Encyclopedia is supported only on Macintosh and UNIX, there has long been a need for a platform-independent graphical map display. A new "Web Map" option on the Linkage Maps query form enables you to retrieve a linkage map from MGD and display it on a Web page. The Web Map formatting produces a GIF (Graphical Image Format) image. The marker symbols appearing on the map are hyper-linked to detailed marker records. (NOTE: Although hyper-links within an image are supported in Netscape 3.X, some browsers do not support this feature.)

The default format option on the Linkage Maps form is now Web Map. The other format options are Encyclopedia and PostScript. The same MGD searching options apply to all formats.

Cytogenetic Maps Query Form
You can now retrieve cytogenetic maps from MGD in Web Map format providing a display of a chromosome divided into cytogenetic bands with markers positioned accordingly. Marker symbols on the map are linked to MGD marker records. The Cytogenetic Maps query form enables you to select a single chromosome and query MGD for information to build the map which reflects the most current data content in MGD. You can print this map but keep in mind that the image may be much wider than an 8x11 page. To print the map on a single page, use the printing and page format options available on your system to set set page orientation to "landscape" or "scale" the image to reduce the image size for printing. On the Macintosh, these options are found within the File/Page Setup... menu.

Cytogenetic placement of genes is obtained from in situ and deletion mapping experiment records in MGD and, for chromosomal aberrations, from karyotypic analysis.

RFLP/PCR Polymorphisms Query Form
You can now search for molecular polymorphisms related to a gene, chromosome or mouse strain. You can also retrieve allelic typings for a specified mouse strain. This information is drawn from PCR primer and probe/clone records in MGD. The form provides some searching tips and a link to a listing of strains. This listing takes the guess work out of specifying correct strain designations in your query.

The RFLP/PCR Polymorphism query is MGD's first pass at making polymorphism information more readily accessible than in previous releases. In future releases, we will provide options to compare strains as well.

List of Inbred Strains of Mouse and Rat (M. Festing)
Information on over 200 rat strains is now available from Michael Festings's List of Inbred Strains of Mouse and Rat. Information on the mouse strains has also been updated. An introduction and list of references is available for both mouse and rat. You can search for information on inbred strains by entering a search string in the search field. The Glimpse full-text search engine executes the search of both Mouse and Rat listings for records containing the text you entered. The query results are separated under Mouse and Rat headings. A link to text-searching help is provided.

Oxford Grid Display
An Oxford Grid graphically displays a genome-wide overwiew of genetic homology between two species. The colors of the Grid have been modified to reflect the extent of homology in each conserved genomic segment known. The option to retrieve an Oxford Grid is found on the Mammalian Homology query form.

On the Oxford Grid display, each cell of the grid represents a comparison of two chromosomes, one from each of the selected species. The number of homologies for each comparison appears inside each colored cell and the color indicates a range in the number: Gray (1), Blue (2-10), Green (11-25), Orange (26-50), Yellow (50+). Each colored cell is linked to homology records for the two chromosomes represented by that cell allowing you to click on a cell and retrieve the detailed homology records.

Changes to Existing Query Forms
You will notice a smaller banner at the top of each query form. There are some modifications and new additions to the text hyperlinks beneath the banner. The title of each query form is now linked directly to the user document related to that form. Links to the Alternative Query Interface and MGI Data and Statistical Reports have been removed from the header area of each query form. The option to Connect terms with AND or OR has been removed from all query forms.

On the References, Probes and Clones, and PCR Primer query forms, the search fields have been re-arranged. The "Sequence AccNum" field appearing on several forms has been renamed DNA Seq AccNum to reflect the name used by the sequence databases.

Short Forms
Previously grouped under the Alternative Query Interface, Short Forms are now accessed from the MGD Home Page under the Searches heading. A short form provides a single search field. Short forms provide a quick alternative to loading full query forms when you want to execute a simple search of an MGD data set. For short forms that use text-entry search fields, a link to information on using wildcards has been provided.

Data and Statistical Reports
A link to these reports is found on the MGD Home Page and there is an "MGD Reports" link located beneath the banner near the top of every MGD query form.

MGD Data and Statistical reports are generated nightly and provide the following listings:

  • Mouse genetic marker lists using various sort options and options for marker types
  • Mouse and Human homologies sorted various ways
  • Rat and Mouse homologies sorted various ways
  • General statistics on the data content of MGD
  • Marker symbols and keys formatted in tab-delimited text

Some of these reports are rather large and may exhaust the browser memory available on some machines. To work around this problem, you may want to download reports in text format. These reports are available as text files from our ftp server.

New Link to ENZYME
If the detailed marker record for an enzyme includes an EC (Enzyme Commission) number, that number is now linked to the ENZYME database.
