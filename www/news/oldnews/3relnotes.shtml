Mouse Genome Database 3.0

Release Notes

November 6, 1995

New Features

Linkage Map Tool

The Linkage Map Tool enables you to retrieve information from MGD, construct a linkage map, and save the information to a PostScript file for printing, or, display the map within the Encyclopedia of the Mouse Genome application. The functionality of the former ``PostScript (paper) Map Tool'' and the option to ``Build an Encyclopedia map using information from MGD'' are now combined in one tool.

MGD 3.0 data sets include the mouse mapping panels listed below (See ``New Data Sets''). The Linkage Map Tool provides access to information from the panels, the latest Chromosome Committee reports, MIT Whitehead Institute mouse genome maps, or MGD marker and homology tables.

In addition, the Linkage Map Tool provides an editing option. You can specify additional markers by entering marker symbol and centimorgan position in the text entry field provided on the Linkage Map Tool form.

Mini-maps

When you search using the ``Genetic Markers and Mouse Locus Catalog'' query form, a ``mini-map'' is displayed along with the details of a marker record. This graphic shows the marker's centimorgan position on the chromosome relative to a set of ``anchor'' loci on that chromosome. (Anchor loci are well-known markers for which mapping information is well established.) The retrieved marker is highlighted on the mini-map.

Enhancements

MGD Home Page Reorganized

Two main sections provide access to MGD searching options and other MGI resources. The Mouse Genome Informatics banner includes a new mouse logo, a ``world'' icon which points to the MGD mirror sites in the UK and Japan, and an ``open door'' icon which points to other Biology resources on the WWW.

Changes to MGD Query Forms and Query Results

Reference query form
  • Renamed from ``Citations'' query form
  • Medline numbers for references are linked to the Entrez records (a subset of Medline)
Genetic Markers and Mouse Locus Catalog query form
  • Renamed from ``Mouse Genetic Marker Information'' query form.
  • Marker type now includes Chromosome Aberration.
  • Detailed results for a marker show nomenclature history
  • Mini-maps in query results
  • Links to MGD information on a retrieved marker are listed under ``Additional Information''
Probes query form
  • Renamed from ``Probes and Clones'' query form.
  • Sequence AccNum field permits search by Entrez, GSDB, EMBL, or DDBJ sequence accession number.
  • Chromosome field permits selection of one or more chromosomes.
  • Vector Type field permits selection from: YAC, Cosmid, Phage, Other, Plasmid, or ALL (default).
  • Summary results show Probe, Sequence AccNum (Yes/No), Markers.
  • Detailed results include Sequence Accession Numbers with links to sequence databases.
  • PCR Amplified checkbox now works.
  • You can sort results by marker.
PCR Primers query form
  • Chromosome field permits selection of a chromosome.
  • Sequence AccNum field permits search by Entrez, GSDB, EMBL, or DDBJ sequence accession number
  • Arrangement of form fields has been modified.
  • Summary results show Primer Pair, Sequence AccNum (Yes/No), Markers.
  • Detailed results include Sequence Accession Numbers with links to sequence databases.
  • Region covered is included in detailed results.

Mammalian Homology query form
  • The query form includes links to database reports and `no forms' queries.
  • Results format options include ``Preformatted Textual Table.''
  • There are now links to records in "Sheep Gbase."
Maps and Mapping Data menu
The ``Maps and Mapping Data'' option links to a menu with links to the following searching options:

  • Marker Mapping Data query form
    For searching MGD mapping data using a query form
  • RI Strain Distribution Patterns
    For searching a composite of MGD RI data sets by RI set designation or Chromosome
  • Backcross Mapping Panel Data Sets
    For access to information from six mapping panels
  • Linkage Maps
    Access the Linkage Map Tool for generating graphic displays or printable files of mouse linkage information
In addition to search options, this menu provides links to download the Encyclopedia of the Mouse Genome and Map Manager software and data files.

Marker Mapping Data query form

  • FISH has been added to Experiment Type options.
  • MGD Accession number field has been added.
