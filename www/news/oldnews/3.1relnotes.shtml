Mouse Genome Database 3.1

Release Notes

February 13, 1996

New Features

Access to the Gene Expression Database (GXD) Index


The Gene Expression Database (GXD) Index is the first component of GXD to become available to the public. The GXD Index consists of a collection of references indexed with respect to their gene expression data content.

The GXD query form enables you to search the index by reference, gene, assay type or DPC (days post conception) values. Use the GXD icon (located on the MGD Home Page and on the MGI "banner" at the top of every MGD page) to access GXD Index documentation and the GXD Index query form.

WWW Access to the 1995 Chromosome Committee Reports

The 1995 Chromosome Committee reports have been converted to html format for easy access. All available reports have been converted although the report for Chromosome 17 is not yet available and several figures and tables for other Chromosome Committee reports are not included. As these items become available, they will be included in the set of WWW available reports.

Oxford Grid format for mammalian homology information

The tabular Oxford grid format provides an overview of homology between the genomes of two selected species. The chromosomes of one species represented by the columns in the grid, the chromosomes of the second species by the rows in the grid. Each box in the grid represents a point of comparison of a chromosome from each species. A colored box indicates there are known homologies. You can point and click on a colored box to retrieve the MGD Homology records for those two chromosomes. This option is accessed from the Mammalian Homology query form.

New MGI Data and Statistical Reports

In response to requests from the user community, the MGD Data and Statistical Reports now include a listing of homologies between mouse and rat sorted in various ways.

In addition, the reports now include a listing of mutant alleles that have been withdrawn and identified as alleles of other markers sorted in various ways.

MGD Schema Query Form available

MGD is built on the SYBASE relational database management system. MGD data is organized and stored in tables. Now, you can get information about the MGD database schema. This information is primarily useful to those who wish to use the public SQL interface to MGD.

The MGD Schema Query Form is available on the MGD Home Page.

You can search by Database Object (tables, views, etc), Database Tables (References, Markers, etc), and Substring Term (i.e., some part of the name of a database object).

Query results provide a summary listing, each item linked to the detailed schema information for the item.

New Data Sets

1995 Chromosome Committee Reports

As mentioned previously, 1995 Chromosome Committee reports are available in hypertext format. In addition, you can download the reports as orginally produced from our ftp server.

Questions or comments regarding the content of the reports should be directed to the Chromosome Committee Chairs.

Enhancements

Mouse Genome Informatics Banner Redesigned

The MGI banner now presents eight icons which link to
  • MGD home page
  • GXD (Gene Expression Database) home page
  • Encyclopedia of the Mouse Genome home page
  • MGI News and Announcements
  • User Support home page
  • MGD User Documentation
  • MGD's UK Mirror site
  • WWW directory of other biological sites

Query by multiple J numbers

On MGD query forms having a J number field, you can enter multiple J numbers separated by commas.

Linkage Map tool changes

The Linkage Map form provides the option to build a custom map using genes entered in the Additional Markers text entry field exclusively. To activate this option, select the item "Use ONLY mnrkers entered on this form" from the Map Source list at the top of the form.

The map building software has been revised so that Linkage Map tool performance is improved.

Search for mutant alleles

On the Genetic Markers...query form, you can search for mutant alleles. Use the Name field with the CONTAINS operator and enter the phrase "allele of." Set the Maximum number of records retrieved to No limit.

Note that you cannot limit the search to a Chromosome.

Link from detailed marker record to GXD data

In the detailed query results for a genetic marker, Expression data will appear in the Additional Information section and link to any GXD Index record(s) related to the marker.

New fields on the References query form

The References query form contains several new fields:
  • Chromosome
    You can search for all the references related to a chromosome.

  • Volume and Pages
    These fields enable you to define a more specific query for references in a journal.

Homology form changes

On the Homology query form, the species lists now begin with the two most commonly used species, Mouse and Human.

When you select two species to compare, the query results will, by default, show only homologies between the selected species. Two options at the top of the form enable you to specify what to include in the homology query results. The default option is only selected species. You can choose to see all homologous species.
