MGD 3.1 Update

June 5, 1996

MGD 3.1 Enhancements now available

  • New data set

    Composite Recombinant Congenic Strain Distribution Patterns data can be retrieved using a new query form accessed in the "Maps and Mapping Data" section of MGD.

    Information on four recombinant congenic strain sets is currently available. Data include the progenitor strains for each RCS set and tables showing the allelic distribution for each genetic marker that has been typed in each of the recombinant congenic strains. Each row represents the typing for a genetic marker and each column represents alleles carried by a specific recombinant congenic strain of the RCS set. Format of the report is very similar to that of the Composite Recombinant Inbred Strains distribution patterns.

  • Chromosome Committee Reports

    The HTML versions of the 1995 Chromosome Committee Reports are now complete.

  • 1996 Chromosome Committee Chairs

    The names and addresses for the 1996 Chromosome Committee chairs are now available.

  • Link to RATMAP via homologies

    MGD now links to RATMAP, http://ratmap.gen.gu.se, a "database covering genes physically mapped to chromosomes in the laboratory rat," maintained by the Department of Genetics, Goteborgs Universitet, Sweden.

  • Back-end changes

    The software group has implemented a number of changes to the database structure and software to enhance the efficiency of the system.
