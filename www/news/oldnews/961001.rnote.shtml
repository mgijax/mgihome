MGD 3.1 Update Release Notes
October 1, 1996

NOTE: If you have used MGD recently, be sure to clear your browser cache before attempting to use the updated MGD.

Mouse Nomenclature Guidelines & Locus Symbol Registry

The Mouse Nomenclature Guidelines and Locus Symbol Registry page provides contact information for the MGD Mouse Nomenclature Committee and links to a membership list of the International Committee on Standardized Genetic Nomenclature for Mice.

A form is now available which enables you to submit a proposed locus symbol via E-mail. Instructions for doing so are provided in the Nomenclature Checklist. In addition, you can select links to various databases to check current information on existing symbols.

The Mouse Nomenclature Rules and Guidelines document has been edited and reformatted to make it much easier to use. A table of contents has been added.

New full-text search engine: Glimpse

Glimpse replaces WAIS as the full-text searching engine in MGD and the GXD Index. This change affects the following forms:

  • Reference and GXD Index query forms - Abstract field
  • Genes, Markers, and Phenotypes query form - Phenotype (MLC) search term field
  • M. Festing's Inbred Strains query form
  • MLC/OMIM query form

Glimpse enhancements to full-text searching in MGI resources:

  • Glimpse retrieves ALL records matching the search criteria
    Previously, a WAIS full-text search would discount some records and not return them in a search. Glimpse searches the entire data set and retrieves all records containing the terms in the search string.

  • Glimpse sorts results alphabetically by symbol or by author
    Glimpse does not score or rank results according to relevance. Query results are displayed as a numbered list sorted alphabetically by Author or Symbol.

    There are exceptions to this. When a search uses a full-text search field in combination with other fields on the query form, the sort options selected for the form override glimpse sorting. For example, on the Reference query form, if you search for all records where 

    	Title CONTAINS serological
	AND
	Phenotype (MLC) search term = blood 
	and you select to 
	SORT by J number
    the results will be sorted by J number and not by Author.

    In MLC/OMIM searches, OMIM searches are managed by the IRX full-text search engine and OMIM results are ranked by relevance.

  • Glimpse provides wildcard search option
    Glimpse uses the asterisk(*) wildcard which can be placed at the end of a partial term, or inside a search term. For example, "membran*" or "myelo*sis."

    WAIS, the search engine used previously to Glimpse, used word-stemming so that, if you entered a term such as "membrane," the search would retrieve records that contain "membrane," "membranous," "transmembrane." Glimpse does not do wordstemming but relies on the wildcard to retrieve the various spellings of a term.

NOTE: Be sure to read updated help text on full-text searching in MGD and GXD.

Chromosome Committee Reports

Some of the 1996 Chromosome Committee Reports are now available on-line. In addition, a new editing interface will enable the Chromosome Committees to submit and update their files electronically. This form requires a username and password and is accessible by members of the Chromosome Committees.

Changes to Query Forms and Web Pages

  • The MGD Home Page has been revised with some links reworded to reflect changes in MGD and a section has been added with links to user documentation.
  • Throughout MGD, the headers of forms and documentation have been simplified and standardized.
  • On all query forms, the Do Query button is now the Retrieve button.
  • Mammalian Homology Query Form
    Two formatting options for query results have been removed:
    • Tabular
    • Pre-formatted textual table

    The new format and sort options are
    • Format by Species and Marker, or, by Reference
    • Sort by Marker in primary species or Chromosomal location in primary species

    Changes to homology query results:
    • Results with "unofficial" symbols (format is *symbol*) will not show those symbols as 'linked' to any database.
    • Offsets "-999" will no longer appear

  • Probes and Clones Query Form (formerly, the Probes query form)
    • Clone name field changed to Probe/Clone name
    • EST probes searchable using;
      Sequence type = EST
    • Sequence type is now a selection list which includes:
      All, cDNA, EST, Genomic, Hybrid, Mitochondrial, oligo, RNA, unspecified
  • Genes, Markers and Phenotypes Query Form
    (formerly, the Genetic Markers and Mouse Locus Catalog query form)
    • MLC search term field renamed Phenotype (MLC) search term

  • Encyclopedia Home Page
    Links to static linkage maps placed under an Archive because of the age of the files.
  • Maps and Mapping Data
    • Backcross Mapping Panel Data Sets renamed DNA Mapping Data Sets
    • Marker Mapping Data Query Form renamed Genetic Mapping Data Query Form
    • On the Composite recombinant Inbred Strains Data Sets Query Form, the selection list of RI Strains designations has been alphabetized.
    • The link to "Backcross Mapping Panel Dat Sets" has been renamed "DNA Mapping Panel Data Sets."
    • The following DNA Mapping Panel data sets have been updated:

      • EUCIB (BSB) (C57BL/6J x SPR or SEG/Pas)F1 x C57BL/6J
      • EUCIB (BSS) (C57BL/6J x SPR or SEG/Pas)F1 x SPR or SEG/Pas
      • Copeland-Jenkins (C57BL/6J x M. spretus)F1 x C57BL/6J

      New data set:
      • MIT (C57BL/6J-ob/ob x CAST)F1 x (C57BL/6J-ob/ob x CAST)F1
