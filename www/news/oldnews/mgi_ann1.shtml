Mouse Genome Informatics (MGI) Release 1.0 Announcement
February 13, 1998

The MGI Web site provides integrated access to various information resources on the genetics and biology of the laboratory mouse, including the Mouse Genome Database (MGD), the Gene Expression Database (GXD) and the Encyclopedia of the Mouse Genome (Encyclopedia).

New Features

Reorganization of the Web Interface

A new MGI home page consolidates access to MGI data resources. A set of menus organizes searching options under various biological topics. A Query Menu window, displayed separately from the main browser window, provides direct links to all MGI query forms.

Searches, Data and Reports
The Searches, Data and Reports section of the MGI home page includes a list of biological topics which link to a new type of menu. When you select a topic and open a menu, all the available options for retrieving information related to the selected biological area are displayed. For example, if you select the link to Genetic and Phenotypic Data, you will see a list of the query forms (each with a brief description) that enable you to search for information on markers and phenotypes, a list of relevant database reports available; and a list of related resources.

Query Menu
While the new menus described above organize database access with respect to biological topics, a new Query Menu provides direct links to the MGI query forms. When you open the Query Menu, a small window is displayed on your desktop separate from the browser window. This menu shows the list of MGI query forms. When you click on a form title, the form opens in the browser window from which you selected the Query Menu. (Note: This feature works best with Java Script enabled.)

cDNA and EST Expression Query Form

This query form enables you to search for cDNA and EST expression information related to specified genes and origin (tissue, age, cell line).

Oxford Grid Query Form

A new Oxford Grid form now provides the option to retrieve an Oxford Grid display of homology between the genomes of two selected species. The Oxford Grid display also provides links to graphical comparative maps showing either a complete map or a region of a mouse chromosome with homologies from the selected comparison species. (This feature only applies to queries which include mouse as a species for comparison.)

Enhancements

Query Forms Reformatted

The form fields have been reformatted to accommodate differences between Netscape and Internet Explorer. In addition, selection lists on several query forms now have a default value of ANY to accommodate Internet Explorer's implementation of selection lists.

Changes to Mammalian Homology Query Form and Query Results

You can now search for homologies using multiple gene symbols. Homology query results have been reorganized to have a more compact, tabular format with a link to References appearing at the bottom of each displayed homology class. In addition, a link to a graphical Comparative map is shown for queries that specify mouse as one of the species to compare. Each comparative map shows the region of a mouse chromosome within 2 cM of the selected mouse marker and shows any homologies in that region for the comparison species.
