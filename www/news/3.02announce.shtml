MGI Release 3.02 Announcement - November xx, 2004  

Version 3.02 of MGI provides the scientific community with data for all mouse cDNA clones from the IMAGE consortium and the National Institute on Aging (including the NIA 15K and NIA 7.4K sets). Clone data, which include EST sequence associations and clone library source information, is mapped to MGI controlled vocabularies. These additions increase the number of cDNA clones in MGI as follows:

cDNA Previous Current
Molecular clones 886,3302,042,369
Sequence/molecular clone associations 598,269 2,067,132
Marker/molecular clone associations 220,033 861,087

You can select the clone sets using the Clone Collection field on these MGI query forms:

  • Molecular Probes and Clones Query Form
  • Mouse Sequence Query Form
  • cDNA Clone Query Form
  • Genes and Markers Query Form
