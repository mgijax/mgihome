MGI Release 3.02 Announcement - November 23, 2004  

Version 3.02 of MGI provides the scientific community with data for all mouse cDNA clones from the I.M.A.G.E. Consortium and the National Institute on Aging (including the NIA, NIA 15K and NIA 7.4K sets). Clone data, which include EST sequence associations and clone library source information, are mapped to MGI controlled vocabularies. These additions increase the number of cDNA clones in MGI as follows:

  cDNA Previous cDNA Current
Molecular clones 886,330 2,042,369
Sequence/molecular clone associations 598,269 2,067,132
Marker/molecular clone associations 220,033 861,087

MGI also contains data for mouse cDNA clones from the RIKEN (FANTOM) and RPCI-23 and -24 sets. We anticipate adding additional clone collections over time. Please contact User Support to request a specific clone collection.

You can select the clone sets using the Clone Collection field on these MGI query forms:

  • Molecular Probes and Clones Query Form
  • Mouse Sequence Query Form
  • cDNA Clone Query Form
  • Genes and Markers Query Form
For additional information, see Using the Clone Collection Field on MGI Query Forms.
