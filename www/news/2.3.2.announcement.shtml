MGI Release 2.3.2 Announcement

March 15, 2000

New MGI Home Site
The MGI home site includes a new MGI Home Page which provides access to items that are stored at this site and not included at MGI mirror sites. The MGI Home Page enables us to promote new additions to MGI resources and services between major releases of MGI.

Updated MGI Main Menu
Because we have reorganized the MGI web interface somewhat, we present a revised main menu. The primary URL for MGI, http://www.informatics.jax.org, points to this main menu. You will see a similar page when you visit mirror sites.

What's New?
This page, accessible from the MGI main menu, gives a brief description of any new developments in MGI. This page is subject to change between releases so it is not a bad idea to check this page now and then.

Genealogy Chart of Inbred Strains
The Genealogy Chart of Inbred Strains graphically depicts the origins and relationships of inbred mouse strains. The chart is a PDF file you can download to your machine for viewing. A "Key to the Chart of Strain Genealogy" provides a list of the strains and includes, for each strain, the location coordinates on the Chart, a reference for its genealogy, and an indication whether it is extant.

Web Interface to MGI-LIST and RAT-LIST
Several months ago, we adopted the Lyris listservice to manage mgi-list and rat-list. A web interface to MGI lists is now available. You can perform any subscriber operations through this interface including read or send mail, unsubscribe, change your subscriber settings. Most importantly, the web interface provides user-friendly options for searching the list archives.

Other Changes and Enhancements

  • You are encouraged to submit your Rat data to MGI.
    The Data and Nomenclature Submission pages have been updated to support submission of rat data.

  • Italic formatting removed from marker symbols in the web interface
    Although nomenclature guidelines dictate that marker symbols should appear in italics, we have removed italics from marker symbols and some other elements in our web interface for the sake of readability. In hardcopy publications, marker symbols should still appear italicized.

  • Maps and Mapping Data Menu provides links to RH Mapping Resources
    These resources include the Jackson Laboratory Radiation Hybrid Database and the EBI Radiation Hybrid Database, and the Whitehead Mouse RH Database at MIT.

  • You can now search for mapping data by marker name.
    The Mapping Data query form has a Marker Name field. You can enter some characters from a marker name and search for mapping data related to any markers that containthat string of characters in the marker name.

  • Homology Information on a Reduced Number of Mammalian Species.
    Previously, MGI contained information on over 70 mammalian species. However, there has been very little information available for most of these species. Now, MGI concentrates on the set of mammalian species of most interest to researchers and for which the homology data are growing.
