Gene Ontology (GO) Project Vocabularies

The GO project is a collaborative effort to develop controlled vocabularies to exhaustively describe biological features of the genome of various species. MGI has taken a first step in using these vocabularies to annotate gene records in the MGI database. In MGI 2.4, you can search for genes, gene expression data, or cDNA clones, and focus these searches on genes annotated using these terms. MGI is changing the way alleles are handled in the database. These changes are part of an overhaul in the way MGI manages phenotypic information. In MGI 2.4, some of the groundwork has been done resulting in a new Alleles query form and a new display of detailed information on alleles. See, for example, the detail record for Leprdp- ad, an allele of Lepr.

Symbol/Name Combined Field

In response to user requests, we have replaced the two fields, marker Name and Symbol, with a single marker Symbol/Name field on most MGI query forms. On any MGI form with this new field, you can enter one or more symbols or names (each separated by a comma) and the search will find records for any markers with symbols or names matching the text entered in the field. Field operators (BEGINS, CONTAINS, ENDS, "=") determine how the comparison is made in a search. See, for example, the Genes, searches/marker_form.shtml">Markers and Phenotypes query form.

Search for Molecular Segments Mapped to a Chromosomal Region

In response to user requesta, the option to search for molecular segments mapped to a region of a chromosome has been added to the Molecular Probes and Segments and cDNA Clones query forms.

Additional Enhancements

  • Map positions for QTL markers
    The linkage positions for QTLs are typically vague. QTLs in MGI no longer include an assigned cM value. Instead, "QTL" appears in the cM position column for QTLS included in a list of mouse markers.

  • Revised semantics for withdrawn symbols, aliases, alleles, synonyms
    The term "alias" used to apply to segments that were known to overlap other mouse markers. When an MGI search yielded a list of markers, if there were any aliases, these would also be included, even if you had restricted your search to genes only. This has been changed so that "STS" replaces "alias" and STSs are treated as separate markers.

    "Synonym" replaces the nomenclature category "Other name". A synonym is any symbol or name for a mouse marker that has appeared in the scientific literature but has never been an approved symbol or name for the marker.

  • References for Sequence IDs
    A MGI marker detail record may include a list of Sequence IDs. These IDs are usually associated with marker records via molecular segments which have been mapped to the the marker. If sequence information for the segment is available in GenBank, MGI provides the Sequence ID. Now, we also provide an MGI reference ID in parentheses right next to the Sequence ID. This is a link to the supporting reference for the association of marker and segment.
