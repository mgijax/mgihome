MGI 3.51 Release Announcement - October, 2006  

MGI is pleased to announce its 3.51 release.

  • In this release, mouse genome coordinates are updated to NCBI Mouse Build 36 and dbSNP Build 126. Data from the manual annotation of the mouse genome is provided by Vega Mouse (Vertebrate Genome Annotation of the mouse genome) at Sanger Institute; computational annotation is from Ensembl and NCBI.

  • dbSNP Build 126 now contains SNPs that map to the Y chromosome. dbSNP is the official source of SNP-based mouse genomic variation in MGI and serves as a central repository for single nucleotide polymorphisms (SNPs), multiple nucleotide polymorphisms (MNPs)m and short insertion/deletion polymorphisms (IN-DELs).

  • MGI integrates the data from VEGA, NCBI, and Ensembl into a single set of distinct genes, pseudogenes, and other genomic features to create a Unified Gene Catalog. The 3.51 release adds 6,496 genes (a 5% increase), 4,615 pseudogenes (a 400% increase), and increases the number of refSNPs (from dbSNP) from 1.8 to 6.5 million.

  • MGI 3.51 also adds, for the first time, physical map data for microRNAs (from mirBASE) as well as coordinates for QTLs. Other changes to MGI data are that multiple gene models are now possible for a single MGI gene record and multiple MGI genes may be associated with a single gene model.

  • Mouse GBrowse, MGI's interactive sequence map, is updated to Build 36 and now contains a microRNA track from mirBASE as well as tracks for SNPs, QTLs, Ensembl genes, NCBI genes, VEGA genes, MGI representative transcripts, STS markers, phenotypes, and alleles. Access to Mouse GBrowse is from www.informatics.jax.org and from the sequence map section on MGI gene detail pages.
