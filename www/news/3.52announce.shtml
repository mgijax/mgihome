MGI Release 3.52 Announcement - March, 2007  
MGI is pleased to announce the release of Version 3.3 of the database. Version 3.3

MGI is pleased to announce the release of Version 3.52 of the database.

  • MGI's 3.41 release features a new Gene Expression Data Images Query by Gene summary page.

  • The release also features new cross-reference links to TreeFam. TreeFam is an ortholog database with curated information about ortholog and paralog assignments and the evolutionary history of various gene families.
    • You can access TreeFam reports using the MGI Marker Query Form.
      1. Enter Cdc2a in the gene/marker symbol field.
      2. Click Cdc2a on the Genes and Markers Query Results -- Summary report. The Gene Detail report for Cdc2a appears. The TreeFam link is in the Mammalian Homology section.
    • You can also access TreeFam reports using the Mammalian Orthology Query Form.
      1. Under Primary Species, select Mouse.
      2. Under Marker Symbol/Name, enter Cdc2a.
      3. Under Comparison Species, select Human (or leave the default, Any.) The Mammalian Orthology Query Results report appears. Locate the Links section in the middle of this report. When available, the link to the TreeFam report for this gene appears here.




  • The remainder of the MGI 3.41 release consists of performance enhancements and bug fixes.
