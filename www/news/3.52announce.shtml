MGI Release 3.52 Announcement - March, 2007  

MGI is pleased to announce the release of Version 3.52 of the database.

  • The 3.52 release features a new Gene Expression Data Images Query by Gene Summary report which displays all the expression images associated with a given gene/marker. Images are individual, related panes collected in figures. You can elect to view 10, 25, 50 or all of the figures on the same page.
    You can report from the MGI Gene Detail page. For example:
    1. Enter Pax6 in the Gene/Marker Symbol/Name field of the MGI Marker Query Form and click Search.
    2. Click Pax6 on the Genes and Markers Query Results -- Summary report. The Gene Detail report for Pax6 appears.
    3. Click the Images link in the Expression section. The Gene Expression Data Images Query by Gene Summary appears.

  • The release also features new cross-references to TreeFam. The Tree Families database is an ortholog database with curated information about ortholog and paralog assignments and the evolutionary history of various gene families.

    You can access TreeFam reports using the MGI Marker Query Form. For example:
    1. Enter Cdc2a in the Gene/Marker Symbol/Name field.
    2. Click Cdc2a on the Genes and Markers Query Results -- Summary report. The Gene Detail report for Cdc2a appears. The TreeFam link is in the Mammalian Homology section.

    You can also access use the MGI Mammalian Orthology Query Form to access TreeFam reports. For example:

    1. Under Primary Species, select mouse, laboratory (Mus musculus domesticus).
    2. Under Marker Symbol/Name, enter Cdc2a.
    3. Under Comparison Species, select Human (or leave the default, Any.) The Mammalian Orthology Query Results report appears.
    4. Scroll down to locate the Links section in the middle of this report. When available, the link to the TreeFam report for this gene appears at the top of this database list.
  • The remainder of the MGI 3.52 release consists of performance enhancements and bug fixes.
