MGI Release 3.52 Announcement - March 28, 2007  

MGI is pleased to announce the release of Version 3.52 of the database.

  • The 3.52 release features a new Gene Expression Data Images Query by Gene Summary report which displays all the expression images associated with a given gene/marker. Images are individual, related panes collected in figures. You can elect to view 10, 25, 50 or all of the figures on the same page. Access to the report is from the MGI Gene Detail report.
    1. Enter Pax6 in the Gene/Marker Symbol/Name field of the MGI Marker Query Form and click Search.
    2. Click Pax6 on the Genes and Markers Query Results -- Summary report. The Gene Detail report for Pax6 appears.
    3. Click Images in the Expression section. The Gene Expression Data Images Query by Gene Summary appears.

  • The release also features new cross-references to TreeFam, an ortholog database containing curated information about ortholog and paralog assignments and the evolutionary history of various gene families.
    • Access to TreeFam reports is from the Genes and Markers Query Form. The TreeFam link (if available for a given gene) appears in the Mammalian Homology section.
    • You can also access use the Mammalian Orthology Query Form to access TreeFam reports. The TreeFam link (if available for a given gene) appears under Links in the center of the query results report.

  • The remainder of the MGI 3.52 release consists of performance enhancements and bug fixes.

The GXD (Gene Expression Database) group is pleased to announce a new version of the Gene Expression Notebook (GEN).
