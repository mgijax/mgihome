September, 2010
 
For the 4.4 release, a new area on the Genes and Markers Query Form lets you focus your search to protein-coding genes, non-coding RNA genes, heritable phenotypic markers, or a variety of feature subcategories. For more information, see Genome Feature Type Definitions.
June 18, 2010
 
For the 4.35 release, the MGI Quick Search returns the alleles most closely associated with your query in the Genome Features list on the results page. As an example, type wavy in the Quick Search box and note the various types of phenotypic allele that this query returns.
May 14, 2010:
 
For the 4.34 release, MGI is retiring several query forms. See FAQ for alternative ways to find the same information on the MGI site.
March 31, 2010:
 
MGI now represents all alleles from KOMP and EUCOMM; for example, see Aamptm1a(KOMP)Wtsi and Zap70tm2e(EUCOMM)Hmgu

An option to direct the output of your query to Excel is now available from the following:

  • Genes and Markers Query Form
