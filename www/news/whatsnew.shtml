Past Release Announcements

May, 2007

The MGI 3.54 release features a new MGI Batch Query form for downloading bulk data for gene identifiers or symbols. You can customize the results so that data appear in web or tab-delimited format..

April 18, 2007

The MGI 3.53 release associates mutant cell line IDs with a given gene/marker. Any gene traps found for that marker appear in the MGI Gene Detail report.

April 4, 2007

For the MGI 3.52 release:
  • A new GXD report features expression images associated with a given gene/marker.
  • MGI Mammalian Homology and Gene Detail reports now provide links to the TreeFam database.
  • There is a new version of the Gene Expression Notebook (GEN).
    • November 22, 2006
  • MGI markers now link to computationally derived transcript assemblies from DFCI (Dana Farber Cancer Institute) Mouse, Human, and Rat Gene Indices (developed originally at TIGR).
  • Associations between MGI markers and assemblies are based on shared GenBank sequence accession identifiers.
  • MGI has also loaded new sequences from DFCI.
  • Links are updated weekly as well as after every major change in MGI databases.
