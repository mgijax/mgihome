News and Announcements for 2002

Last Updated April 1, 2002

April, 2002

  • MGI Tutorials and FAQs
    Tutorials are now available on topics such as finding a gene, exploring homology, getting information on gene expression, and navigating MGI. There is also a link to a Frequently Asked Questions (FAQ) page.

  • Recent Publications
    The following staff publications derive from recent work on Mouse Genome Informatics projects.
    Eppig JT,. Blake, JA, Burkhart DL, Goldsmith CW, Lutz CM, Smith CL. 2002. Corralling conditional mutations: a unified resource for mouse phenotypes. Genesis 32:63-65. MEDLINE

    Begley DA, Ringwald M. 2002. Electronic tools to manage gene expression data. Trends Genet 18:108-110.
    March, 2002

  • MGI Release 2.8 is now available: March 21, 2002.


    February, 2002

  • Issue #3 January 2002 of the  Mouse Genome Monthly Newsletter is now available online.

    January, 2002


  • UCSC Human Genome Browser, January 23, 2002
    There is now a link from human genes represented in the Browser to gene records of mouse orthologs in MGI. Thanks to David Haussler and Jim Kent for adding this link to the UCSC resource.

  • Glycine Receptor, Alpha Family joins the Gene Family Reports, January 23, 2002.

  • An article describing resources available through the Mouse Genome Informatics web site appears in the database issue (January 2002) of Nucleic Acids Research (NAR).
    Blake JA, Richardson JE, Bult CJ, Kadin JA, Eppig JT and the Mouse Genome Database Group. 2002. The Mouse Genome Database (MGD): The Model Organism Database for the Laboratory Mouse. Nucl Acids Res 30: 113-115.
