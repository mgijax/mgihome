July 7, 2011
The MGI implementation of BioMart now includes gene expression data from the Gene Expression Database (GXD).
  • BioMart is a query-oriented data management and integration system that performs like a batch query. You can use the MGI BioMart to find expression data for a list of genes or for specific anatomical structures or Theiler Stages. You can manipulate the fields you see in your results and choose to return your results as HTML or comma or tab-separated values.
  • Batch searches for gene expression data are also possible through the MGI Batch Query Form.
June 24, 2011
The MGI database is available in 3 new ways: public access PostgreSQL server as well as database dumps for MySQL and PostgreSQL.
  • The MGI Public Ad Hoc PostgreSQL Server is available for running custom SQL queries against the MGI database. Account information can be found here.
  • MGI now offers MySQL and PostgreSQL database dumps via FTP. FTP site details and loading instructions are available here.
April 13, 2011
The Gene Expression Database (GXD) is pleased to announce that in addition to the recently added Eurexpress data (J:153498), we have now included RNA in situ hybridization data from the Brain Gene Expression Map (BGEM) database. You can view the data set at accession number J:162220. This set includes 53,496 results for:
    1,451 genes
    1,474 assays (RNA in situ hybridizations at E11.5, E15.5, P7 and P42)
    16,112 images
Some useful links for querying subsets of data in GXD and MGI are:
  • Gene Expression Data Query Form
  • Genes and Markers Query Form
If you are interested in submitting your data to GXD, please see our guidelines for submitting expression data to the Gene Expression Database or contact GEN@informatics.jax.org for further information.
