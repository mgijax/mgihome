April 15, 2011
The Gene Expression Database (GXD)is pleased to announce that in addition to the recently added Eurexpress data (J:153498), we have now included RNA in situ hybridization data from the Brain Gene Expression Map (BGEM) database. You can view the data set at accession number J:162220. This set includes over 53,475 results for:
    4,820 genes
    1,471 assays (RNA in situ hybridizations at E11.5, E15.5, P7 and P42)
    16,065 images
Some useful links for querying subsets of data in GXD and MGI are:
