What's New?

Last Updated December 4, 2000

Late November: Gene Families

New members of gene families are being added rapidly. Mouse Genome Informatics Nomenclature Committee and HUGO are working with researchers to organize and update nomenclature on important gene family groups. You can now access information about these gene families in MGI. See Links to Nomenclature Web Resources for Gene Families.

Late October: Allele and Mutant Data

The new Allele and New Mutant Submissions Form is available for contributing data to MGD. Information can be submitted for new or known alleles of existing genes or alleles where a phenotype has now identified a new gene. Help with nomenclature is provided from linked pages. Each new allele or mutant receives a permanent database accession ID. Mammalian Genome periodically publishes a list enumerating the data electronically submitted to MGD, including accession ID, chromosome, and locus and allele symbols as a cross-referencing service for readers. For submission of other types of data to MGD, see the Data and Nomenclature Submissions page.

Mid-October: MGI Release 2.5 Announcement

MGI Release 2.5 is now available. Information on other recent MGI releases is also available.

New MGI Vocabulary Browsers

You can now view and query the following structured vocabularies:

  • ">Gene Ontology (GO) Browser

    The GO Browser associates each ontology term with a precise definition of that term, complete with citations and synonyms. Each ontology is organized as a Directed Acyclic Graph (DAG). Terms in this hierarchy are organized from the general to the specific and you can immediately visualize their relationship. Although the structure of the DAG is "tree-like," a term may actually have more than one parent and appear several times in the browser. A given term may occur in any or all of the following categories:

      GO.cgi?id=GO:0003674"> Molecular Function - Tasks performed by individual gene products. Examples: transcription factor; DNA helicase.
      GO.cgi?id=GO:0008150"> Biological Process - Broad biological goals accomplished by ordered assemblies of molecular functions. Examples: mitosis; purine metabolism.
      GO.cgi?id=GO:0005575"> Cellular Component - Subcellular structures, locations, and macromolecular complexes. Examples: nucleus; telomere; origin recognition complex.

  • searches/anatdict_form.shtml">Mouse Anatomical Dictionary

    The Anatomical Dictionary is a vocabulary comprised of 28 Theiler stages. Each Theiler stage is a tree which is a subvocabulary of the Dictionary. There are brief stage descriptions next to each stage name. You can select one or more of the Theiler stages to focus a search on a particular stage of embryonic development

Homology Maps

This release includes a new way of viewing and accessing homology data. Two new maps are available showing global views of human and rat orthology data plotted against the mouse genome. Beside each chromosome is a set of histograms, each corresponding to a human or rat chromosome. The histogram effectively shows the density of known orthologies in a particular mouse chromosomal region. As always, the maps are active.
  • You can click on a mouse chromosome to retrieve a detailed linkage map showing all orthologs.
  • You can click on a histogram to retrieve a compact tabular listing of orthologs for that part of a chromosome.

The following whole genome homology maps are available:

  • reports/homologymap/mouse_human.shtml"> human on mouse background
  • reports/homologymap/mouse_rat.shtml"> rat on mouse background

Expanded MGI Glossary

We have expanded the userdocs/glossary.shtml"> MGI Glossary. The Glossary contains terms useful in understanding mouse genetics. You can find terms defined in a general sense as they apply to eukaryotic genetics, particularly mouse genetics. You will also find some terms defined as they apply to MGI (for example, gene). The list of terms included in the Glossary will grow with each new release.

Mouse/Drosophila Homology Report
