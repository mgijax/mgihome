Combined Name/Symbol Search Field On MGI Query Forms

The marker symbol and marker name fields on MGI query forms have been combined in one field, Symbol/Name. When using this field, your search (by default) spans current symbols or names, withdrawn symbols or names, and synonyms. You have the option to limit the search to current symbols only (no search of names), current symbols and names only, or to current and withdrawn symbols or names.

You can enter one or more symbols or names in this field. The search will retrieve any records for markers having names or symbols that match the characters entered on the Symbol/Name field.

The combined Symbol/Name field appears in the following query forms:

    Genes, Markers and Phenotypes
     Alleles
     Polymorphisms
    Molecular Probes and Segments
    Mammalian Homology
    Mapping Data
    Linkage Maps
    GXD Index
    GXD Data
    cDNA Clone

Gene Ontology Project

The goal of the Gene Ontology (GO) Project is to produce controlled vocabularies that can be applied to all eukaryotes as knowledge of gene and protein roles in cells is accumulating and changing. MGI is committed to annotating gene records using GO vocabularies. To date, close to 5000 genes in MGI have been annotated. GO classification fields have been added to the following MGI query forms: Genes, Markers and Phenotypes, Gene Expression Data, cDNA Clone and Linkage Maps.

Visit the Home Page of the Gene Ontology Consortium for more information.

Search Molecular Probes and Segments by Chromosome Region

You can now limit your search for molecular probes and segments to a specified chromosomal region. You also have the option to return results from this form as tab-delimited text.

Primary Reference For Sequence IDs

An MGI marker record may include a list of related Sequence IDs. Most often, this association is made via molecular segments whose sequence is known and which have been mapped to the given mouse marker. This association of marker and sequence is reported in the literature. Now, the reference supporting this association is displayed next to the Sequence ID in the marker detail record.
