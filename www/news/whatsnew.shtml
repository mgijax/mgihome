April 1, 2010: MGI announces the following enhancements for the 4.33 release:
 
All alleles from KOMP and EUCOMM are now represented in MGI.
Gene detail pages may now include (for a given marker):

  • Vega and Ensembl transcript/protein sequences;

  • links to gene model evidence;

  • links to protein ontology annotations;

  • notes for any MGI marker associations to VEGA, Ensembl or NCBI gene models with gene/pseudogene discrepancies (a new report provides details);

  • strain specificity information (for example, a notation that a gene is not present in a given strain).
An option to direct query output to Excel is now available from the following:
  • Genes and Markers Query Form
  • Mouse SNP Query Form
  • Mouse Sequence Query Form
  • MGI Batch Query
Additionally, the MGI Batch Query can now handle very large files.
