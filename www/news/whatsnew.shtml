August 6, 2007

A revised version of the Rules and Guidelines for Nomenclature of Mouse and Rat Strains is now available. Most notably, the International Committee on Standardized Nomenclature for Mice added a section on nomenclature for Outbreds and Closed Colonies. Previous versions of the official mouse nomenclature guidelines are archived for your reference.

June 14, 2007

The MGI 3.54 release features a MGI Batch Query tool to use for downloading bulk data for gene identifiers or symbols. You can select the type of gene/marker data to return and customize the appearance of any results into web or tab-delimited format.

April 18, 2007

The MGI 3.53 release associates mutant cell line IDs with a given gene/marker. Any gene traps found for that marker appear in the MGI Gene Detail report.

April 4, 2007

For the MGI 3.52 release:
  • A new GXD report features expression images associated with a given gene/marker.
  • MGI Mammalian Homology and Gene Detail reports now provide links to the TreeFam database.
  • There is a new version of the Gene Expression Notebook (GEN).
