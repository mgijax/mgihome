March 31, 2010:
 
MGI now represents all alleles from KOMP and EUCOMM; for example, Aamptm1a(KOMP)Wtsi and Zap70tm2e(EUCOMM)Hmgu

For the 4.33 release, gene detail pages may now include (for a given MGI marker):

  • A note when there are gene/pseudogene discrepancies between VEGA (e.g. Amy2b), Ensembl (e.g. Cecr6, or NCBI (e.g. Cdc34-ps) marker associations.   A new report provides details);

  • Vega and Ensembl transcript and/or protein sequences (see Sox2, Sequences);

  • links to gene model evidence (see Sox2, Other database links)

  • links to protein ontology annotations (see Sox2, Protein-related information).

An option to direct the output of your query to Excel is now available from the following:

  • Genes and Markers Query Form

  • Mouse SNP Query Form

  • Mouse Sequence Query Form

  • MGI Batch Query

Additionally, the MGI Batch Query can now handle very large files.
