March 31, 2010:
 
MGI now represents all alleles from KOMP and EUCOMM.
For the 4.33 release, gene detail pages may now include (for a given marker):
  • notes identifying any MGI marker associations to VEGA, Ensembl, or NCBI gene models with gene/pseudogene discrepancies (a new report provides details);
  • strain specificity information (for example, a notation that a gene is not present in a given strain);
  • Vega and Ensembl transcript and/or protein sequences;
  • links to gene model evidence;
  • links to protein ontology annotations.
An option to direct query output to Excel is now available from the following:
  • Genes and Markers Query Form
  • Mouse SNP Query Form
  • Mouse Sequence Query Form
  • MGI Batch Query
Additionally, the MGI Batch Query can now handle very large files.
