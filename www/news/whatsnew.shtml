Past Release Announcements

April 27, 2006
    The MGI 3.44 release features a new consists of performance enhancements, bug fixes, and updates.
February 16, 2006
  • MGI Release 3.43
    The 3.43 release features the "in sync" version of the Mouse SNP Query Form (genome coordinates for mouse genes and markers in MGI are derived from the same NCBI mouse genome build as the coordinates for mouse SNPs).
  • NIH Knockout Mice Repatriation Project
    The goal of this project is to make existing mouse strains carrying targeted knockouts more accessible to the scientific community.
December 15, 2005
  • New additions for MGI Version 3.41 include an electronic book, Origins of Inbred Mice, as well as performance enhancements, bug fixes, and updates.
  • A new version of the Gene Expression Notebook (GEN) was also released.
