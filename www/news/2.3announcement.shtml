Mouse Genome Informatics (MGI) 2.3 Release Announcement

November 29, 1999

Enhanced Display of Homology Information

The display of homology query results has been reformatted and now contains information on homology criteria and options for viewing references.

Homology information in MGD is extracted from the scientific literature. This information includes the type of experimental evidence used to support homology assertions. Now, MGD homology query results indicate the criteria for homology assertions. Some homologies are reported by a number of sources. There can be a number of different criteria shown for the same set of homologs. See, for example, the homology report on the Cftr gene. You will see a new Criteria column on the right showing, in this example, several values, AA (Amino acid sequence comparison), CL (Conserved map location) and NT (Nucleotide sequence comparison).

You will also see the Number of References associated with each criterion. The number is linked to the references. Beneath the tabular display of a set of homologies, the References line now provides options to view a list of either all the references for a set of homologies, or, view the references for particular species represented in the set.

New Options for Searching Polymorphism Information

The Polymorphisms query form now provides the option to search for differences between mouse strains. You enter one or more strains in the Primary strains field, enter one or more strains in the Comparison strains field, and then execute the search to find the differences between the primary and comparison strains.

You can now search for polymorphism information in a region of a chromosome by specifying the end-points of the region. You can search for polymorphisms in the area between two markers (using marker symbols), between two cM positions, between a marker and a cM position, or within a specified number of cM around a locus.

Mouse Gene Records Now Associated with Unigene Clusters

Previously, MGD showed putative associations between mouse genes and ESTs based on information in dbEST records. Now, MGD shows putative associations between genes and I.M.A.G.E. clones based on information in Unigene clusters. In many cases, a MGD mouse gene detail record will include a link to a Unigene cluster in the "Other Database Links" section of the record. See, for example, detail record for Pax3

MouseBLAST Prototype

MouseBLAST enables you to compare a DNA or protein sequence against sequence data in one of several rodent sequence databases (all data representing subsets of either GenBank, TrEMBL, or SwissProt). Matching sequence records may include links to GenBank or SwissProt. If a retrieved sequence is associated with a mouse gene, the sequence record will include a link to the appropriate marker detail record in MGD.

MouseBLAST provides a simple search form for searching by sequence or Sequence ID (from GenBank, EMBL, SwissProt, TrEMBL, or GenPept). MouseBLAST relies on several WU-BLAST programs for searching DNA or protein sequences. (WU-BLAST is an implementation of BLAST).

Enhancements to the Mouse Tumor Biology Database Prototype

The Mouse Tumor Biology Database (MTB) prototype provides access to descriptions of tumors and tumor cell lines; tumor pathology reports and images; information on genetic factors associated with tumors and tumor development; tumor incidence and latency data; links to related resources.

A new Tumor Frequency Grid (Inbred Strain x Organ) provides a summary of MTB Tumor Frequency and Incidence records for spontaneous tumors observed in particular strain and organ combinations. This summary, in turn, is linked to the sources of the values used to make this grid.

Also, a new column has been added to the Strain Results page providing links to external resources including the Biology of the Mammary Gland Web Site.
