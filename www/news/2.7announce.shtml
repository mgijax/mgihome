MGI Release 2.7 Announcement - May 4, 2001

MGI 2.7 features enhancements to the Allele query form.

  • There is new query field for Promoter Notes. You can use this field to search for when and where a transgene is expressed. The notes appear on the Allele detail page.
  • The Molecular Information (previously Molecular Description) section of the Allele Detail page may now include the following:
    • Mutation(s)
    • ES Cell Line (if applicable)
    • ES Cell Line Strain (if ES Cell Line appears)
    • Promoter
    • Notes (molecular)
    • Reference (molecular)
  • There is a new section for additional references on the Allele Detail page. The new reference is associated with a phenotypic allele used experimentally or included as a major part of a review article. The objective is to associate a particular allele with a set of publications. Previously, the page displayed three other kinds of references:
    • The Original Reference (one only) reports the creation or discovery of a spontaneous or genetically-engineered allele.
    • The Molecular Information reference (one only) describes or identifies the molecular nature of the mutation responsible for a given phenotypic allele.
    • Synonym references (one per synonym) contain an allele synonym.
  • Also on the Detail page is Gene Expression in this Mutant; beside it, the number of assays appears with a link to a display on the GXD Expression Summary page of all expression data assayed in mice carrying the allele.
  • The Allele detail page no longer displays the Mode of Inheritance if it is not applicable.
