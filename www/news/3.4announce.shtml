MGI Release 3.4 Announcement - November 1, 2005  

MGI 3.4 release

MGI is pleased to announce its 3.4 release comprised of a new Mouse SNP Query Form, a Protein Superfamily Vocabulary Browser, and Fantom GO annotation updates to RIKEN large-insert cDNA clones.

  • Mouse SNP Query Form

    • This release includes over 500,000 SNPs for 75 mouse strains from dbSNP build 125, mapped to Build 34 of the mouse genome sequence. You can:

    • use the new Mouse SNP Query Form to search dbSNP for SNPs by strains, strain comparisons, SNP attributes (variation type, functional class, accession ID), map position, marker range, or by association to MGI genes and genetic markers.
    • search for SNPs within specified distances from a gene of interest.

    Query results include a summary of each returned SNP, including consensus strain allele calls derived from SNP assays. You can:

    • choose an output strain sort order.
    • elect to see strain allele calls marked as Same or Different relative to a reference strain.
    • use the spreadsheet-compatible tab-limited option for graphical displays of SNP query results and for visualizing presumptive haplotype blocks.

    You can perform queries such as:

    • Show me all SNPs on Chromosome 1 between D1Mit30 and D1Mit37 that are different between C57BL/6J and DBA/2 mice.
    • Return all SNPs, searching A/J, C57BL/6J, DBA/2J, C3H/HeJ, and CZECHII/EiJ, for any marker symbol/name beginning with pax.
    • Compare A/HeJ, A/J, BALB/cJ, C3H/HeJ, SM/J, 129S1/SvImJ, AKR/J, DBA/2J, LP/J, NZB/BlNJ and NZW/LacJ strains with C57BL/6J, and, searching Chromosome 1, display all variation alleles with nonsynonymous substitutions.

  • Protein Superfamily Vocabulary Browser

    • This release also incorporates UniProt PIRSF protein classifications into a Protein Superfamily Vocabulary Browser for locating links between MGI mouse genes and the protein classes.

    • You can enter, for example, cadherin in the browser search box and click Search.
    • The resulting MGI Superfamily Protein Detail page lists all mouse superfamily members matching your query as well as any human and rat orthologs derived from MGI-curated mammalian orthology data.
    • The Detail page also provides options for downloading any available sequences in FASTA format or forwarding them to MouseBLAST.

  • FANTOM3 GO annotation updates

    • MGI recently (September 30, 2005) updated GO annotations to include all 102,801 large-insert cDNA clones generated by the RIKEN group in Japan and described in the journal, Science:

    Carninci et al., The Transcriptional Landscape of the Mammalian Genome. 2005. Science 309:1559-1563.

    These annotations have an RCA evidence code, indicating a reviewed computational analysis. As part of this process, the newer annotations replaced annotations assigned to genes during earlier phases of the FANTOM project. To reflect the process of annotation more accurately, annotations retained from earlier analyses also received the RCA evidence code.

MGI is the community model organism database for the laboratory mouse.
