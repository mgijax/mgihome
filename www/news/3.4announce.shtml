MGI Release 3.4 Announcement - October 26, 2005  

MGI 3.4 release

MGI is pleased to announce Version 3.4 of the database.

  • This release incorporates SNPs data from dbSNP at NCBI to include over n,nnn,nnn: need number? SNPs for nnn: need number? strains of mice mapped to Build 33: which build? of the mouse genome sequence. The Mouse RefSNPs Query Form lets you search for RefSNPs by strains, strain comparisons, RefSNP attributes, map position, marker range, or associated genes.

    You can perform a query such as "Show me all refSNPs on Chromosome 1 between D1Mit30 and DMit37 that are different between C57BL/6J and DBA/2 mice." Is there an example you prefer? You can view your results in web format or save them as tab-delimited files.

  • The release also incorporates UniProt PIRSF protein classifications into a Protein Superfamily Vocabulary Browser for locating links between MGI mouse genes and the protein classes. You can enter, for example, DNA repair in the browser search box and get back a list of at least 12 superfamilies containing this term and click your term of interest. The resulting MGI SuperFamily Protein Detail page lists all mouse superfamily members matching your query as well as any human and rat orthologs derived from MGI-curated mammalian orthology data.

FANTOM3 GO data loaded into MGI

MGI recently (September 30, 2005) updated GO annotations to include all 102,801 large-insert cDNA clones generated by the RIKEN group in Japan and described in a manuscript in the journal, Science:

Carninci et al., The Transcriptional Landscape of the Mammalian Genome. 2005. Science 309:1559-1563.

The newest set of annotations was loaded into MGI with an RCA evidence code, indicating a reviewed computational analysis. As part of this process, annotations assigned to genes during earlier phases of the FANTOM project were replaced with the newer annotations. Annotations retained from earlier analyses were also reassigned the RCA evidence code to reflect the process of annotation more accurately.

MGI is the community model organism database for the laboratory mouse.
