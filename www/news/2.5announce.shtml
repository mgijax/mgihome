MGI Release 2.5 Announcement - October 17, 2000  

New MGI Vocabulary Browser

The 2.5 release introduces a new browser for viewing and querying the following vocabularies:

  • Gene Ontology (GO)
  • Mouse Anatomical Dictionary

The browser links to detail pages for each entry in a vocabulary. The GO terms are grouped into three categories to search singly or in combination: molecular function, biological process, and cellular component. The Anatomical Dictionary links to each of the 28 Theiler stages. The Browser query field accepts any text string and searches for all terms containing that string and any synonyms. For example, entering "muscle" in the GO Browser and using the option to search all categories returns 2 molecular function, 15 biological process, and 3 cellular component terms including "muscle development," "non-muscle myosin," and "structural protein of muscle."

Homology Maps

Two new maps are available for viewing and accessing homology data. The maps display global views of human and rat orthology data plotted against the mouse genome. Beside each chromosome is a set of histograms, each corresponding to a human or rat chromosome. The histogram shows the density of known orthologies in a particular mouse chromosomal region. From this display, you can get detailed comparative maps showing all orthologs, and you can retrieve a compact tabular listing of orthologs for that part of a chromosome. The whole genome homology maps are:

  • human on mouse background
  • rat on mouse background

MGI Glossary

The MGI Glossary has expanded. It provides definitions of terms from our help documentation as well as those useful for understanding mouse genetics. If you find any terms that are not defined or have suggestions or requests, please contact User Support.

Mouse/Drosophila Homology Report

The list of similarity associations between mouse and Drosophila genes has greatly expanded. There are now 3,017 associations between mouse and Drosophila genes.
