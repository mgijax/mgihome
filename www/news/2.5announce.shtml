MGI Release 2.5 Announcement

New MGI Vocabulary Browsers

From the MGI Home Page, you can now view and query the following structured vocabularies:

  • GO/go_url.html">Gene Ontology (GO) Browser

    The GO Browser associates each ontology term with a precise definition of that term, complete with citations and synonyms. Each ontology is organized as a Directed Acyclic Graph (DAG). Terms in this hierarchy are organized from the general to the specific and you can immediately visualize their relationship. However, even though the structure of the DAG is "tree-like," a term may actually have more than one parent and appear several times in the browser. A given term may occur in any or all of the following categories:

    • GO.cgi?id=GO:0003674"> Molecular Function - Tasks performed by individual gene products.
      Examples: transcription factor; DNA helicase
    • GO.cgi?id=GO:0008150"> Biological Process - Broad biological goals accomplished by ordered assemblies of molecular functions.
      Examples: mitosis; purine metabolism
    • GO.cgi?id=GO:0005575"> Cellular Component - Subcellular structures, locations, and macromolecular complexes.
      Examples: nucleus; telomere; origin recognition complex

    As of September, there are approximately 4800 terms in the GO. In MGD, approximately 5300 mouse genes are associated with at least one GO term. You can use the GO Browser to query for MGI genes, gene expression data, or cDNA clones, and focus your searches on genes annotated using these terms.

  • Mouse Anatomical Dictionary

    The Anatomical Dictionary is a vocabulary comprised of 28 Theiler stages. Each Theiler stage is a tree which is a subvocabulary of the Dictionary. There are brief stage descriptions next to each stage name.

    • You can select one or more of the Theiler stages to focus a search on a particular stage of embryonic development
    • You can use terms from the Dictionary to describe source data for clones, probes and cDNA and genomic libraries, as well as morphological variation caused by mutant alleles.

Homology Maps

The homology maps are overviews of selected species orthology data plotted against the MGD consensus genetic map for the mouse. The following maps are available:

  • reports/homologymap/mouse_human.shtml"> Whole Genome Homology Map: human on mouse background
  • reports/homologymap/mouse_rat.shtml"> Whole Genome Homology Map: rat on mouse background

Beside each mouse chromosome is a set of histograms, each corresponding to the human or rat chromosome.

For each species, you can view a detailed homology map of each chromosome and you can retrieve a compact tabular listing of all the orthologies for that pair of chromosomes.
Each colored bar is a conserved region between a mouse and the selected species. The chromosome number for that species appears beside each bar.

  • You can click on marker symbols to retrieve a detailed marker record from MGD.
  • You can click on a contig to retrieve a detailed contig record from MGD.
  • You can click on the chromosome numbers or the colored bars to see more detailed linkage maps.

Mutant Allele Data

The MGD now accepts electronic submissions of mutant allele from the scientific community. Each electronic submission will receive a permanent database accession ID. MGD reference pages provide a link to the addociated data sets.

  • You can access the submission form from the Data and Nomenclature Submissions menu.
  • You can find general rules for naming alleles at Submitting New Mutant and Allele Data

Expanded Glossary

We have expanded the MGI Glossary. The Glossary contains definitions for terms found in the help documentation as well as other terms useful in understanding mouse genetics. You will find terms defined in a general sense as they apply to eukaryotic genetics, particularly mouse genetics. You will also find some terms defined as they apply to MGI (for example, gene). The list of terms included in the Glossary will grow with each new release.
