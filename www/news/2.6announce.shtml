MGI Release 2.6 Announcement - February 12, 2001

MGD Links to Locus Link and RefSeq

With the 2.6 release, database links for DNA sequences associated with a marker include NCBI/Locus Link and NCBI/RefSeq IDs in addition to DDBJ, EMBL, GSDB, GenBank, SWISS-PROT (protein database), EC (enzyme), UniGene, and Interpro domain records.

Your Input Welcome Button

A Your Input Welcome button is now available on the Marker Detail and the Alleles Detail pages. Clicking on the button brings up a form where you can submit updates and additions to the marker or allele data you are viewing. We hope this provides an easy way for you to help keep data current.

SWISS-PROT Load Enhancements

MGI marker records are now linked to InterPro domain records. Interpro - EMBL EBI (European Bioinformatics Institute) in Cambridge, UK, is an integrated resource with combined information from a number of protein domain and functional site sources (Pfam, PRINTS, PROSITE, ProDom) together with SWISS-PROT and TrEMBL data.

Allele Enhancements

The Allele summary page now includes a Molecular Notes column. This makes it easier to find the allele(s) of interest because available details about an allele at a molecular level now appear on this page.

References Linked to Phenotypic Alleles

The Reference detail record contains hypertext links to data extracted from publications and includes such items as marker descriptions, homologies, probes, PCR primers, mapping data, and antibodies. With MGI Release 2.6, the links now extend to phenotypic alleles.

Help Documentation Linked to MGI Glossary

Terms in our help documentation are now linked to the MGI Glossary. The glossary contains terms and definitions useful for understanding mouse genetics and is constantly expanding. If you have suggestions or requests for inclusion, please contact User Support.

New Rules and Guidelines for Genes, Alleles, and Mutation Nomenclature

The International Committee on Standardized Nomenclature for Mice (Ian Jackson, Chairperson) has completely rewritten the Rules and Guidelines for Gene Nomenclature to explain the existing rules more clearly and to detail new guidelines for naming transgenic mutations and wild-type alleles. See Mouse Nomenclature Rules and Guidelines for the Table of Contents and links to this document.
