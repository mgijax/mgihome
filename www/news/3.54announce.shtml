MGI Release 3.54 Announcement - June 11, 2007  

MGI's 3.54 release features a new batch query tool for accessing information about many genes/markers all at once.

  • Given a set of gene symbols or input identifiers (IDs) (such as MGI accession, RefSNP, or VEGA IDs) from, for example, a spreadsheet, you can copy and paste (or upload a file containing) a column of data into the query tool.

  • You can specify the output you want in the MGI Batch Query Results (for example, nomenclature; genome location; Ensembl, Entrez, or VEGA gene identifiers; gene ontology or mammalian phenotype data; GenBank/RefSeq, UniProt, RefSNP, or MGI Allele IDs).

  • You can also use the batch query to find duplications, ambiguities, and variations in data. You could check, for example, that an input list of MGI identifiers returns the same number of Ensembl, Entrez, and VEGA IDs on any given day, and that switching the input and output IDs returns the identical data.

  • You can view results in either web or tab-delimited format.
