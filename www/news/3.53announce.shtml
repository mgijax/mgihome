MGI Release 3.53 Announcement - April 18, 2007  

MGI is pleased to announce the release of Version 3.53 of the database.

  • MGI's newest feature is the association of mutant cell line IDs with MGI markers. You can now query with the question, Are there any gene traps associated with my gene? To find, for example, the gene traps associated with Adk, follow these steps:
    1. On the MGI Genes and Markers Query Form, enter Adk in the Gene/Marker Symbol/Name field and click Search. The Genes and Markers Query Results -- Summary report appears.
    2. Locate Adk and click it. The Gene Detail report for Adk appears.
    3. Scroll to Other Database Links and find the list of mutant cell line identifiers beside Gene Traps. Click to access a given gene trap's report in the dbGSS division of GenBank.
      Note: If there are more than five links associated with a marker, five are listed followed by All (with a count in parentheses), linked to the Gene Traps report. See Interpreting the Gene Traps Report for details.

    You can also use the Search Tool to find these associations. Enter either the mutant cell line identifier (for example, FHCRC-GT-S15-11C1) or the gene symbol/name in the box (on the home page) and click Search.

  • The Gene Expression Database's newest feature is a link to the Edinburgh Mouse Atlas and Gene Expression EMAGE database section browser. The EMAGE browser contains sections of embryos with labelled anatomical structures. Clicking on a structure in one of these sections highlights that term in the accompanying hierarchical anatomical term list. Right (or double) clicking on that term lets you query either the Gene Expression Database (GXD) or the EMAGE database for studies examining gene expression in that structure. Access to the EMAGE browser is from the Gene Expression menu page.
