MGI Release 2.9 Announcement - November 11, 2002

The MGI 2.9 release features:

  • Look and Feel
    • New look for MGI Home: Help page, Resources, About MGI, What's New
    • Toolbar
    • Marker detail page - tase of what's to come ...
    • New summary reports: sequence summary, gene ontology,m



  • Links to Ensembl Database
    MGI now provides links from marker detail pages for any markers with Ensembl gene models. You can access up-to-date sequence data and the best possible automatic annotation for mouse and human genomes.
