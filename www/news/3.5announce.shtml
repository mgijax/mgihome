MGI Release 3.5 Announcement - June 1, 2006  

MGI 3.5 release

MGI is pleased to announce version 3.5. Release highlights are:

  • You may now use MIT markers as query parameters in the Marker range field (e.g., between D6Mit128 and D6Mit129) on many MGI query forms (e.g., Mouse SNP, Genes and Markers, Gene Expression Data, Gene Expression Data Expanded, cDNA Clone, Anatomical Dictionary Browser Query Refiner). (MIT markers have been updated to reflect the most current Ensembl build. Chromosome position now accurately reflects genome position and markers that map to multiple locations are noted.)
  • You may now enter genome coordinates on expression query forms (Gene Expression, Gene Expression Expanded) as well as on the the Anatomical Dictionary Refiner, Molecular Probes and Clones, and cDNA Clone query forms.
  • We now provide tab-delimited output options from the Genes and Markers Query Form, as well as Excel output to an FTP site from the Mouse SNP Query Form.
  • We redesigned the Mammalian Orthology Query Results report to enhance interaction with and access to our data. This summary report now includes links to any relevant PIRSF superfamilies and MGI gene families, as well as to VISTA and Homologene databases. From the report, you can download representative mouse, human, or rat transcript and protein sequences to FASTA or forward them to MouseBLAST.
