MGI Release 3.5 Announcement - June 14, 2006  

MGI 3.5 release

MGI is pleased to announce the release of version 3.5.

  • We now store MIT marker coordinates in the MGI database. You can them as query parameters in the new Marker range field (located in the Map Position section) on many of our query forms. You can use any two markers (having genome coordinates and located on the same chromosome) as start and end parameters. The query is inclusive of the markers you enter and returns any marker with at least one base pair overlapping the range you specify. See Using Map Position Fields on MGI Query Forms for further details.

    As an example, for the QTL, Alcp1, alcohol preference locus 1, male specific, a region of mouse Chromsome 2, bounded proximally by D2Mit241 and distally by D2Mit125 is associated with alcohol consumption in male mice and given the symbol Alcp1.. Uuse the Genes and Markers query form to find all genes on Chromosome 2 between these markers and sort the results by genome coordinates.

    Type: Gene
    Marker range: between D2Mit241 and D2Mit125
    Sort by: Genome Coordinates

    Try using marker ranges on any of the following MGI query forms:

    • Genes and Markers Query Form (search by locus symbol/name, map position, GO classification, protein domain, phenotype/human disease, clone collection)
    • Gene Expression Data Query Form (search for detailed gene expression assay results using a variety of search parameters)
    • Gene Expression Data Expanded Query Form (search for genes expressed in some anatomical structures and/or developmental stages but not in others)
    • Molecular Probes and Clones Query Form (search by probe name, sequence type, vector type, marker symbol, origin, or reference)
    • cDNA Clone Query Form (search for information based on tissue or cell-line source)
    • Anatomical Dictionary Query Refiner (refine searches on anatomical structures or developmental stages accessed using the Mouse Anatomical Dictionary Browser)
    • Mouse SNP Query Form

  • You can now search by genome coordinates (see the Map Position section) on the query forms: Gene Expression, Gene Expression Expanded, Anatomical Dictionary Refiner, Molecular Probes and Clones, and cDNA Clone.

  • There is now a tab-delimited output option on the Genes and Markers Query Form, and an option to send Excel output to an FTP site on the Mouse SNP Query Form.

  • The Mammalian Orthology Query Results report is enhanced to include links to any MGI PIRSF superfamilies and MGI gene families, as well as to VISTA and Homologene databases. From the report, you can download representative mouse, human, or rat transcript and protein sequences to FASTA or forward them to MouseBLAST.

  • New Gene Ontology (GO) annotation graphs are available.
    • The Gene Ontology Classifications GO Graph (accessed from the MGI Gene Detail page) displays a visual relationship between molecular function, cellular component, and biological process annotations for a given gene or marker. As an example, see Ren1.
    • The Mouse-Human-Rat Comparative Graph (available from MGI Mammalian Orthology reports)is a visual display of curated GO classifications for mouse, human, and rat orthologs annotated from the biomedical literature. As an example, see Ren1.
