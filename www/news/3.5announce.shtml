MGI Release 3.5 Announcement - June 14, 2006  

MGI is pleased to announce the release of version 3.5.

  • We added the ability to query by Marker range (this field is located in the Map Position section) on many of our query forms. You can use any two markers (having genome coordinates and located on the same chromosome) as start and end parameters. The query is inclusive of the markers you enter and returns any marker with at least one base pair overlapping the range you specify (see Using Map Position Fields on MGI Query Forms for further details). As an example, a region of mouse Chromosome 2, bounded proximally by D2Mit241 and distally by D2Mit125, is associated with alcohol consumption in male mice and given the symbol Alcp1. You can use the MGI Genes and Markers Query Form to search for candidate genes on Chromosome 2 between these markers and sort the results by genome coordinates.

    You can specify a marker range on any of the following MGI query forms:

    • Genes and Markers Query Form (search for genes, DNA segments, cytogenetic markers, QTLs, pseudogenes, BAC/YAC ends, other genome features)
    • Gene Expression Data Query Form (search for detailed gene expression assay results)
    • Gene Expression Data Expanded Query Form (search for genes expressed in some anatomical structures and/or developmental stages but not in others)
    • Molecular Probes and Clones Query Form (search by probe name, sequence type, vector type, marker symbol, origin, or reference)
    • cDNA Clone Query Form (search for information based on tissue or cell-line source)
    • Anatomical Dictionary Query Refiner (refine searches on anatomical structures or developmental stages accessed using The Mouse Anatomical Dictionary Browser)
    • Mouse SNP Query Form (search by strains, strain comparisons, SNP attributes, map position, marker range, or associated genes)

  • Genome coordinates for all MIT markers are now available.

  • You can use genome coordinates as search parameters on the Gene Expression, Gene Expression Expanded, Anatomical Dictionary Browser Query Refiner, Molecular Probes and Clones, and cDNA Clone query forms.

  • There is a new tab-delimited output option on the Genes and Markers Query Form and the ability to save your data in an Excel file on the Mouse SNP Query Form.

  • The Mammalian Orthology Query Results report now includes links to any relevant MGI PIRSF superfamilies, MGI gene families, VISTA, and Homologene databases. From the report, you can select representative mouse, human, or rat transcript and protein sequences to download in FASTA format or forward to MouseBLAST.

  • New Gene Ontology (GO) annotation graphs are available.
    • The Gene Ontology Classifications GO Graph (accessed from the MGI Gene Detail page) displays the visual relationship between molecular function, cellular component, and biological process annotations for a given gene or marker. As an example, see Ren1.
    • The Mouse-Human-Rat Comparative Graph (available from MGI Mammalian Orthology reports) is a visual display of curated GO classifications for mouse, human, and rat orthologs annotated from the biomedical literature. As an example, see Ren1.
    Terms in the boxes link to additional GO information at MGI.

MGI is the community model organism database for the laboratory mouse.
