Mouse Genome Informatics (MGI) 2.0 Release Notes  

June 8, 1998

New Gene Expression Data and Query Form Now Available

A data set of RT-PCR assays produced by Dr. Tom Freeman's laboratory (J:46439) is available from the database. This data set represents the analysis of 517 genes in 46 mouse tissues from animals 6 to 8 weeks old. In addition, the data set includes studies of tissue from 15 day old mouse embryo. Additional data for this release of GXD will soon be made available and will include RNA in situ hybridization and Immunohistochemistry data, Northern and Western Blot data and RT-PCR data. Thereafter, the database will be updated daily. Future GXD releases will accommodate additional types of expression data.

Gene Expression Data Query Form

This query form enables you to ask two basic kinds of questions:

  • Where are specified genes expressed and/or at what developmental stages?

  • What genes are expressed in specified anatomical structures and/or during specified developmental stages?

You can search for expression data related to specified genes, anatomical structures, or developmental stages. Query results can be returned as a listing of assays or assay results. See the help documentation for details. In this first release, you can only search for RT-PCR assays. In the near future, more data will become available.

http://www.informatics.jax.org/searches/anatdict_form.shtml Expression patterns are described using The Mouse Anatomical Dictionary Browser that has been developed in collaboration with the Anatomy Department, University of Edinburgh and the MRC Human Genetics Unit (Edinburgh, UK). Using the Gene Expression query form, you can limit your query to one or more specified anatomical structures by entering one or more values in the Anatomical Structures field. If unsure of nomenclature, you can browse the Anatomical Dictionary on-line.

Enhanced Linkage Maps Form

This form has been reorganized and simplified. You can generate a linkage map using information retrieved from the database. The view of the chromosome is determined by the criteria used to search the database. You can select a chromosome, a region of the selected chromosome, include markers from selected phenotypic classes, include DNA segments or syntenic markers, or add your own markers. You can also select a mammalian species to compare and generate a comparative map. A new option allows you to choose to display only mouse markers that have homologies in the selected comparison species.

AXB/BXA Mapping Data Sets Available Via FTP

These data sets were produced by Dr. Beverly Paigen's laboratory at The Jackson Laboratory (J:45463). In MGI 2.0, these data files are available in EXCEL format and can be downloaded from our ftp server. These data present new typings and revised typings in the AXB and BXA RI mapping data sets. This information is also being incorporated into the database and will be available in the experimental mapping data and the composite RI data sets.

Revised Data Submission Guidelines

The guidelines for data submissions have been updated to facilitate E-mail submission of MGD data. It is anticipated that electronic data submission direct from the research laboratory to MGI databases will become the norm in the near future. In MGI 2.0, steps have been taken to facilitate E-mail submissions as an incremental step toward automating the submission process. Contact Jennifer Bobish at submissions@informatics.jax.org if you have questions or need assistance.

1998 Chromosome Committee Reports Available On-line

The 1998 Chromosome Committee reports are complete and available on-line.

Enhanced linkage map display from marker detail record

When you display the detail record for a marker, it may include a "mini-map" image. This map shows the selected marker highlighted on the map in relation to a set of other "anchor" loci on the chromosome. Now, you can click on this map image and retrieve a Web Map display of the chromosomal region within 1 cM of the selected marker. This display shows ALL the markers in the region including DNA segments. Each marker symbol on the map is linked to the marker detail record. This feature allows you to explore the markers immediately surrounding a gene of interest without having to execute a new search.

Other Enhancements to MGI Query Forms and the Query Menu

Various enhancements to MGI query forms made in release 2.0 are summarized here.

Search by one or multiple gene symbols (current or withdrawn)
This feature replaces the option to search by only a single symbol. In addition, the Name field can be used to search for either a gene name or an unoffical symbol and affects the following query forms:

  • Genes, Markers and Phenotypes
  • Polymorphisms
  • Molecular Probes and Segments
  • cDNA Clone and EST Expression

In addition, the following quick forms have been modified in a similar fashion:

  • Genes, Markers and Phenotypes by Symbol
  • Molecualr Probes and Segments by Symbol

Modification Date field added to Mapping Data query form
Now you can filter a search for mapping data records by Modification Date. This enables you to, for example, retrieve only records that have been modified in the last six months. This field is available on other MGI query forms.

Changes to the Query Menu Entries have been added for the Gene Expression Data query form, Quick Forms and Mapping Panels. In addition, the Query Menu has been fixed so that whenever you click on the Query Menu link from an MGI page, that page becomes the "active" page for the Query Menu. This feature enables you to always choose the browser window in which you want the Query Menu selection to be displayed.
