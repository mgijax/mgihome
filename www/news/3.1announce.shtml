MGI Release 3.1 Announcement - December 16, 2004  

MGI version 3.1 provides the scientific community with the ability to query for genes based on their coordinates in the current mouse genome assembly.

As a result, you can now use these coordinates to query for:

  • sequences using the Mouse Sequence Query Form. You define attributes or source or certain properties and annotations of associated genes and markers. The search returns sequences associated with markers that overlap or are included by the coordinates you enter in the Genome Coordinates field on the query form.

  • sequences associated with markers mapped to a set of assembly coordinates using the Mouse Sequence Query Form.

  • markers using the Genes and Markers Query Form. You enter a chromosome and the coordinates in these fields on the query form.

Genome coordinates for genes (as annotated by NCBI or Ensembl) appear on the the MGI gene and sequence summary and detail reports. Information about sequence attributes, such as length, type, strain, and provider descriptions, are also provided. You can have the summary reports sorted by genome coordinate.

When forwarding a genome assembly sequence to MouseBLAST (the sequence similarity server at MGI) or downloading the sequence in FASTA format, you can specify an amount of flanking sequence. Using the MouseBLAST (WU-BLAST 2.0), you can search a sequence within mouse/rodent sequence databases to find matching sequences that may be associated with mouse genes. Any results are linked to MGI mouse gene records. All sequence databases are updated on a weekly basis.
