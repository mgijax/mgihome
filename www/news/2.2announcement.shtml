Mouse Genome Informatics (MGI) 2.2 Release Announcement  
April 12, 1999

MGI Web Site Reorganized

Locations and names of all files in the MGI web site have changed.

The directory structure and file-naming conventions within the MGI web site have changed. An overhaul of our web site organization has long been needed to facilitate MGI mirror-site installation and maintenance. We are trying to make this transition as easy as possible for our users and for web sites that link to MGI resources. If you have created bookmarks to access MGI web pages, you will still be able to get to the appropriate resource with the old URL but you will see a message with a link to the new URL. Please update your bookmarks to reflect the new locations and file names.

For web sites using links to CGI scripts to retrieve information from MGD/GXD, web administrators should refer to the updated instructions for creating links to MGI. The old CGI scripts will continue to be supported through mid-summer 1999. Web sites that provide hyper-links to MGI web pages other than the MGI Home Page will need to have these links updated to reflect the new URLs. The URL for the MGI Home Page, http://www.informatics.jax.org/, remains unchanged.

Enhancements to the Display of Marker Details

In response to user requests, the marker detail display now includes sequence database accession IDs with hyper-links to DDBJ, EMBL, GSDB and GenBank. Sequence accession IDs are included in the listing, Other Database Links for this Marker, shown below the Additional Information portion of the detail record. See the marker detail for Tyr, for example.

The list of alleles for a marker is now accessed via the Phenotypic Alleles hyper-link in the Additional Information section of the marker detail record.

Quick Gene Search

The Quick Gene Search, located on the MGI Home Page, enables you to search for mouse genes using a single gene symbol or name. You can use a partial symbol or name with appropriate wildcards. A search yielding a single result will open a window on the detail record for the gene.

Mouse Facts Pages

The Mouse Facts pages present information from various sources and include the following items:
  • Mouse sequencing progress summary
  • Genome length estimates
  • MGD chromosomal length and gene distribution
  • Link to Codon Usage Tabulated from GenBank (CUTG) search form
  • Links to NIH, molecular resources, and other sites of interest to researchers
  • Mouse physiology information

Search Forms Menu

All MGI pages now include a Search Forms selection list directly under the MGI banner. While the text links in the banner provide access to various menus and other resources, the Search Forms menu provides quick access to the MGI query forms. The Search Forms menu replaces the Query Menu and works with all platforms and browsers.

Submitting Data

We encourage contributions of electronic data sets from the scientific community. Any type of data that MGD or GXD maintains can be submitted as an electronic contribution although mapping data, polymorphisms, and mammalian homologies are currently the most common. Each electronic submission receives a permanent database accession ID. For data submission guidelines, see the submissions page or send E-mail to submissions@informatics.jax.org.

To submit a proposed locus or gene symbol, please use the form at http://www.informatics.jax.org/support/nomen/.
