MGI Release of FANTOM2 Data - January 24, 2003  

In collaboration with the Genome Exploration Research Group of the RIKEN Institute of Physical and Chemical Research in Japan and the National Center of Biotechnology Information (NCBI), Mouse Genome Informatics has released integrated information regarding the FANTOM2 mouse cDNA clone set (see J:80000). The FANTOM2 set consists of 60,770 mouse cDNA clones isolated from full-length enriched libraries. This set contains 21,076 cDNAs from the FANTOM1 effort (see J:65060), included for reannotation, and 39,694 novel FANTOM2 clones. Clone and sequence information for these data are now available via standard MGI queries and MouseBLAST. All FANTOM2 new clones were issued DDBJ sequence identifiers, and many of the FANTOM1 clones that were not issued sequence identifiers for the FANTOM1 release were resequenced in FANTOM2 and issued sequence identifiers. However, there remain 801 FANTOM1 clones without DDBJ sequence identifiers. Composite sequences for these clones, obtained by combining incomplete RIKEN sequence with public EST data, are queriable via MouseBLAST and available in FASTA format at the FANTOM2 web site. Associations between RIKEN clones and existing MGI genes were established by a combination of curatorial and automated methods as part of a effort to annotate function for the FANTOM2 clones by the RIKEN group and the FANTOM2 Consortium. For this load into MGI, the FANTOM2 data were associated with 19,988 gene records, 3,036 of which are novel to MGI. As was done for the FANTOM1 data, new mouse genes discovered in the RIKEN clone set have been assigned official nomenclature in MGI following a defined syntax:

Gene Symbol =  <Riken Clone Identifier>"Rik"
Gene Name =  "Riken cDNA" <Riken Clone Identifier>  "gene"
e.g. 2610307C23Rik, Riken cDNA 2610307C23 gene

For novel genes represented by multiple RIKEN clones that show sequence overlap (clusters), nomenclature was taken from the clone identifiers of representative clones. The nomenclature for some novel genes was derived from ortholog gene nomenclature. A total of 18,794 FANTOM2 clones were loaded without associations to MGI gene records; thus, the gene count for this clone set is an underestimate. These clones will be associated to gene records following the completion of additional redundancy reduction steps. As the new mouse genes from this clone set become better characterized, revised nomenclature and other biological data will be incorporated regularly.

The FANTOM2 load into MGI is being implemented in two phases. Phase I, which is the subject of this release announcement, involves the following:

Phase I

  • Molecular Segment records and source information for the FANTOM2-new clones
  • New gene records for FANTOM2 data that were either curated or that show no clustering discrepancies (Note: No new gene records were created for uncurated singletons for this load.)
  • FANTOM2-based nomenclature merges
  • Data associations (Molecular Segment-Gene, Seqid-Gene) and reassociations (FANTOM1 data reassociated due to FANTOM2 curation)
  • Problem sequence annotation
  • MouseBLAST access to FANTOM2 data
  • Links to FANTOM2 annotation pages

Phase II of the load will be implemented over the next few weeks, and will involve the following data types:

Phase II

  • Mapping data (chromosome assignments obtained from mapping the FANTOM2 data to the mouse genome assembly)
  • Gene Ontology (GO) associations
  • Nomenclature updates
  • Orthology data

For more information about the FANTOM2 data in MGI or this load, please contact the staff of MGI User Support (mgi-help@informatics.jax.org).
