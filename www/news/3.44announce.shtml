MGI 3.44 Release Announcement - April 27, 2006  

MGI 3.44 Release

MGI is pleased to announce its 3.44 release. The main features are:

  • A new SNP data summary. The SNP Data in MGI report contains information about the:
    • dbSNP build number
    • number of submitted SNP assays (ss) loaded
    • number of reference SNPs (rs) loaded
    • average number of ss per rs
    • total number of strains with SNPs loaded
    • number of ss per submitter
    • number of rs per strain, by chromosome.

  • The Gene Expression Notebook (GEN) Version 2.2.0 release.
    GEN Version 2.2.0 adds Excel 2003 (running on Windows XP and Windows 2000) as a supported version of Excel. As with our previous release, this application is also stable on Excel X for Mac and Excel 2000 for PCs.

  • The retirement of out-of-date reports.
    MGI is removing the following outdated static reports from the web interface: the Mouse Locus Catalog (MLC) and the Chromosome Committee Reports. These reports have not been updated since 2000. Current data for phenotypes and genome displays are available in MGI.

MGI is the community model organism database for the laboratory mouse.
