MGI Release 3.44 Announcement - April 27, 2006  

MGI 3.44 release

MGI is pleased to announce its 3.44 release. The main features are the:

  • Addition of a SNP data summary. The SNP Data in MGI report contains information about the:
    • dbSNP build number
    • number of submitted SNP assays (ss) loaded
    • number of reference SNPs (rs) loaded
    • average number of ss per rs
    • total number of strains with SNPs loaded
    • number of ss per submitter
    • number of rs per strain, by chromosome.
  • Removal of the Mouse Locus Catalog (MLC). This descriptive phenotypic resource is replaced by annotations to structured vocabularies including the Mammalian Phenotype Ontology and the Human Disease Vocabulary.
  • Removal of access to the Chromosome Committee Reports from the Maps and Mapping Data menu. (The last year that these reports were available was 2001.)

The remainder of the 3.44 release consists of performance enhancements, bug fixes, and updates.

MGI is the community model organism database for the laboratory mouse.
