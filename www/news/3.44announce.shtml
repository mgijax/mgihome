MGI Release 3.44 Announcement - April 27, 2006  

MGI 3.44 release

MGI is pleased to announce its 3.44 release featuring a summary of the SNP Data in MGI.

MGI loads mouse SNP data from dbSNP in conjunction with new dbSNP build releases. The data consists of mouse reference SNPs (rs) and the corresponding submitted SNP assays (ss) where at least one ss assay of a reference SNP has strains defined for the alleles submitted. MGI does not load rs or ss data for Reference SNPs having no submitted assays with defined strain/allele relationships. SNPs that map to multiple locations are loaded only if all locations for the SNP map to the same chromosome.

The summary page contains details about the:

  • dbSNP build number
  • Number of ss loaded
  • Number of rs loaded
  • Average number of ss per rs
  • Total number of swtrains w/ SNPs
  • Number of rs per strain
  • Number of ss per submitter

In addition there are links to:

  • dbSNP at NCBI
  • dbSNP build statistics
  • each assay submitter's web site

The remainder of the 3.44 release consists of performance enhancements, bug fixes, and updates.

MGI is the community model organism database for the laboratory mouse.
