MGI Build 36 Release Announcement - September, 2006  

New features and changes/additions to MGI data

  • Multiple gene models are possible for a single MGI gene record.
  • Multiple MGI genes may be associated with a single gene model.
  • Coordinates for QTLs and miRNAs (microRNAs)
  • Links to miRBase and to VEGA.
  • MGI genome coordinates for mouse genes are updated to Build 36. These coordinates are from manual annotation from Vega Mouse (Vertebrate Genome Annotation mouse gene model) at Sanger Institute, and computational annotations provided by EnsemblNCBI, and The following is an example of an MGI gene detail page with link to curated locus report from VEGA Gene Model
    Oip5 (Vega accession ID OTTMUSG00000015949)

  • Unified Gene Catalog
    MGI is producing a Unified Gene Catalog, integrating the data from VEGA, NCBI, and Ensembl into a single set of distinct genes, pseudogenes, and other genomic features. MGI will synchronize data with that of the three providers from now on.
  • MGI now contains physical map data for microRNAs from mirBASE
    . Set the Type field on the Genes and Markers Query Form to microRNA to get a summary of all the data that MGI contains. Click on one of the symbol/names to find the details. The following is an example of a an MGI microRNA detail page with coordinates:
    Mirn106a

  • Mouse GBrowse, MGI's interactive sequence map, is also updated to Build 36 and now contains a microRNA track from mirBASE as well as tracks for SNPs, QTLs, Ensembl genes, NCBI genes, VEGA genes, MGI representitive transcripts, STS markers, phenotypes, and alleles. Access to Mouse GBrowse is from the MGI home page (www.informatics.jax.org) and from the sequence map section on MGI gene detail pages.

  • You can query by marker range on many MGI query forms using two markers (having genome coordinates and located on the same chromosome) as start and end parameters. The query is inclusive of the markers you enter and returns any marker with at least one base pair overlapping the range you specify (see Using Map Position Fields on MGI Query Forms for further details). Note that you may not enter QTLs in the Marker Range field.

    MGI is the community model organism database for the laboratory mouse.
