Uploading a File  

How do I upload a file?

  1. Click the Browse button on the web form appearing next to the "upload file" box.
  2. In the File Upload dialog box that appears, navigate your hard drive to the location of the file you want to upload.
  3. Click the name of the file you want to upload.
  4. Click Open. The path name to this file appears next to File Path on the web form. (Example: C:/Documents/Justif/kojust.PDF)

When you complete the web form and click Submit, the content of the file is sent along with your responses. A page with the heading Results Sent! appears. At the bottom of this page, File Uploaded appears, confirming that this file has been sent.

Note: Be sure not to edit, delete, or move this file until you've clicked the Submit button.

Please contact MGI User Support if you need additional help.
