Basic Nomenclature Rules and Guidelines

When symbolizing and naming novel genes and genomic features, it is important that these symbols and names be unique identifiers. The MGD Nomenclature Committee (nomen@informatics.jax.org) provides advice and assistance in assigning new symbols and names. Below are some basic rules and guidelines to assist in the symbol and naming process.

Mouse symbols:

  1. Should be 3-5 characters; should not exceed 10 characters.
  2. Should begin with an upper case letter, followed by all lower case letters, except for recessive mutations which should begin with a lower case letter.
  3. Should not include tissue specificity and molecular weight designations.
  4. Should be italicized in published articles.
  5. Should be punctuated only to separate two adjacent numbers (e.g., Lamb1-2) or for related (e.g., Es10-rs1) sequences and pseudogenes (e.g., Adh-ps1).
  6. For family members:  Use common stem symbol (e.g., Cldn#)
  7. For orthologous genes in other vertebrate species:   Use the same symbol whenever possible in human, rat, and mouse.
  8. For homologous genes in invertebrate or prokaryote species:
    1. If it is a true homolog, use the approved symbol for that species, but include the word "homolog" at the end of the name followed by the name of the species in parentheses; for example, Cdc20 for cell division cycle 20 homolog (S. cerevisiae).
    2. If there is more than one mouse homolog, assign the serial number after the word "homolog;" for example, Atoh1 and Atoh2 for atonal homolog 1 (Drosophila) and atonal homolog 2 (Drosophila), respectively.
    3. If the gene in question is not a true homolog, use the letter "l" to denote "-like" designations; for example, Ash2l for ash2 (absent, small, or homeotic)-like (Drosophila).
  9. ESTs and BAC/YAC ends:  Use the assigned sequence accession identification numbers.
  10. STSs:
    1. Use the convention D#Xxx%e, where: a
      # = chromosome location
      Xxx = approved lab code
      %= serial number in series
    2. Obtain approved lab codes from ILAR (Institute for Laboratory Animal Research: http://www4.nas.edu/cls/ilarhome.nsf
    3. Add the trailing "e" only when the DNA segment is expressed.

Mouse names:

  1. Should be brief and specific.
  2. Should convey the character or function of the gene.

Other information when publishing:

  • Chromosome designations:
    1. Use upper case C when referring to a specific mouse chromosome (e.g., Chromosome 15).
    2. Do not use a period (.) after the abbreviation of a chromosome (e.g., do not write Chr 15 as Chr.15).
  • Protein designations:
    1. Use all upper case letters.
    2. Do not italicize.

Note: For more detail and additional special case rules, see the Mouse Nomenclature Rules and Guidelines.
