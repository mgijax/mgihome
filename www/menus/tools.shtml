Genome Analysis Tools
• Batch Query
Input a list of gene IDs or symbols and retrieve other database IDs and gene attributes (e.g. phenotypes, GO).
• MouseMine
Access MGI data via InterMine.
• NCBI BLAST
Search for sequence similarity using NCBI BLAST.
Prototype Analysis Tools
These tools are still in the prototyping and development phase, and may change, break or disappear at any time. Your feedback can help us improve them. Please feel free to experiment with them and send us your comments.
• VLAD - Visual Annotation Display
VLAD is a tool for visualizing GO annotations. The annotation data are a subset of those available at the GO web site. You specify an annotation data set and a set of genes, and the tool will produce a graphical summary of the annotations.
• GO Slim Chart Tool
This GO_Slim Chart Tool bins the genes in your list according to MGI GO Slim definitions to help you discover what the genes may have in common.
• GO Term Mapper
This GO Term Finder tool searches for significant shared GO terms used to describe the genes in your list to help you discover what the genes may have in common.
