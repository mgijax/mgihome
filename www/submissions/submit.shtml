Submitting Data to the Mouse Genome Database (MGD)  

The Mouse Genome Database (MGD) accepts electronic data submissions from the scientific community. Any type of data that MGD maintains can be submitted as an electronic contribution, although mapping data, polymorphisms, and mammalian orthologies are currently the most common. Each electronic submission receives a permanent database accession ID.

All data sets are associated with either an electronic submission reference or a published paper. MGD reference pages provide links to associated data sets. Mammalian Genome will publish a list enumerating the data electronically submitted to MGD, including accession ID, chromosome, and locus symbols as a cross-referencing service for readers.

General Guidelines

  • Select the Data Sections relevant to your electronic submission.  Review the detailed examples and additional instructions by selecting the appropriate hyper-text links.  The data contained in the samples is for exemplary purposes only and is not to be mistaken for actual data.
  • If you are reporting original data for more than one chromosome/experiment, please submit individual data sections for each.
  • Follow the format and file type guidelines given in each section.  For example, submit mapping data either as an Excel file or tab-delimited text attached to an E-mail message.  Your submission should consist of one E-mail message with data files attached as necessary.  Be sure to label each data section.
  • All symbols for genes or genetic markers must be verified as valid before submitting data.  Please check symbols against the MGD marker data:

Detailed Instructions and Links to Examples

Data Section I 

Reference Information  Sample

This section must be completed for all data sets and can be included in the body of an E-mail message.  The reference information should include the following items as applicable to your submission:

  • Corresponding author
  • Institution
  • Address
  • E-mail
  • Phone
  • Complete author list with institutions
  • Title of submission
  • Status:
      • ___ submitted as electronic-only data set
      ___ submitted to a journal (journal name:___________)
      ___ accepted for publication  (journal name: __________)
  • Are you requesting data privacy? (Y / N)
  • Abstract  (Optional, only for data that are electronic-submission)

Data Section II Genetic Mapping Information  Sample

For mapping data, provide the following on separate lines in the body of an E-mail message:
  • Approved symbol of primary locus mapped
  • Name of locus mapped
  • Chromosome and map position (cytogenetic band or cM position)

Data Section III

Linkage Experiments  Sample

For each Linkage Experiment complete the following as they apply to your data on separate lines in the body of an E-mail message:

  • DNA mapping panel used
  • Female parent
  • Male parent
  • Key to symbols used in typings
  • Chromosome
  • Total number of chromosomes typed

Data Section IV Haplotype Data  Sample

Haplotype data should be submitted as a file attached to the E-mail contents above in either Excel or tab-delimited text format.  If your locus was mapped using the Jax BSS or Jax BSB mapping panels, you need not provide haplotype data. For loci mapped using other mapping panels, be sure to include the following items:
  • Approved locus symbols
  • Typings (according to your key)
  • Assay type
  • Probe/Primer pair name
  • Number of chromosomes typed

Data Section V Recombinant Inbred/Recombinant Congenic Experiments  Sample

In the body of an E-mail message, be sure to include the following items:
  • RI or RC set designation
  • If this is a newly created set, specify progenitors
  • Key to symbols used in typings
  • Chromosome
Recombinant Inbred/Recombinant Congenic tables should be submitted as a file attached to the E-mail contents above in either Excel or tab-delimited text format.  In your table, be sure to include:
  • Approved locus symbols
  • Individual RI/RC strain designations
  • Typings (according to your key)
  • Assay type
  • Probe/Primer pair name

Data Section VI FISH Experiments  Sample

For FISH data, complete the following as they apply to your data on separate lines in the body of an E-mail message:
  • Approved locus symbol
  • Chromosome
  • Cytogenetic band
  • Cell type
  • Strain
  • Karyotype method
  • Probe name
  • Label
  • Number of metaphases examined
  • Chromosome aberrations involved
 

PROBES, CLONES, ESTs AND PCR PRIMERS USED IN MAPPING
 

Data Section VII Probes, Clones, and ESTs  Sample

Information on Probes, Clones, and ESTs should be submitted on separate lines in the body of an E-mail message.  Be sure to include the following items as they apply to your data:
  • Probe/Clone/EST name
  • Corresponding locus symbol
  • Encodes/Hybridizes
  • Probe Sequence Type
  • Corresponding region of gene
  • Vector type
  • Species of origin
  • Strain of origin
  • Tissue
  • Cell line
  • Age
  • Sex
  • Size of probe/clone insert
  • DNA sequence accession number
  • Holder

    •  

Data Section VIII PCR Primers  Sample

For each PCR primer, be sure to include the following items in the body of an E-mail message:
  • Locus symbol
  • PCR primer names
  • Primer sequences

Data Section IX Polymorphisms  Sample

Polymorphism information should be submitted as a file attached to the E-mail contents above in either Excel or tab-delimited text format.  Be sure to include the following items as they apply to your data:
  • Assay type
  • Endonuclease
  • Locus
  • Probe or primer name
  • Alleles
  • Fragment size
  • Strains

Data Section X Mammalian Orthology  Sample

Be sure to include the following items on separate lines in the body of an E-mail message:
  • For the primary mammalian species:
    • Species
    • Approved locus symbol
    • Approved locus name
    • Chromosomal location
    • Cytogenetic band or cM position
    • Nucleotide sequence accession number
  • For the comparison mammalian species:
    • Species
    • Approved locus symbol
    • Approved locus name
    • Chromosomal location
    • Cytogenetic band or cM position
    • Nucleotide sequence accession number
  • Published reference supporting orthology assertion
  • Assay type(s)
