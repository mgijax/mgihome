Querying the GO in MGI

Genes in MGI are annotated using GO terms. Several query forms, "Genes, and Markers," "Linkage Map," "Gene Expression Data," and "cDNA Clone" provide a "Gene Classifications" search field that enable you to enter a GO term or phrase for searching. Once you submit the query, the system checks the ontologies to be examined. To improve and refine your searches, you should become familiar with the available terms in the ontologies (see the Molecular Function, Cellular Component, and Biological Process links in the left frame).

  • You can search by entering all or a portion of a single GO vocabulary entry. Often, GO entries are phrases containing several words. The default text operator is CONTAINS. If you enter a few characters of a GO term, the search will find any gene records having gene classifications that include the specified characters (and belong to the checked categories).

  • You can also use the logical operator "AND" between GO terms within the Gene Classifications search field. A search of term1 AND term2 will return the set of markers that are annotated to both term1 and term2. For example, the search string "nucleus AND kinase" will return all the markers whose products are a kinase AND are found in the nucleus.

  • You can browse the GO classifications, copy an entry, and paste it into the Gene Classifications field.

  • You can restrict your search to markers with attributes in only one or two of the three categories (the same term can two or three GO categories).

  • Please note that the GO vocabularies are hierarchical. Consequently, a search returns gene records annotated not only to terms matching your query but also gene records annotated to more detailed terms related to the query term (but not necessarily containing the text you entered). For example, if you enter: trypsin, you will get only:
    trypsin [GO:0004295] (59 genes, 59 annotations)
    But, if you type serine-type endopeptidase, the system will return
    serine-type endopeptidase [GO:0004252] (123 genes, 276 annotations)
    because the subterm trypsin is included in the results.

Each gene returned by the query is linked to a marker detail page with the associated ontology, GO term, evidence statement, and publication from which the annotation was discerned.

MGI queries return information about mouse genes. To query the GO simultaneously across multiple species, a prototype query interface is available (see Gene Ontology Applet).
