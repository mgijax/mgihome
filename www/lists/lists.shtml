MGI E-mail List Service

MGI maintains several E-mail lists for the genetics research community. The list service uses the Lyris list service software package, version 3.0.

To get information and instructions regarding the list service, send a message containing the word help in the subject or message body to

lists@lists.informatics.jax.org

Click on one of the following list names to get to the main menu for the list. You can visit the list as a non-member to see if you want to subscribe. You can also search the list archives for discussions of interest that may have been covered in the past.

  • Search MGI-LIST - Forum for topics in mouse genetics, MGI news updates
  • Search RAT-LIST - Forum for rat genetics
  • Search PHENOME-LIST - Forum for strain characterization and phenotypic data collection
