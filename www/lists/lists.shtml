MGI E-mail List Service

MGI maintains an e-mail list for the genetics research community. The list service uses the Lyris list service software package, version 4.2.1.

To get information and instructions regarding the list service, send a message to:

list_help@lists.informatics.jax.org

Click the link below to get to the list on the main menu. You can visit this list as a non-member to see if you want to subscribe. You can also search the list archives for discussions that may have been covered in the past. Once you subscribe, you can manage your subscription (for example, to read or create messages or to unsubscribe) using the web version of the list.

  • Enter MGI-LIST - Forum for topics in mouse genetics, MGI news updates.
  • Enter MGI-TECHNICAL-LIST - Forum for technical information about accessing MGI data, APIs and making links to MGI.

How to Subscribe

You can subscribe here. Enter your name and E-mail address below and click the list. Click on the Submit button to send your subscription request. You will receive an acknowledgement and instructions via E-mail.
