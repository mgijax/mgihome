Web Browser Compatibility

System Compatibility
We test our website releases for compatibility with the following systems:
Operating System Web Browser
Microsoft Windows®; XP: Internet Explorer® 7.0
  Netscape Navigator® 8.1
  Mozilla Firefox® 2.0
Mac OS® 10.3:  Safari™ 2.0
  Mozilla Firefox® 2.0
Older computer operating systems and/or web browsers may display the pages differently than we intended.
JavaScript
The MGI website makes use of JavaScript® and cascading style sheets. To obtain the best experience with our web pages, please make sure you have JavaScript enabled and that you have not disabled linked style sheets.
Note to Internet Explorer® users: you may have unknowingly disabled JavaScript. If you are using the highest security setting, JavaScript has been disabled automatically. To enable it, set Active Scripting to "Enabled" using Custom Level... on the Security tab of Internet Options.

Colors
We test our text and background colors with members of our staff who are color blind, and we use color-blindness simulators. However, most modern web browsers allow users to create and apply a user style sheet with preferred colors, font sizes, etc., and you may wish to investigate such style sheets for yours.
Screen Resolution
We design our web pages using a screen resolution of 800 x 600. Lower resolutions (smaller screen area displayed) may display the pages differently than we intended.
