Any other OS/browser configurations may not display as intended.
JavaScript
The MGI website uses JavaScript® and cascading style sheets. For the best experience with our web pages, make sure you have JavaScript enabled and have not disabled linked style sheets.
Note to Internet Explorer® users: If you are using the highest security setting, JavaScript is disabled automatically. To enable it, click Tools->Internet Options->Security->Custom Level.... Scroll toward the bottom of the list and locate Scripting. Click Enable under Active Scripting.
Colors
We test background colors and text on color-blind subjects, and we use color-blindness simulators. However, you may have your own preferred colors, font sizes and so on; most modern web browsers allow you to create and apply a custom style sheet for that purpose.
Screen Resolution
We design our web pages using a screen resolution of 800 x 600. Lower resolutions (smaller screen area displayed) may display the pages differently than we intended.
