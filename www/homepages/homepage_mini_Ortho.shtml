Mammalian Orthology News
02 Jan 2008
  • MGI comparative annotation is collaborating with the GO Consortium to develop a community ortholog set that will be used by the GO Reference Genome Project to guide community annotation projects for related genes. Each of 12 model organism informatics groups have contributed a FASTA file of reference protein sequences for comparative set analysis and clustering.
