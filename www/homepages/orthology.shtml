Vertebrate Homology News
May 28, 2015
  • MGI uses homology class data from HomoloGene and can support a "many-to-many" relationship between mouse genes and their vertebrate homologs. These data include human, chimpanzee, rhesus macaque, dog, cattle, rat, chicken, western clawed frog, and zebrafish homologs. MGI also uses HGNC/HCOP data for human and mouse orthology.
  • Vertebrate Homology Class pages include EntrezGene links, Genetic Location and sequence information. See this complement component Vertebrate Homology Class page for an example.
