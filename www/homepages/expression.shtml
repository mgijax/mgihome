Gene Expression Database (GXD)

Gene Expression News August 16, 2011
  • We are pleased to announce the inclusion of RNA in situ hybridization and immunohistochemistry data from the GenitoUrinary Development Molecular Anatomy Project (GUDMAP) database. You can view the data set at accession number J:171409. This set includes 92,569 results for: 3,123 genes; 3,561 assays (Immunohistochemistry and RNA in situ hybridizations for E10.5-E17.5 and postnatal specimens); and 8,730 image montages for ~25,000 individual images.

  • GXD's gene expression data have been added, as a new dataset, to the MGI implementation of BioMart. The GXD BioMart provides rapid access to gene expression results. Search returns can be customized, and the results easily downloaded for analysis in the lab. In addition, the GXD BioMart can be interconnected with other BioMarts to enable queries across the combined resources.

  • We are pleased to announce the inclusion of RNA in situ hybridization data from the Brain Gene Expression Map (BGEM) database. You can view the data set at accession number J:162220. This set includes 53,496 results for: 1,451 genes; 1,474 assays (RNA in situ hybridizations at E11.5, E15.5, P7 and P42); and 16,112 images. RNA in situ hybridization data from the Eurexpress project was also recently added to GXD. You can view the data set at accession number J:153498. This set includes ~497,000 results for 6,594 genes in 7,052 assays with more than 118,400 images.
