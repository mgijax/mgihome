Introduction

This document provides links to external genomic sequence analysis servers and software applications. The tools are organized in these groups.
  • Comprehensive sequence analysis servers
  • Sequence analysis pipeline software
  • Results visualization and curation tools
  • Large-scale comparative methods
  • Genomic sequence analysis servers

Comprehensive Sequence Analysis Servers

 These are analysis servers that run multiple gene prediction and analysis programs with a given query sequence.
  • Genome Analysis Pipeline (ORNL) - Predicts exons (GRAIL, GRAIL-EXP), gene-structures (GENSCAN), GRAIL feature tools: CpG islands, BAC end hits, PolyA, Trna, Simple Repeats, Complex Repeats, homology searches (BLAST).

  • CGG Nucleotide Sequence Analysis launcher (Sanger Institute) - Predicts gene structure (FGENESH+, BESTORF, FGENE, FEX, SPL), promoter and transctiption start sites (TSSG, TSSW), polyadenation site (POLYAH), and other features (CDSB, RNASPL, HBR).

  • RUMMAGE (The Genome Sequencing Center at the Institute of Molecular Biotechnology in Jena, Germany)- A High-Throughput Sequence Annotation Server. The following functional and organizational elements have been addressed so far: Repetitive elements (RepeatMasker, SST, CENSOR and XNUN), promotors (ProScanII), exons (GRAIL, FEXHB, MZEF, Xpound and GENSCAN), gene-structures (GENSCAN, DPS and ExonSampler), tRNA coding (tRNA), CpG islands (LCP and CPG), GC-profiling (LCP) and homology and motif searches against sequence data base (BLAST, and FASTA). (This site requires registration.)

  • NIX (The UK Human Genome Mapping Project Resource Center)- A high-throughput sequence annotation server. For genomic sequences, the functional and organizational elements addressed include: Repetitive elements (RepeatMasker), exons (GRAIL, GeneFinder, GeneMark, FEX, HEXON, FGENE), tRNA coding (tRNAscan), and sequence similarity (BLAST versus DNA and protein sequence databases).

    For transcribed sequences, the functional and organizational elements addressed so far include: Repetitive elements RepeatMasker), exons (GRAIL), and sequence similarity (BLAST versus DNA sequence databases, and TBLASTX versus protein sequence databases). (This site requires registration.)

Sequence Analysis Pipeline Software

These are pipeline software applications written in Perl that you can download and install on your UNIX system.
  • CGAT (Stanford or Johns Hopkins) - The Comprehensive Genome Analysis Tools developed in Roger Reeves' lab at Johns Hopkins by Jim Lund who is now at Stanford. Runs RepeatMasker, NCBI BLASTN (vs. nr, dbest, htgs, gss), MZEF, GRAIL, GENSCAN, tRNAscan-SE, GRAIL CpG islands, PolyA, polII promoters. Maps all results onto sequence, and generates GIF-formatted graphical display of results.

  • Adapt (Stanford DNA Sequence & Techology Center) - Developed by Curtis Palm. Runs GRAIL, FEXA, GENSCAN, MZEF, and AAT (EST and protein similarity).

Analysis Results Visualization and Curation Tools

These are software applications that display annotated features and analysis results in context with the sequence.
  • Apollo (Berkeley Drosophila Genome Project or Ensembl)- Java-based tool developed by BDGP and Ensembl.

  • WAV (Stanford DNA Sequence & Techology Center) - Web Annotation Viewer developed by Curtis Palm as part of the Adapt system.

  • CGAT (Stanford or Johns Hopkins)- Perl-based software that draws GIF images of results. Developed in Roger Reeves' lab at Johns Hopkins by Jim Lund who is now at Stanford.

Large-Scale Comparative Methods

Using genomic sequences for homologous loci in two organisms, these methods can elucidate genomic features based on the assumption that they are conserved between the two organisms.
  • PipMaker (Penn State) - Pairwise alignments of genomic sequences displayed as PIPs (Percent Identity Plots). The software is described in R.C. Hardison, J. Oeltjen, and W. Miller. Long Human-Mouse Sequence Alignments Reveal Novel Regulatory Elements: A Reason to Sequence the Mouse Genome. Genome Research, 7:10 (1997), 959-966.

  • MUMmer (TIGR) - A system for aligning whole genome sequences. Using an efficient data structure called a suffix tree, the system is able rapidly to align sequences containing millions of nucleotides. The software is described in A.L. Delcher, S. Kasif, R.D. Fleischmann, J. Peterson, O. White, and S.L. Salzberg. Alignment of whole genomes. Nucleic Acids Research, 27:11 (1999), 2369-2376.
